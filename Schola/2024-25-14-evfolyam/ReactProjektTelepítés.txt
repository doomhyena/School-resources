React:
	- JavaScript keretrendszer
	- Olyan webes alkalmazásokhoz, amelyek egy oldalból állnak
	- Erőforrásokat spórol az által, hogy csak az aktuálisan megváltozó 
		részt rendereli újra a weblapon
	- Ehhez részekre bontja az app felépítését
	- Ennek megvalósítására specializált metódusokat használ
___________________________________________________
Meglévő react projekt telepítése/futtatása:
	- A gépre előzetesen legyen telepítve a Node.js és a Git
		* Enélkül nem fogja felismerni az npm parancsokat
	- A projekt mappán belül cmd nyitása
	- Első parancs a cmd-ben: npm create vite@latest my-app
		* ez telepíti a projekt futásához szükséges modulokat
		* létrehoz egy node_modules mappát a gyökérkönyvtárban
		* (a mögöttes instrukciók a package.json fájlban vannak)
		* Jelentés: npm --> Node.js beépített parancs, i --> install
	- Második parancs: npm audit fix (Ez opcionális)
		* Ez az esetleges sebezhetőségek automatikus javítására kell
		* Sablon eljárás, a biztonság kedvéért
	- Harmadik parancs: cd my-app
		* Ezzel be navigál
	- npm install
	- Hatodik parancs: npm run dev
		* Ez lesz a projekt futtatása 
		* A localhost-on található az elérés
___________________________________________________
Megjegyzések:
	- A modulok telepítése után sokszorosára nő a projekt mérete,
	  így érdemes azok nélkül menteni a saját helyre és utána újból
	  telepíteni azokat.
	- A gyökérkönyvtár egyszerű, ékezetek nélküli elnevezést kapjon
	- Egyszerre csak egy react projekt fusson
___________________________________________________
Egy react projekt felépítése:
	- A projektmappába automatikusan generálódik egy keret 
	- a package fájlok tartalmazzák a szükséges modulok listáját
	- a readme automatikusan generált magyarázat an npm parancsokhoz
	- node_modules mappa: ő hozza létre, oda települnek a modulok 
	- public mappa: react fájlok + index.html
	- src mappa: a JavaScript és CSS fájkok helye
