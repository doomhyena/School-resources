# Szoftverfejlesztő Vizsgafeladatok – C# Megoldások

Ez a repository a **Szoftverfejlesztő és -tesztelő vizsgafeladatok C# nyelvű megoldásait** tartalmazza.
A cél: **felkészülni a vizsgára**, miközben **gyakorolod a C# és a .NET alapjait**.

---

## Használat

1. Nyisd meg **Visual Studio-ban** vagy **Rider-ben** (esetleg **VS Code + .NET SDK** kombóval).
2. Futtasd az egyes `Program.cs` fájlokat a projektmappákon belül.

---

## Függőségek

* .NET 6.0 vagy újabb
* Visual Studio / Rider / VS Code
* (Nincs extra NuGet-csomag — minden alapértelmezett .NET könyvtárból megoldva)

---

## Közreműködés

Ha szeretnéd bővíteni a repót (új feladatokkal, refaktorált megoldásokkal vagy jobb ötletekkel),
nyugodtan nyiss egy **Pull Requestet**, vagy írj egy **Issue-t**.
A cél, hogy ez egy **közösségi tudásbázis** legyen mindenkinek, aki a vizsgára készül.
