## Alap Java fejlesztő beállítások

**G1 – Build & Run**  
Billentyűparancs: `Ctrl + Shift + F10` (IntelliJ IDEA)  
Leírás: A program futtatása.

**G2 – Stop Execution**  
Billentyűparancs: `Ctrl + F2` (IntelliJ IDEA)  
Leírás: A futó program leállítása.

**G3 – Reformat Code**  
Billentyűparancs: `Ctrl + Alt + L`  
Leírás: A kód automatikus formázása.

**G4 – Auto Import / Optimize Imports**  
Billentyűparancs: `Ctrl + Alt + O`  
Leírás: Az importok optimalizálása és rendezése.

**G5 – Run Tests**  
Billentyűparancs: `Ctrl + Shift + F10` (tesztosztályon belül)  
Leírás: A kijelölt tesztosztály futtatása.  
Megjegyzés: A „Run All Tests” külön konfigurációként is beállítható.

## Produktivitási és segéd funkciók

**G6 – Git Commit + Push (macro)**  
Billentyűparancs: `Ctrl + K` → `Ctrl + Shift + K`  
Leírás: Commit és push művelet egy lépésben.

**G7 – Open Terminal**  
Billentyűparancs: `Alt + F12`  
Leírás: Beépített terminál megnyitása az IDE-ben.

**G8 – Search Everywhere**  
Billentyűparancs: `Shift + Shift`  
Leírás: Fájlok, osztályok, metódusok és szimbólumok globális keresése.

**G9 – Duplicate Line / Code Block**  
Billentyűparancs: `Ctrl + D`  
Leírás: A kijelölt sor vagy blokk megkettőzése.

**G10 – Comment / Uncomment Line**  
Billentyűparancs: `Ctrl + /`  
Leírás: Sorok gyors kommentelése vagy komment törlése.
