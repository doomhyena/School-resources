## Alap cuccok
**Alapszemlélet:**
- nem pánikolok
- egyszerű dolgokkal kezdek
- logokat nézek
- backup előtt nem nyúlok bele
- ha nem tudom, utánanézek
**Linux:**
- ls = fájlok listázása
- cd = mappaváltás
- pwd = hol vagyok
- cp = másolás
- mv = mozgatás / átnevezés
- rm = törlés
- cat / less = fájl megtekintés
- tail -f = log figyelés
- grep = keresés fájlban
- chmod = jogosultság
- chown = tulajdonos
- ps / top = futó folyamatok
- systemctl status = szolgáltatás állapot
- ssh user@ip = belépés szerverre
- logok: /var/log
**Windows:**
- RDP = távoli belépés
- Server Manager = szerepkörök
- Services = szolgáltatások kezelése
- Event Viewer = hibák
- user ≠ admin
- PowerShell:
  Get-Service
  Restart-Service
**Felhasználók és jogosultságok:**
- user = felhasználó
- group = jogosultság csoport
- Linux: rwx
- Windows: NTFS jogok
- sudo = ideiglenes admin jog
- least privilege elv
**Hálózat:**
- IP cím = gép címe
- subnet = hálózat mérete
- gateway = kijárat
- DNS = név → IP
- DHCP = IP kiosztás
- TCP = megbízható
- UDP = gyors
- ping = elérhető?
- ip a / ifconfig = IP ellenőrzés
**Gyakori portok:**
- 22 SSH
- 80 HTTP
- 443 HTTPS
- 3389 RDP
**Active Directory:**
- Domain = központi kezelés
- User / Group / Computer
- OU = rendszerezés
- GPO = szabályok kiosztása
- domain login ≠ lokális login
**Szerver szerepkörök:**
- File server = megosztás + jog
- Web server = weboldal
- Print server
- DB server = adatbázis
- szolgáltatás újraindítás gyakori megoldás
**Virtualizáció:**
- VM ≠ fizikai gép
- hypervisor
- CPU / RAM / disk kiosztás
- snapshot ≠ backup
- VM nem indul → log, erőforrás
**Biztonság:**
- tűzfal = mi mehet ki-be
- csak szükséges port nyitva
- frissítések
- antivirus
- erős jelszó
**Backup:**
- full / incremental
- restore tesztelés
- 3-2-1 szabály
- ha nem állítható vissza, nincs backup
**Hibaelhárítás:**
- mi nem működik?
- mióta?
- mi változott?
- mindenkinél vagy egy usernél?
- log ellenőrzés
- egyszerű dolgok először
**Automatizálás:**
- script = automatizálás
- Bash / PowerShell
- cron / scheduled task
**Interjún jól jövő mondatok:**
- először az alapokat ellenőrzöm
- megnézem a logokat
- nem kísérletezek élesben
- utánanézek dokumentációban
- fontos a backup és a dokumentáció

## Interjúkérdések:
1. **Mi a rendszergazda feladata?**
	Rendszerek üzemeltetése, felhasználók kezelése, hibák elhárítása, mentések és biztonság.
2. **Mit csinálsz, ha valami nem működik?**
	Megnézem a hibaüzenetet, ellenőrzöm az alapokat, logokat nézek.
3. **Mit csinálsz, ha nem tudod a megoldást?**
	Utánanézek dokumentációban, kérdezek, nem találgatok.
4. **Hol keresel hibát Linuxon?**
	A logokban, főleg a /var/log alatt.
5. **Hogyan nézed meg, fut-e egy szolgáltatás Linuxon?**
	systemctl status szolgáltatás_neve
6. **Hogyan lépsz be távoli Linux szerverre?**
	SSH-val: ssh user@ip
7. **Mit csinál a sudo?**
	Ideiglenes admin jogot ad.
8. **Mit jelent a chmod 755?**
	Tulajdonos mindent, mások olvasás és futtatás.
9. **Hol nézed meg a hibákat Windows Serveren?**
	Event Viewerben.
10. **Hogyan indítasz újra egy szolgáltatást Windows alatt?**
	Services-ben vagy PowerShellből Restart-Service.
11. **Mi a különbség user és admin között?**
	Admin mindent kezelhet, user csak amihez joga van.
12. **Mi az IP cím?**
	A gép címe a hálózaton.
13. **Mi a DNS?**
	Névfeloldás IP címre.
14. **Mi a DHCP?**
	Automatikusan IP címet oszt.
15. **Mi a különbség TCP és UDP között?**
	TCP megbízható, UDP gyorsabb.
16. **Mit csinál a ping?**
	Ellenőrzi, hogy elérhető-e a másik gép.
17. **Melyik port az SSH?**
	22
18. **Melyik port az RDP?**
	3389
19. **Melyik port a HTTPS?**
	443
20. **Mi az Active Directory?**
	Központi felhasználó- és gépkezelő rendszer.
21. **Mi az OU?**
	Logikai csoportosítás az Active Directoryban.
22. **Mire jó a GPO?**
	Szabályokat oszt ki felhasználóknak és gépeknek.
23. **Mi az a virtuális gép?**
	Szoftveresen futó szerver.
24. **Mi az a hypervisor?**
	A virtualizációt kezelő rendszer.
25. **Mi a snapshot?**
	Gyors állapotmentés, nem teljes backup.
26. **Miért fontos a backup?**
	Mert hiba esetén vissza lehet állítani az adatokat.
27. **Mi a 3-2-1 szabály?**
	3 mentés, 2 külön médium, 1 külön helyen.
28. **Mit csinál a tűzfal?**
	Szabályozza a bejövő és kimenő forgalmat.
29. **Mi az a least privilege?**
	Csak a szükséges jogokat adjuk meg.
30. **Hogyan kezdesz egy hibaelhárítást?**
	Egyszerű dolgokkal, szolgáltatás, hálózat, logok.
31. **Azonnal újraindítod a szervert?**
	Nem, előbb megnézem mi a probléma.
32. **Mit mondasz, ha nem tudod a választ interjún?**
	Őszintén mondom, hogy utánanéznék, és elmondom hogyan.
