## 1. Importok

```java
import javax.naming.InvalidNameException;
import javax.naming.Name;
import javax.swing.*;
import java.awt.*;
import java.awt.event.ActionEvent;
import java.awt.event.ActionListener;
import java.io.IOException;
import java.util.Random;
```

- `javax.swing.*;`  
    - A Swing GUI elemek (JFrame, JButton, JRadioButton, stb.) innen jönnek.
- `java.awt.*;`  
    - AWT-s dolgok, pl. `Button`, `Label`, `Layout` stb.  
    (Itt amúgy Swing + AWT keveredik, nem szép, de oktatási példának oké.)
- `java.awt.event.*;`  
    - Eseménykezelés: `ActionEvent`, `ActionListener` – ezek kellenek a gombnyomásokhoz.
- `java.util.Random;`  
    - Véletlenszám-generátor (a kommentelt részben lett volna használva).
- `static javax.swing.JOptionPane.showMessageDialog;`  
    - Így elég csak annyit írni, hogy `showMessageDialog(...)`, nem kell `JOptionPane.showMessageDialog(...)`.

A `InvalidNameException`, `Name`, `IOException`, `A`, `B` itt _valójában nem kellenek_ ehhez a kódhoz – csak importálva vannak, de nem használod őket.

## 2. main metódus – hol indul a program?

```java
public static void main(String[] args) throws IOException, InvalidNameException {
    ...
}
```
- Ez a Java program belépési pontja.
- A `throws IOException, InvalidNameException` azt jelenti, hogy ha ilyen kivétel történik, akkor a `main` nem kezeli, hanem továbbdobná – de jelenleg semmi nem dob ilyet, szóval ez most csak dísz.

## 3. JFrame létrehozása – maga az ablak

```java
JFrame frame = new JFrame("Grafikus program");
frame.setTitle("Grafikus program címe");
frame.setDefaultCloseOperation(JFrame.EXIT_ON_CLOSE);
frame.setResizable(true);
frame.setSize(420, 420);
frame.setVisible(true);
ImageIcon icon = new ImageIcon("icon.png");
frame.setIconImage(icon.getImage());
```

Részletezve:

- `JFrame frame = new JFrame("Grafikus program");`  
    - Létrehozol egy ablakot. A konstruktorban megadott szöveg az ablak címsorában jelenik meg (bár utána `setTitle`-lel felülírod).
- `frame.setTitle("Grafikus program címe");`  
    - Beállítod külön a címet. Ez az ablak tetején lévő szöveg.
- `frame.setDefaultCloseOperation(JFrame.EXIT_ON_CLOSE);`  
    - Ha az X-re kattint a user, a program ténylegesen bezár (különben lehet, hogy csak az ablak tűnne el, de a program futna tovább háttérben).
- `frame.setResizable(true);`  
    - Lehessen-e átméretezni az ablakot. `true` → igen.
- `frame.setSize(420, 420);`  
    -  Ablak mérete: szélesség = 420px, magasság = 420px.
- `frame.setVisible(true);`  
    - Megjeleníti az ablakot. Ha ezt nem hívod meg, nem látod az egészet.
- `ImageIcon icon = new ImageIcon("icon.png");`  
    - Betölt egy képet fájlból (ugyanabban a mappában keresi, ahol a program fut).
- `frame.setIconImage(icon.getImage());`  
    - Beállítja az ablak ikonjaként (bal felső sarok, tálca stb.).

## 4. A kommentbe tett rész – gomb, label, számláló, üzenetek

Ez a blokk most kommentben van (`/* ... */`), de ettől még érthetjük, mit csinálna.

### 4.1. Gomb + label

```java
frame.setLayout(null);
Button button = new Button("Ez egy gomb");
button.setBounds(20,30,100,30);
frame.add(button);
```

- `frame.setLayout(null);`  
    - Kikapcsolod a layout managert. Magyarul: **mindent kézzel kell pozicionálni** `setBounds`-szal.
- `Button button = new Button("Ez egy gomb");`  
    - AWT-s gomb. (Swinges párja: `JButton`.)
- `button.setBounds(20,30,100,30);`  
    - X = 20, Y = 30, szélesség = 100, magasság = 30.
- `frame.add(button);`  
    -  Hozzáadod az ablakhoz.

ActionListener:

```java
button.addActionListener(new ActionListener() {
    public void actionPerformed(ActionEvent e) {
        // button.setLocation(new Random().nextInt(200), new Random().nextInt(200));
    }
});
```

- `addActionListener(...)`  
    - Beállítasz egy **eseménykezelőt** a gombhoz.
- `public void actionPerformed(ActionEvent e)`  
    - Ez a metódus fut le, amikor rákattintanak a gombra.
- A kommentelt sor:  
    `button.setLocation(new Random().nextInt(200), new Random().nextInt(200));`  
    - Gombot véletlen helyre rakná 0–200 közötti x-y koordinátákkal.

Label:

```java
Label label = new Label("Ez egy label");
label.setBounds(120,30,100,30);
frame.add(label);
```

Simán egy felirat, ami odakerül a megadott helyre.

### 4.2. Számláló gomb (hányszor nyomták meg)

```java
Button szamlalo = new Button("Számolj");
szamlalo.setBounds(20,70,100,30);
frame.add(szamlalo);
Label szam = new Label();
szam.setBounds(120,30,100,30);
frame.add(szam);
```

- `szamlalo` = egy gomb, amit nyomogatunk.
- `szam` = label, ahova kiírhatnád az aktuális számot (bár most msgbox-ot használsz).

ActionListener:

```java
szamlalo.addActionListener(new ActionListener() {
    int counter = 0;
    public void actionPerformed(ActionEvent e) {
        counter++;
        // szam.setText("Szám: " + counter);
        showMessageDialog(null, "Szám: " + counter);
    }
});
```

Fontos dolgok itt:

- `int counter = 0;`  
    - Ez az **anonym inner class** mezője, ezért megjegyzi az értékét hívások között. Minden gombnyomásnál **nem nulláról indul újra**, hanem nő.
- `counter++;`  
    - Növeled a számlálót.
- `showMessageDialog(null, "Szám: " + counter);`  
    - Felugró ablakot mutat a képernyő közepén (`null` → nincs konkrét szülő komponens).

Ha azt a kommentelt sort használnád:

```java
szam.setText("Szám: " + counter);
```

akkor a labelben jelenne meg az érték, nem egy dialogban.

### 4.3. Anti-spam gomb

```java
Button szamol = new Button("Ne spamemelj");
szamol.setBounds(20,70,100,30);
frame.add(szamol);
```

Másik gomb, saját ActionListenerrel:

```java
szamol.addActionListener(new ActionListener() {
    int counter = 0;
    public void actionPerformed(ActionEvent e) {
        counter++;
        if (counter == 5) {
            showMessageDialog(null, "Ne spamemelj, már 5x megnyomtad!");
        } else if (counter == 10) {
            showMessageDialog(null, "Ne spamemelj, 10x megnyomtad így bezárok!");
            System.exit(0);
        }
    }
});
```

Logika:

- Van egy `counter`, ami szintén az anonym osztály mezője → megőrzi az értéket.
- Minden kattintásnál:
    - `counter++`
    - Ha 5: kiír egy figyelmeztetést.
    - Ha 10: kiír egy üzenetet, majd `System.exit(0);` → kilép a programból.

## 5. K1 és K2 gombok – láthatóság kapcsolgatása

Ez már **nincs kommentelve**, ténylegesen fut.

```java
Button btnk1 = new Button("K1");
btnk1.setBounds(140,60,100,30);
frame.add(btnk1);
Button btnk2 = new Button("K2");
btnk2.setBounds(140,100,100,30);
frame.add(btnk2);
btnk2.setVisible(false);
```

- Két gombod van:
    
    - `btnk1` – látszik induláskor.
    - `btnk2` – ugyanúgy hozzáadod a frame-hez, de `setVisible(false);` → kezdetben nem látszik.

ActionListenerek:

```java
// K2 fog látszani
btnk1.addActionListener(new ActionListener() {
    public void actionPerformed(ActionEvent e) {
        btnk2.setVisible(true);
        btnk1.setVisible(false);
    }
});
```

- Ha rákattintasz `K1`-re:
    - `btnk2.setVisible(true);` → K2 látható
    - `btnk1.setVisible(false);` → K1 eltűnik

```java
// K1 fog látszani
btnk2.addActionListener(new ActionListener() {
    public void actionPerformed(ActionEvent e) {
        btnk1.setVisible(true);
        btnk2.setVisible(false);
    }
});
```

- Ha rákattintasz `K2`-re:
    - K1 újra látszik
    - K2 eltűnik

Ez kb. olyan, mintha **kapcsolgatnál két állapot között**.

## 6. RadioButton-ok + ButtonGroup

```java
// radio button
JRadioButton rdbtA = new JRadioButton("A");
JRadioButton rdbtB = new JRadioButton("B");
JRadioButton rdbtC = new JRadioButton("C");
```

- Három **választó gomb** (rádiógomb), mindegyik mellett szöveg: `"A"`, `"B"`, `"C"`.
    

```java
ButtonGroup bg = new ButtonGroup();
bg.add(rdbtA);
bg.add(rdbtB);
bg.add(rdbtC);
```

- A `ButtonGroup` azt biztosítja, hogy **egyszerre csak egy** legyen kijelölve.
    - Ha rákattintasz B-re, akkor A automatikusan "felenged".
    - Klasszikus feleletválasztós kérdés stílus.

Pozicionálás:

```java
frame.setLayout(null);
frame.add(rdbtA);
frame.add(rdbtB);
frame.add(rdbtC);
rdbtA.setBounds(140,140,100,30);
rdbtB.setBounds(140,180,100,30);
rdbtC.setBounds(140,220,100,30);
```

- Megint `null` layout → a rádiógombokat is kézzel pozicionálod.
- `setBounds`-szal rakod őket egymás alá.

## 7. Összefoglalva a fontos fogalmak

- **JFrame** – maga az ablak.
- **Component-ek (Button, Label, JRadioButton, stb.)** – az ablakra pakolt elemek.
- **Layout manager**:
    - `null` layout → te adod meg fix koordinátákkal (`setBounds`).
- **ActionListener**:
    - `addActionListener(new ActionListener() { ... })`
    - Az `actionPerformed` fut le kattintáskor.
    - Az `int counter` az anonym osztályon belül elmentődik, ezért tudsz kattintásszámot számolni.
- **ButtonGroup + JRadioButton**:
    - Kölcsönösen kizáró választás, mint a tesztkérdések.
- **setVisible(true/false)**:
    - Ezzel tudsz komponenseket megjeleníteni/eltüntetni anélkül, hogy törölnéd őket.
