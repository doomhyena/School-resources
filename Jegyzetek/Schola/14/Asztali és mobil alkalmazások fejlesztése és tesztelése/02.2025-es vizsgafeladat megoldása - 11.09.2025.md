## Mit csinál ez a program?

1. **Beolvassa** egy `eloadok.csv` fájl tartalmát.  
    -> Minden sor tartalmaz egy zeneszám adatait: előadó (band), cím (song), év (year), streamek száma (streams).
2. **Objektumokat (Zene)** hoz létre ezekből az adatokból, és eltárolja őket egy `ArrayList`-ben.
3. Lefuttat különböző műveleteket:
    - Kiírja a **Linkin Park legtöbbet streamelt számát**.
    - Összesíti az **előadók streameit**.
    - Kiírja az adatokat egy másik fájlba (`streamek.txt`).

---

## Részek magyarázata

### 1. `Zene` osztály

Ez egy **adattároló osztály (POJO – Plain Old Java Object)**:

```java
static class Zene {
    private String band;
    private String song;
    private int year;
    private long streams;
}
```

- `private` mezők → csak getter/setter-en keresztül érhetők el.
- Konstruktor: `public Zene(...)` → példányosításkor feltölti az adatokat.
- Getter/setter → kapszulázás (encapsulation).
- `toString()` → szabályozza, hogy a `System.out.println(zene)` mit írjon ki.

---

### 2. `feltolt()` – fájl beolvasása

```java
File f = new File("eloadok.csv");
Scanner be = new Scanner(f, "UTF-8");
```

- Megnyitja az `eloadok.csv` fájlt.
- `be.nextLine();` → első sor átugrása (fejléc).
- `sor.split(";")` → minden sort `;` mentén feldarabol.
- Létrehoz egy új `Zene` objektumot, és hozzáadja a listához (`zenek.add(...)`).

Végén lesz egy **ArrayList zenek**, ami tele van a fájl adataival.

---

### 3. `linkinParkMax()` – Linkin Park legnagyobb streames dala

```java
if (z.getBand().equalsIgnoreCase("Linkin Park")) {
    if (max == null || z.getStreams() > max.getStreams()) {
        max = z;
    }
}
```

- Végigmegy az összes zenén.
- Ha az előadó neve **"Linkin Park"**, akkor megnézi, hogy ez-e a legnagyobb `streams` érték.
- A végén kiírja a legtöbbet streamelt Linkin Park számot.

---

### 4. `osszesStreameles()` – előadónként összegzés

```java
Map<String, Integer> streamelesek = new HashMap<>();
```

- Készít egy **HashMap-et**, ahol:
    - kulcs = előadó neve (`String`)
    - érték = összes stream (`Integer`)

Működés:

- Ha az előadó még nincs a map-ben → berakja.
- Ha már benne van → hozzáadja az új streameket.

Hiba: `System.out.println` a for cikluson belül van, így minden iterációban újra és újra kiírja az egész mapet.  
Helyesebb lenne a ciklus után kiírni.

---

### 5. `fajlstreamek()` – fájlba kiírás

```java
FileWriter fw = new FileWriter("streamek.txt");
for (Zene zene : zenek) {
    fw.write(zene + "\n");
}
fw.close();
```

- Létrehoz egy `streamek.txt` fájlt.
- Soronként beleírja az összes zenét.
- A `toString()` miatt szép formátumban jelenik meg.

---

### 6. `main()` – a program futtatása

```java
feltolt();          // betöltjük az adatokat
for (Zene n : zenek) {
    System.out.println(n); // kiírjuk
}
linkinParkMax();    // Linkin Park legtöbb streamelt dala
osszesStreameles(); // előadónkénti streamek
fajlstreamek();     // kiírás fájlba
```
