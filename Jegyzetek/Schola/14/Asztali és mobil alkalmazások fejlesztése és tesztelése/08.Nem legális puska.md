## 2) SABLON: CSV beolvasás Scannerrel (zenés “feltolt”)

**Minta:**

```java
import java.io.File;
import java.io.FileNotFoundException;
import java.util.Scanner;

public static void feltolt() {
    try (Scanner sc = new Scanner(new File("eloadok.csv"), "UTF-8")) {
        sc.nextLine(); // fejléc átugrás

        while (sc.hasNextLine()) {
            String line = sc.nextLine();
            String[] p = line.split(";");

            if (p.length == 4) {
                String band = p[0];
                String song = p[1];
                int year = Integer.parseInt(p[2]);
                int streams = Integer.parseInt(p[3]);
                eloadok.addElement(new Eloado(band, song, year, streams));
            }
        }

    } catch (FileNotFoundException e) {
        System.err.println("Hiba: A fájl nem található!");
    } catch (Exception e) {
        System.err.println("Hiba a beolvasás során: " + e.getMessage());
    }
}
```

## 6) SABLON: “3 soros” beolvasás + szűrés + kiírás (COUNTRY feladat)

### Beolvasás (BufferedReader UTF-8)

```java
public static List<Country> loadFromFile(String fileName) throws IOException {
    List<Country> list = new ArrayList<>();

    try (BufferedReader br = new BufferedReader(
            new InputStreamReader(new FileInputStream(fileName), StandardCharsets.UTF_8))) {

        String name;
        while ((name = br.readLine()) != null) {
            String popLine = br.readLine();
            String contLine = br.readLine();
            if (popLine == null || contLine == null) break;

            int pop = Integer.parseInt(popLine.trim());
            String cont = contLine.trim();

            list.add(new Country(name.trim(), pop, cont));
        }
    }
    return list;
}
```

### Szűrés + kiírás

```java
public static void saveEuropeans(List<Country> countries, String fileName) throws IOException {
    try (BufferedWriter bw = new BufferedWriter(
            new OutputStreamWriter(new FileOutputStream(fileName), StandardCharsets.UTF_8))) {

        for (Country c : countries) {
            if (c.getContinent().equalsIgnoreCase("Európa")) {
                bw.write(c.getName()); bw.newLine();
                bw.write(Integer.toString(c.getPopulation())); bw.newLine();
                bw.write(c.getContinent()); bw.newLine();
            }
        }
    }
}
```
