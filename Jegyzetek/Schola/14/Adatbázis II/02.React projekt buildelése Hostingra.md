Ha beírod a projekted mappájába (fontos, hogy konzolba írd be):

```bash
npm run build
```

parancsot, akkor lesz egy ilyen mappád:

```
my-react-app/
 ├── build/
 │    ├── index.html
 │    ├── asset-manifest.json
 │    ├── static/
 │    │    ├── js/
 │    │    └── css/
 │    └── ...
 ├── src/
 ├── public/
 ├── package.json
 └── stb.
```

Ebből **csak a `build` mappán belüli fájlokat** (pl. `index.html`, `static/`, stb.) kell feltöltened a hosting **gyökérmappájába** (pl. `public_html` vagy `www`).  
Ne legyen az, hogy a weboldalad útvonala így néz ki:

```
https://sajatoldal.hu/build/index.html
```

hanem így:

```
https://sajatoldal.hu/index.html 
```

---

### TL;DR:

- **Ne töltsd fel:** `src`, `node_modules`, `package.json`, stb.
- **Ne töltsd fel a `build` mappát egészben.**
- **Csak a tartalmát** (a benne lévő fájlokat és mappákat) töltsd fel a tárhely gyökerébe.
