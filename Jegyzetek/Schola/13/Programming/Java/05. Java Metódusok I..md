**Metódusok, avagy a programozás igáslovai**

A Java nyelvben rengeteg előre megírt metódust használhatunk, amelyek segítik a fejlesztő munkáját. Ezek végzik a programunk működésének nagy részét, kezelik a változókat, és biztosítják a program különböző részei közötti kapcsolatot.

### Objektumok és osztályok

- **Objektum**: A valós világ egy elemét modellező elem, amely rendelkezik tulajdonságokkal és viselkedésekkel. Például egy kutya objektumnak lehet neve, fajtája, színe, magassága, valamint viselkedései, mint az evés, ivás, ugatás.
- **Osztály**: Az objektumok tervrajza, amely tartalmazza azok tulajdonságait és viselkedéseit. Az objektumokat osztályok alapján hozzuk létre.

### Metódusok

A metódusokat tekinthetjük egyfajta **üzeneteknek**, amelyek valamilyen feladat végrehajtását kérik. Az üzenet 3 részből állhat:

1. **Címzett objektum vagy osztály**
2. **A címzett metódusának neve**
3. **Esetleges paraméterek**

Ha a címzett nem önmaga, akkor kötelező megadni a címzettet. A metódusok neve egy adott osztályon belül egyedi kell legyen, figyelembe véve a bemenő adatok típusait és számát (túlterhelés lehetséges).

### Metódusok használata

Példák olyan metódusokra, ahol a címzettet is meg kell adni:

```java
Math.min(23, b); // Két szám közül a kisebbet adja vissza
Math.sqrt(18);   // 18 négyzetgyökét számolja ki
Integer.parseInt("123"); // Szöveget egész számmá alakít
System.out.println("Ez egy rövid mondat."); // Szöveg kiírása a konzolra
```

### A metódusok két fő csoportja

1. **Értéket előállító metódusok**:
    - Számítást végeznek és visszatérési értéket adnak vissza (pl. `Math.sqrt()`).
    - Paramétereket is használhatnak, de nem kötelező.
2. **Tevékenységet végrehajtó metódusok**:
    - Valamilyen műveletet hajtanak végre, például adatokat írnak ki a konzolra (`System.out.println()`).

A metódusokat az alábbi formátumban használjuk:

```java
Osztalynev.metodus();
Osztalynev.metodus(parameter);
Osztalynev.metodus(parameter1, parameter2, ...);
```

A metódushoz tartozó osztály neve szükséges, mivel különböző osztályokban lehetnek azonos nevű metódusok.

### A Math osztály legfontosabb metódusai

A `Math` osztály matematikai műveletek végrehajtására szolgál. Néhány gyakran használt metódus:

```java
Math.min(a, b);  // Két szám közül a kisebbet adja vissza
Math.random();   // Véletlenszámot generál [0;1[ intervallumban
Math.round(a);   // Matematikai szabály szerinti kerekítés
Math.abs(a);     // Abszolút érték
Math.sqrt(a);    // Négyzetgyök
Math.pow(a, b);  // Hatványozás (a^b)
Math.PI;        // Pi értéke (nem metódus, hanem konstans)
Math.E;         // Euler-féle szám (nem metódus, hanem konstans)
```

Ezek a metódusok nagy segítséget nyújtanak a programozásban, és érdemes őket alaposan megismerni.
