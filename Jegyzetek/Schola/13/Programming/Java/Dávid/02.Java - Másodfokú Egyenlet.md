### **1. Véletlenszerű együtthatók generálása**

```java
Random rnd = new Random();
double a = rnd.nextInt(100 - 10 + 1) + 10; // 10 és 100 közötti véletlenszerű érték
```

- `a` véletlenszerűen generált szám, amely biztosan 10 és 100 közé esik.

---

### **2. Első ellenőrzés: Másodfokú egyenlet érvényessége**

```java
if (a == 0) {
    System.out.println("Nem másodfokú egyenlet!");
}
```

- Ha `a` értéke 0 lenne, az nem másodfokú egyenlet lenne. Mivel a generált `a` soha nem lehet 0, ez az ellenőrzés redundáns a kód jelenlegi formájában.

---

### **3. Másodfokú egyenlet megoldása**

Ha az egyenlet érvényes, generál további együtthatókat:

```java
double b = rnd.nextInt(100 - 10 + 1) + 10; // Véletlen érték 10 és 100 között
double c = rnd.nextInt(100 - 10 + 1) + 10;
```

#### Diszkrimináns kiszámítása:

```java
double diszkr = b * b - 4 * a * c;
```

- A diszkrimináns segítségével eldönthető, hogy a másodfokú egyenletnek hány megoldása van:
    - Ha `diszkr > 0`, két különböző valós gyöke van.
    - Ha `diszkr == 0`, pontosan egy valós gyöke van.
    - Ha `diszkr < 0`, nincs valós megoldás.

---

### **4. Esetek kezelése**

- **Két különböző valós megoldás:**
    
    ```java
    if (diszkr > 0) {
        double x1 = (-b + Math.sqrt(diszkr)) / (2 * a);
        double x2 = (-b - Math.sqrt(diszkr)) / (2 * a);
        System.out.println("x1: " + x1);
        System.out.println("x2: " + x2);
    }
    ```
    
    - A két gyök képletét alkalmazza:  
        x1=−b+Δ2ax_1 = \frac{-b + \sqrt{\Delta}}{2a}  
        x2=−b−Δ2ax_2 = \frac{-b - \sqrt{\Delta}}{2a}
- **Egy valós megoldás:**
    
    ```java
    else if (diszkr == 0) {
        double x = (-b) / (2 * a);
        System.out.println("x: " + x);
    }
    ```
    
    - A gyök képlete ilyenkor:  
        x=−b2ax = \frac{-b}{2a}
- **Nincs valós megoldás:**
    
    ```java
    else {
        System.out.println("Nincs valós megoldás!");
    }
    ```
