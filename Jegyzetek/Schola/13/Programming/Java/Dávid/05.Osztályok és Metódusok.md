### **1. A `Szemely` osztály**
Ez az osztály egy személyt reprezentál, amelynek két adattagja van:  
- `nev`: a személy neve (szöveges típus, `String`)  
- `kor`: a személy életkora (egész szám, `int`)  

#### **Konstruktor:**
A konstruktort az `insert code -> constructor -> mindent pipa és generate` módszerrel hozták létre a fejlesztői környezetben.  

```java
public Szemely(String neve, int kora) {
    this.nev = neve;
    this.kor = kora;
}
```
Ez az osztály példányosításakor beállítja a személy nevét és korát.

#### **toString() metódus felülírása:**
```java
@Override
public String toString() {
    return nev + ", " + kor;
}
```
Ez azért kell, hogy ha egy `Szemely` objektumot kiírunk (`System.out.println(sz1);`), akkor az osztály megfelelően jelenítse meg az adatokat (`Név, Kor` formátumban).

### **3. A `main()` metódus**
Itt történik a program fő futása.

#### **Példa létrehozása és módosítása:**
```java
Szemely sz1 = new Szemely("Béla", 40);
System.out.println(sz1);
System.out.println(sz1.nev);
System.out.println(sz1.kor);
```
- Létrehoz egy `Szemely` objektumot "Béla" névvel és 40 évvel.  
- Kiírja az objektum adatait a képernyőre.  

Majd módosítja az objektum adatait:
```java
sz1.nev = "Lajos";
sz1.kor = 30;
System.out.println(sz1.nev);
System.out.println(sz1.kor);
```
Itt megváltoztatja Béla nevét Lajosra és az életkorát 30-ra.

### **4. A `feltolt()` metódus**
Ez generál **113 db random személyt**, akiknek az életkora **20 és 60 év között** van.  
```java
private static void feltolt() {
    Random rnd = new Random();
    Szemely sz;
    for (int i = 0; i < 113; i++) {
        sz = new Szemely("Név"+i, rnd.nextInt(60-20+1)+20);
        szemelyek.add(sz);
    }
}
```
Ez a rész:
```java
rnd.nextInt(60-20+1)+20
```
- A `nextInt(41)` egy 0 és 40 közötti számot ad vissza.  
- Ehhez hozzáadunk `20`-at, így a végeredmény **20 és 60 között** lesz.

### **5. A `legidosebb()` metódus**
Ez megkeresi a **legidősebb személyt** a listában.
```java
private static void legidosebb() {
    Szemely max = szemelyek.get(0); // Kezdőérték az első személy
    
    for (Szemely n : szemelyek){
        if (n.kor > max.kor)
            max = n;
    }
    
    System.out.println("Legidősebb: "+max);
}
```
- Az első személyt (`szemelyek.get(0)`) tekinti kezdetben a legidősebbnek.  
- Végigmegy a listán, és ha talál egy idősebb személyt, frissíti a `max` változót.  
- A ciklus végén kiírja a legidősebb személyt.
