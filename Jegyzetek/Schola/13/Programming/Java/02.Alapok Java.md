### 1. If-Else

Az `if-else` feltételes utasításokat akkor használjuk, ha különböző kódblokkokat szeretnénk végrehajtani adott feltételek teljesülése esetén.

```java
int x = 10;

if (x > 5) {
    System.out.println("x nagyobb, mint 5");
} else {
    System.out.println("x nem nagyobb, mint 5");
}

```

Az `if` blokk akkor fut le, ha a feltétel igaz (`x > 5`), míg az `else` blokk akkor, ha nem teljesül.

### Több ágú elágazás

```java
if (x > 10) {
    System.out.println("x nagyobb, mint 10");
} else if (x == 10) {
    System.out.println("x egyenlő 10-zel");
} else {
    System.out.println("x kisebb, mint 10");
}

```

### 2. Switch

A `switch` utasítás több ágat biztosít különböző értékek kezelésére, és kifejezetten akkor hasznos, ha egy változó értékére sok különböző esetet kell vizsgálni.

```java
int day = 3;

switch (day) {
    case 1:
        System.out.println("Hétfő");
        break;
    case 2:
        System.out.println("Kedd");
        break;
    case 3:
        System.out.println("Szerda");
        break;
    default:
        System.out.println("Ismeretlen nap");
}

```

A `break` megakadályozza a további esetek lefutását. Ha nincs `break`, akkor a következő esetek is végrehajtódnak.

### 3. While ciklus

A `while` ciklus addig ismétel egy kódrészt, amíg egy adott feltétel igaz.

```java
int i = 0;

while (i < 5) {
    System.out.println("i értéke: " + i);
    i++;
}

```

Ez addig fog futni, amíg az `i < 5` feltétel igaz.

### 4. For ciklus

A `for` ciklus a ciklus változó inicializálására, feltételre és az értékének növelésére/csökkentésére épül. Többnyire előre ismert ismétlések számánál használatos.

```java
for (int i = 0; i < 5; i++) {
    System.out.println("i értéke: " + i);
}

```

Itt a ciklus három részből áll:

1. `int i = 0;` - inicializálás
2. `i < 5;` - feltétel, amíg a ciklus fut
3. `i++` - ciklusváltozó módosítása

### 5. Break és Continue

- **Break:** Megszakítja a ciklust, és kilép belőle.
- **Continue:** Átlépi a ciklus aktuális iterációját, és a következő iterációval folytatja.

```java
for (int i = 0; i < 10; i++) {
    if (i == 5) {
        break; // Kilép a ciklusból, ha i értéke 5
    }
    System.out.println(i);
}

for (int i = 0; i < 10; i++) {
    if (i % 2 == 0) {
        continue; // Átlépi a páros számokat
    }
    System.out.println(i);
}

```

### 6. 1D és 2D tömbök

- **1D tömb (egydimenziós):** Egyszerű elemek listája.

```java
int[] szamok = {1, 2, 3, 4, 5};
System.out.println(szamok[0]); // Első elem kiírása

```

- **2D tömb (kétdimenziós):** Tömbök tömbje, ahol egy mátrix-szerű struktúrát alakítunk ki.

```java
int[][] matrix = {
    {1, 2, 3},
    {4, 5, 6},
    {7, 8, 9}
};
System.out.println(matrix[0][0]); // Első sor, első elem

```

### 7. Osztályok (Classes)

Az osztályok az objektum-orientált programozás alapját képezik. Egy osztályban adatokat (mezők) és funkciókat (metódusok) tárolunk, amelyekkel az objektumok viselkedését határozzuk meg.

```java
public class Auto {
    // Mezők
    String szin;
    int sebesseg;

    // Konstruktor
    public Auto(String szin, int sebesseg) {
        this.szin = szin;
        this.sebesseg = sebesseg;
    }

    // Metódusok
    public void gyorsit(int novekmeny) {
        this.sebesseg += novekmeny;
    }
}

// Objektum példányosítása
Auto auto = new Auto("piros", 50);
auto.gyorsit(10);
System.out.println(auto.sebesseg); // Kimenet: 60

```

### 8. Felhasználói Bemenet (User Input)

Java-ban a felhasználói bemenet kezelésére leggyakrabban a `Scanner` osztályt használjuk.

```java
import java.util.Scanner;

public class BemenetPeldaja {
    public static void main(String[] args) {
        Scanner scanner = new Scanner(System.in);

        System.out.print("Add meg a neved: ");
        String nev = scanner.nextLine();

        System.out.print("Add meg az életkorodat: ");
        int kor = scanner.nextInt();

        System.out.println("Szia " + nev + ", te " + kor + " éves vagy.");

        scanner.close(); // Scanner lezárása
    }
}

```

## 9. Try - catch

A `try-catch` blokk a Java-ban az **hibakezelésre** szolgál. Segítségével megadhatjuk, hogyan reagáljon a program bizonyos kivételes (hibás) helyzetekre, így elkerülhető a program összeomlása.

### Hogyan működik?

A `try` blokkban olyan kódot írunk, amely potenciálisan hibát (kivételt) okozhat. Ha hiba történik, a program a `catch` blokkba ugrik, ahol lekezelhetjük ezt a hibát. A `try-catch` szerkezet alapvetően így néz ki:

```java
try {
    // Kód, amelyben hiba (kivétel) történhet
} catch (Exception e) {
    // A hiba kezelése
}

```

### Példa

Például egy osztásnál `ArithmeticException` keletkezhet, ha nullával próbálunk osztani.

```java
public class TryCatchPeldak {
    public static void main(String[] args) {
        try {
            int a = 10;
            int b = 0;
            int eredmeny = a / b; // Nullával való osztás!
            System.out.println("Eredmény: " + eredmeny);
        } catch (ArithmeticException e) {
            System.out.println("Hiba történt: " + e.getMessage());
        }
    }
}

```

Ebben a példában, mivel `b` értéke nulla, a `try` blokkban `ArithmeticException` keletkezik, így a program a `catch` blokkba lép, és kiírja: **"Hiba történt: / by zero"**.

### Több `catch` blokk

Különböző kivételtípusokra külön `catch` blokkokat adhatunk meg:

```java
try {
    int[] tomb = {1, 2, 3};
    System.out.println(tomb[5]); // IndexOutOfBoundsException
} catch (ArithmeticException e) {
    System.out.println("Aritmetikai hiba: " + e.getMessage());
} catch (ArrayIndexOutOfBoundsException e) {
    System.out.println("Tömb index hiba: " + e.getMessage());
}

```

Ebben a példában egy `ArrayIndexOutOfBoundsException` jön létre, mert a 5. indexű elem nem létezik a tömbben.

### Végül blokk (`finally`)

A `finally` blokk mindig lefut, függetlenül attól, hogy történt-e kivétel vagy sem. Erre tipikusan erőforrások felszabadításához (pl. fájl lezárása, adatbázis-kapcsolat bontása) van szükség.

```java
try {
    int eredmeny = 10 / 2;
    System.out.println("Eredmény: " + eredmeny);
} catch (Exception e) {
    System.out.println("Hiba történt: " + e.getMessage());
} finally {
    System.out.println("Ez mindig lefut.");
}
```
