### 1. Üdvözlő üzenet és nap kiíratása

A kód elején egy egyszerű feltételes elágazás van, amely ellenőrzi, hogy a hónap napja 2-e:

```java
int dayofMonths = 2;

System.out.println("Welcome @doomhyena!");
System.out.print("Today is ");

if (dayofMonths == 2) {
    System.out.println("Monday.");
} else {
    System.out.println("not Monday.");
}

System.out.println("Goodbye @doomhyena!");

```

Ez a rész kiírja, hogy "Today is Monday", ha a `dayofMonths` változó értéke 2, különben azt, hogy "not Monday". Ezután egy búcsúüzenetet ír ki.

### 2. Alapvető műveletek két szám között

Ezután két szám szorzatát és összegét írja ki:

```java
int a = 10;
int b = 5;

System.out.println(a * b); // 50
System.out.println(a + b); // 15

```

Egyszerű matematikai műveleteket végez: szorzást és összeadást.

### 3. Véletlenszám generálása

A kód véletlenszerű számokat generál a `Random` osztály segítségével:

```java
Random random = new Random();
int randomNumber = random.nextInt(11);  // Véletlenszerű szám 0 és 10 között
int randomNumber0 = random.nextInt(11);

System.out.println(randomNumber);
System.out.println("A jelenlegi generált szám: " + randomNumber0);

```

Itt két véletlenszám generálódik 0 és 10 között.

### 4. Lista kezelése

A kód listákkal dolgozik, és több helyen használ listát:

```java
List<String> list = Arrays.asList("Albert", "Balázs", "Cecil", "Igor");
System.out.println(list);
System.out.println(list.get(3)); // A lista 3. indexén lévő elem, azaz "Igor"

```

Ez a rész létrehoz egy listát nevekkel, és kiírja a 3. indexű elemet, amely "Igor". A listák méretét és maximális indexét is kezeli:

```java
int elemszam = list.size(); // Lista mérete
int maxIndex = list.size() - 1; // Max index
System.out.println("Max index: " + maxIndex);

```

### 5. Random index és elem választása

A kód véletlenszerűen választ egy elemet a listából:

```java
int randomIndex = random.nextInt(maxIndex + 1);
System.out.println("Lista random eleme: " + list.get(randomIndex));

```

Itt egy véletlen indexet választ a lista elemeiből, és kiírja az ehhez tartozó elemet.

### 6. Listakezelés - Üres lista ellenőrzése

Az üres listák kezelésére is van példa:

```java
if (lista2.size() > 0) {
    int maxIndex3 = lista2.size() - 1;
    int randomIndex3 = random.nextInt(maxIndex3 + 1);
    System.out.println("Random index: " + randomIndex3);
} else {
    System.out.println("Nincsenek elemek a listában!");
}

```

Ez a kódrészlet ellenőrzi, hogy van-e elem egy listában, és ha van, véletlenszerűen választ egyet, különben jelzi, hogy a lista üres.

### 7. Véletlenszerű számok hozzáadása egy listához

Többszörösen ismétlődő művelet véletlenszámokkal, ahol egy listába ad hozzá elemeket:

```java
List<Integer> listList = new ArrayList<>();
for (int i = 0; i < 5; i++) {
    int randomSzam = random.nextInt(1001);
    listList.add(randomSzam);
}
System.out.println(listList);

```

Itt a kód 5 véletlenszerű számot ad egy listához, amelyek 0 és 1000 között vannak.

### 8. Páros-páratlan ellenőrzés

Ez a ciklus ellenőrzi, hogy egy szám páros vagy páratlan:

```java
for (int i = 0; i < 10; i++) {
    if (i % 2 == 0) {
        System.out.println(i + "/2 0 maradékot ad");
    } else {
        System.out.println(i + "/2 1 maradékot ad.");
    }
}

```

A fenti ciklus 0-tól 9-ig iterál, és minden számnál ellenőrzi, hogy páros-e (maradéka 0), vagy páratlan.

### 9. Véletlenszerű páros számok egy másik listába

Ez a ciklus páros véletlenszerű számokat ad egy új listához:

```java
for (int i = 0; i < 20; i++) {
    int szalami = random.nextInt(1001);
    if (szalami % 2 == 0) {
        kecske.add(szalami);
    }
}

```

Csak akkor adja hozzá a számot a listához, ha páros.
