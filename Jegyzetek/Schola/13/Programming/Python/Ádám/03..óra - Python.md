### 1. **Autóra spórolás számítása**

Ez a függvény azt számolja ki, hogy hány nap alatt sikerül összegyűjteni egy adott összeget autóra spórolás közben.

#### **Magyarázat:**

- **Bemenő paraméterek:**
    
    - `ár`: az autó teljes ára
    - `tőke`: az a pénz, ami már megvan
    - `kezd`: az első nap félretett összeg
- **Fontos lépések:**
    
    - Kezdetben az `összeg` (spórolt pénz) és `napok` (eltelt napok száma) 0.
    - A ténylegesen szükséges pénzt (`ár -= tőke`) csökkentjük a meglévő tőkével.
    - Minden nap hozzáadjuk a napi félretett összeget (`i`), amely hetente növekszik.
    - Amint az összeg eléri a kívánt összeget, visszatérünk a napok számával.

### 2. **Prímtulajdonság vizsgálata (osztó)**

Ez a függvény azt vizsgálja, hogy egy számnak pontosan **egy nem triviális osztója van-e**.

#### **Magyarázat:**

- **Ha a szám 1, akkor False** (mert csak az 1 az osztója).
- A ciklus a `2`-tól indul és addig megy, amíg `i * i <= szám`.
- Vizsgálja, hogy a `szám` osztható-e `i`-vel vagy 10-zel.
- Ha már **kettőnél több** osztót talál, azonnal visszatér `False`-szal.

Ez gyakorlatilag a **félig optimalizált prímszámvizsgálat**.

### 3. **Számlista generálás adott határok között**

Ez a függvény egy **növekvő vagy csökkenő számlistát** generál. Lisák -> [[04.óra - Python]]

#### **Magyarázat:**

- Ha `kezd < vég`, akkor **növekvő** listát hozunk létre (`range(kezd, vég+1)`).
- Ha `kezd > vég`, akkor **csökkenő** listát generálunk (`range(kezd, vég-1, -1)`).

Példa:

```python
print(generáló(2,8))  # [2,3,4,5,6,7,8]
print(generáló(8,2))  # [8,7,6,5,4,3,2]
```

### 5. **Számjegyek megszámlálása**

Ez a függvény egy számjegyek számát adja vissza.

#### **Magyarázat:**

### 6. **Egyforma számjegyek vizsgálata**

Ez a függvény ellenőrzi, hogy egy szám minden számjegye **ugyanolyan-e**.

#### **Első módszer:**

- `for` ciklus végigmegy a számjegyeken.
- Ha **bármelyik számjegy eltér az elsőtől**, akkor `False`-t ad vissza.

#### **Második módszer (boolean lista használatával):**

- Létrehoz egy `boollista`-t, amely `True`-kat és `False`-okat tárol.
- Ha **van benne `False`**, akkor az egész hamis, egyébként igaz.

Példa:

```python
print(egyformajegyek(66))   # True
print(egyformajegyek(667))  # False
```

### 7. **Számjegyek összegének páros/páratlan volta**

Ez a függvény egy szám számjegyeinek összegét számolja ki, majd megmondja, hogy **páros vagy páratlan**.

#### **Magyarázat:**

- A számot **sztringgé alakítjuk**, így végigjárható `for` ciklussal.
- Az összes számjegyet **egyesével átalakítjuk egész számmá** és hozzáadjuk egy változóhoz.
- Az összeg **párossága vizsgálható `összeg % 2 == 0` kifejezéssel**.

Példa:

```python
print(jegyösszeg(67))  # "Páratlan" (6+7=13)
```

### 8. **Fibonacci sorozat adott eleme**

Ez a függvény a **Fibonacci-sorozat n-edik elemét** adja vissza.

#### **Magyarázat:**

- **Ha `n=0`, akkor `0` az eredmény**.
- **Ha `n=1`, akkor `1` az eredmény**.
- Egyébként a `t1` és `t2` változók **végigmennek a sorozaton** a kívánt elemig.

Példa:

```python
print(fib(20))  # 6765
```

```python
def jegyösszeg(szám):

def fib(szám):
    if szám == 0: #1. elem esetén
        return '0'
    elif szám == 1: #2. elem esetén
        return '1'
    else: #minden más elem esetén
        #A 2 legutóbbi elemnek kell 1-1 segédváltozó:
        t1 = 0 #legutóbbi előtti elem
        t2 = 1 #legutóbbi elem
        for i in range(2,szám+1):
            összeg = t1 + t2
            t1 = t2 #t1 értéke a legutóbbi elem legyen
            t2 = összeg #t2 értéke pedig az újonnan kapott összeg
    return összeg
```
