## 1. Alap OOP fogalmak

**Osztály (`class`)**
- Sablon/terv, ami leírja, hogy egy adott típusú objektumnak:
  - milyen adatai vannak (attribútumok),
  - milyen műveletei vannak (metódusok).
**Objektum (példány)**
- Egy konkrét "példány" az osztályból.
- Példányosítás: `obj = OsztalyNev(...)`
**Konstruktor (`__init__`)**
- Speciális metódus, ami akkor fut le, amikor az osztályból példányt hozunk létre.
- Itt hozzuk létre és inicializáljuk az attribútumokat.
- Mindig `self` az első paramétere.
**`self`**
- Az aktuális objektumra mutat.
- Így érjük el az attribútumokat: `self.nev`, `self.fizetes`, stb.
**Attribútum**
- Az objektum adatai (pl. név, fizetés, részleg).
**Metódus**
- Az osztályhoz tartozó függvény, ami az objektum adataival dolgozik.

## 2. `Alkalmazott` osztály

```python
class Alkalmazott:
    def __init__(self,nev,id,fizetes,reszleg):
        self.nev = nev
        self.id = id
        self.fizetes = fizetes
        self.reszleg = reszleg
```
### Cél

Egy alkalmazott adatait tárolja és kezeli:
- név
- azonosító (id)
- fizetés
- részleg
    

### Attribútumok

- `self.nev` – az alkalmazott neve
- `self.id` – az alkalmazott egyedi azonosítója
- `self.fizetes` – aktuális fizetés
- `self.reszleg` – melyik részlegen dolgozik

### Metódusok

#### 2.1. `fizuszámol`
```python
def fizuszámol(self,fizetes,ledolgozott_orak):
    tulora = 0
    if ledolgozott_orak > 50:
        tulora = ledolgozott_orak - 50
    self.fizetes = self.fizetes + (tulora * (self.fizetes/50))
    print(self.fizetes)
```
- Logika:
    - Ha a ledolgozott órák száma több mint 50, akkor a 50 feletti órák túlórának számítanak.
    - A túlórákra plusz fizetést kap:
        - `tulora * (self.fizetes / 50)` – itt az 50 óra alapján számít valamilyen túlóradíjat.
- A végén kiírja az új fizetést.
Megjegyzés: a metódus paraméterben kap egy `fizetes` változót is, de az nincs használva.
#### 2.2. `reszlegcsere`
```python
def reszlegcsere(self,alk_reszleg):
    self.reszleg = alk_reszleg
```
- A paraméterként kapott új részlegre állítja az alkalmazott részlegét.
#### 2.3. `adatkiir`
```python
def adatkiir(self):
    print("\nNév: ", self.nev)
    print("\nID: ", self.id)
    print("\nFizetés: ", self.fizetes)
    print("\nRészleg: ", self.reszleg)
    print("\n------------------------")
```
- Szépen, tagoltan kiírja az alkalmazott összes adatát.
### Példányosítás és használat

```python
alkalmazott1 = Alkalmazott("Adam",1,300000,"IT")
alkalmazott2 = Alkalmazott("Bela",2,330000,"HR")
alkalmazott3 = Alkalmazott("Csaba",3,340000,"Marketing")
alkalmazott4 = Alkalmazott("Dezso",4,400000,"Sales")

alkalmazott1.adatkiir()                       # Adatai kiírása
alkalmazott2.reszlegcsere("Vezetőség")       # Részlegcsere
alkalmazott2.adatkiir()
alkalmazott3.fizuszámol(340000,60)           # Fizetés számolása túlórával
```

## 3. `Etterem` osztály
```python
class Etterem:
    def __init__(self):
        self.menu_etelek = {}
        self.asztalfoglalas = []
        self.rendelesek = {}
```
### Cél
Egy étterem egyszerű rendszerét modellezi:
- étlap
- asztalfoglalások
- rendelések

### Attribútumok
- `self.menu_etelek` – szótár (dictionary), amely az étel nevét és árát tárolja:
    - kulcs: étel neve (pl. "Pizza")
    - érték: ár (pl. `1500`)
- `self.asztalfoglalas` – lista a lefoglalt asztalszámokról
- `self.rendelesek` – rendelések tárolására szánt szerkezet (a kódban még nincs rendesen feltöltve)
### Metódusok

#### 3.1. `menu_felvisz`
```python
def menu_felvisz(self,etel,ar):
    self.menu_etelek[etel] = ar
```
- Hozzáad egy új ételt az étlaphoz.
- Kulcs: `etel`, érték: `ar`.
#### 3.2. `foglalasok`
```python
def foglalasok(self,asztal_szam):
    self.asztalfoglalas.append(asztal_szam)
```
- Hozzáad egy asztalszámot a lefoglalt asztalok listájához.

#### 3.3. `etlap_kiir`
```python
def etlap_kiir(self):
    for etel, ar in self.menu_etelek.items():
        print("{}: {} forint".format(etel,ar))
```
- Végigmegy az étlapon és kiírja:
    - étel neve
    - ára forintban
#### 3.4. `asztalf_kiir`

```python
def asztalf_kiir(self):
    for asztal in self.asztalfoglalas:
        print("Asztak: {}".format(asztal))
```

- Kiírja az összes lefoglalt asztalt.
#### 3.5. `rendeles_kiir`

```python
def rendeles_kiir(self):
    for rendeles in self.rendelesek:
        print("Asztal {}: {}.".format(rendeles['asztal_szam'], rendeles['rendeles']))
```
- A kódban a megjegyzés alapján a rendelések listája szótárrá alakulna.
- Itt úgy kezelné, mintha `self.rendelesek` egy olyan szerkezet lenne, ahol minden elem egy szótár:
    - `{'asztal_szam': ..., 'rendeles': ...}`
Jelen állapotban a `rendelesek` nincs ténylegesen feltöltve, ez inkább váz.
### Példányosítás és használat
```python
etterem1 = Etterem()

etterem1.menu_felvisz("Hamburger",1000)
etterem1.menu_felvisz("Pizza",1500)
etterem1.menu_felvisz("Kóla",500)

etterem1.foglalasok(1)
etterem1.foglalasok(2)
etterem1.foglalasok(3)

etterem1.etlap_kiir()
```

## 4. `BankUgyfel` osztály
```python
class BankUgyfel:
    def __init__(self,szlaszam,nyitasdatum,egyenleg, nev):
        self.szlaszam =szlaszam
        self.nyitasdatum = nyitasdatum
        self.egyenleg = egyenleg
        self.nev = nev
```

### Cél
Egy banki ügyfél adatait és számla műveleteit kezeli:
- számlaszám
- számlanyitás dátuma
- egyenleg
- ügyfél neve

### Attribútumok
- `self.szlaszam` – számlaszám
- `self.nyitasdatum` – mikor nyitották a számlát
- `self.egyenleg` – jelenlegi egyenleg
- `self.nev` – ügyfél neve

### Metódusok
#### 4.1. `berak`
```python
def berak(self, osszeg):
    self.egyenleg += osszeg
    print(f"{osszeg} forint lett berakva.")
```
- Növeli az egyenleget a megadott összeggel.
- Kiírja, mennyi pénzt raktunk be.
#### 4.2. `kivesz`
```python
def kivesz(self,osszeg):
    if osszeg > self.egyenleg:
        print("Nincs elég pénzed ehhez.")
    else:
        self.egyenleg -= osszeg
        print(f"{osszeg} forint lett kivéve.")
```
- Ha több pénzt akar kivenni, mint amennyi van:
    - hibaüzenetet ír ki.
- Különben levonja az összeget az egyenlegből.

#### 4.3. `lekerdez`
```python
def lekerdez(self):
    print(f"Jelenlegi egyenleg: {self.egyenleg} forint.")
```
- Kiírja az aktuális egyenleget.

#### 4.4. `reszletezes`
```python
def reszletezes(self):
    print("\nNév: ", self.nev)
    print("\nSzámlaszám: ", self.szlaszam)
    print("\nNyitás dátuma: ", self.nyitasdatum)
    print("\nEgyenleg: ", self.egyenleg, " forint")
```
- Kiírja az ügyfél összes adatát, külön sorokban.
### Példányosítás és használat
```python
ugyfel1 = BankUgyfel(1234,"01-01-2024",1000000,"Ernő")
ugyfel1.lekerdez()
ugyfel1.reszletezes()
ugyfel1.berak(10000)
ugyfel1.kivesz(2000000)
```

## 5. `Leltar` osztály
```python
class Leltar:
    def __init__(self):
        self.leltar = {}
```
### Cél

Egy raktár/leltár egyszerű kezelésére szolgál:
- termékek tárolása azonosítóval
- darabszám és ár nyilvántartása
- adatok frissítése és lekérdezése
### Attribútum

- `self.leltar` – szótár, ahol:
    - kulcs: `elem_id` (pl. "I001")
    - érték: egy másik szótár a termék adataival:
        - `"Név"`
        - `"Darabszám"`
        - `"Ár"`

Példa belső szerkezetre:
```python
self.leltar = {
    "I001": {
        "Név": "Laptop",
        "Darabszám": 100,
        "Ár": 200000
    }
}
```
### Metódusok
#### 5.1. `hozzaad`
```python
def hozzaad(self,elem_id,megnevezes,darabszam,ar):
    self.leltar[elem_id] = {"Név":megnevezes, "Darabszám":darabszam, "Ár":ar}
```
- Új terméket vesz fel a leltárba.
- Az `elem_id` alapján hoz létre egy bejegyzést.

#### 5.2. `frissit`
```python
def frissit(self,elem_id, darabszam, ar):
    if elem_id in self.leltar:
        self.leltar[elem_id]["Darabszám"] = darabszam
        self.leltar[elem_id]["Ár"] = ar
    else:
        print("Nincs ilyen termék a leltárban.")
```

- Ha létezik a megadott azonosító:
    - módosítja a darabszámot és az árat.
- Ha nem létezik:
    - hibaüzenetet ír ki.

#### 5.3. `megnez`
```python
def megnez(self,elem_id):
    if elem_id in self.leltar:
        elem = self.leltar[elem_id]
        return f"Terméknév: {elem['Név']}, Darabszám: {elem['Darabszám']}, Ár: {elem['Ár']}"
    else:
        print("Nincs ilyen elem.")
```

(Az eredeti kódban az idézőjelek miatt hibázna, itt a javított verzió látható.)
- Ha létezik az adott `elem_id`:
    - visszaad egy formázott szöveget a termék adataival.
- Ha nem:
    - hibaüzenetet ír ki.
### Példányosítás és használat
```python
leltar1 = Leltar()
leltar1.hozzaad("I001","Laptop",100,200000)
print(leltar1.frissit("I001",80,150000))
print(leltar1.megnez("I001"))
```

## 6. Összefoglalás
A fenti osztályok mind ugyanazt a mintát követik:
- Van egy **osztály**, ami egy valós fogalmat modellez:
    - `Alkalmazott`, `Etterem`, `BankUgyfel`, `Leltar`.
- A konstruktorban (`__init__`) létrejönnek az **attribútumok**.
- A **metódusok**:
    - vagy módosítják az állapotot (pl. `berak`, `kivesz`, `frissit`, `reszlegcsere`),
    - vagy kiírnak / visszaadnak adatokat (`adatkiir`, `etlap_kiir`, `lekerdez`, `megnez`),
    - vagy számolást végeznek (`fizuszámol`).

    def reszlegcsere(self,alk_reszleg):
        self.reszleg = alk_reszleg

    #Egy segédmetúdus írja ki az alkalmazott adatait, de minden infót külön sorba:
    def adatkiir(self):

'''
Egy leltárhoz készítünk határozatlan tartalmú osztályt. 
3 segédfüggvény lesz: leltárbővítés, meglévő elem módosítása és a termékek listázása.

class Leltar:
    def __init__(self):
        self.leltar = {} #létrehoz egy üres szótártömböt
    #Utólag rakjuk bele a tartalmat: id, megnevezés, darabszám és ár (+a konkrét értékek)

    def hozzaad(self,elem_id,megnevezes,darabszam,ar):

    def frissit(self,elem_id, darabszam, ar):

            self.leltar[elem_id]["Darabszám"] = darabszam
            self.leltar[elem_id]["Ár"] = ar
        else: #Ha nem találta az id alapján:
            print("Nincs ilyen termék a leltárban.")
    #Segédmetódus, ami listázza 1 db termék adatait:
    def megnez(self,elem_id):
        #Ha van ilyen id-jű elem:
        if elem_id in self.leltar:
            elem = self.leltar[elem_id]
            return f"Terméknév: {elem["Név"]}, Darabszám: {elem["Darabszám"]}, Ár: {elem["Ár"]}"
        else:
            print("Nincs ilyen elem.")
