### 1. **`mondatkiír(mondat)`** – CamelCase-ből szóközökkel elválasztott mondatot csinál

```python
def mondatkiír(mondat):
    válasz = ''
    for betű in mondat:
        if betű.isupper():
            válasz += ' ' + betű.lower()
        else:
            válasz += betű
    return válasz
```

**Példa:**
```python
mondatkiír("ezEgyPéldaMondat")  # → "ez egy példa mondat"
```

### 2. **`cenzura(mondat)`** – Cenzúrázza a négy betűnél hosszabb szavakat `*`-okkal
```python
def cenzura(mondat):
    for szo in mondat.split():
        if len(szo) > 4:
            mondat = mondat.replace(szo, "*" * len(szo))
    return mondat
```

**Példa:**
```python
cenzura("Ez egy példa mondat.")  # → "Ez egy ***** *******"
```

### 3. **`pénzes(régi, új)`** – Kiszámolja a pénzértékek változását százalékban

```python
def pénzes(régi, új):
    régi = int(régi[1:])  # Az első karakter levágása ($)
    új = int(új[1:])
    százalék = int(((új - régi) / régi) * 100)
    if százalék < 0:
        százalék = -százalék
        return "{}%-os csökkenés.".format(százalék)
    else:
        return "{}%-os növekedés.".format(százalék)
```

**Példa:**
```python
pénzes("$800", "$600")  # → "25%-os csökkenés."
pénzes("$400", "$700")  # → "75%-os növekedés."
```

### 4. **`közös(szo1, szo2)`** – Közös betűk egy stringben

```python
def közös(szo1, szo2):
    lista = [betu for betu in szo1.lower() if betu in szo2.lower()]
    return ''.join(sorted(set(lista)))
```
**Példa:**
```python
közös("háló", "faház")  # → "áh"
```

### 5. **`igazak(egyenletlista)`** – Csak az igaz matematikai egyenleteket adja vissza

```python
def igazak(egyenletlista):
    megoldás = []
    for egyenlet in egyenletlista:
        a, b = egyenlet.split('=')
        if eval(a) == int(b):
            megoldás.append(egyenlet)
    return megoldás
```

🔵 **Példa:**

```python
igazak(["1+1=2", "2+2=4", "3+3=9"])  # → ["1+1=2", "2+2=4"]
```

⚠️ **Megjegyzés:** Az `eval()` biztonsági kockázatot jelenthet.

### 6. **`átlagbetűk(mondat)`** – Kiszámolja az átlagos szóhosszúságot

```python
def átlagbetűk(mondat):
    betűszám = 0
    for betű in mondat:
        if not betű in ":.-! ?":
            betűszám += 1
    return round((betűszám / (mondat.count(" ") + 1)), 2)
```

🔵 **Példa:**

```python
átlagbetűk("Ez egy példa mondat.")  # → 4.00
```
