## **1. Sorozatszámítás (Általánosítás)**

- **Lényege:** Egy sorozat elemein végzünk valamilyen műveletet (pl. összeadás, szorzás).
- **Általános forma:** Minden elemhez hozzárendelünk egy értéket, majd az eredményeket egyetlen értékké alakítjuk (pl. összeg, szorzat).

**Példa (Összegzés):**

- Adott egy lista, pl.: `[1, 2, 3, 4]`
- Az összegzés eredménye: `1 + 2 + 3 + 4 = 10`

**Pszeudokóddal:**

```
Sorozatszámítás:
    Változó i:Egész
    S:=F0
    Ciklus i=1-től N-ig
        S:=f(S,X[i])
    Ciklus vége
Eljárás vége.
```

**Python példa (for ciklussal):**

```python
lista = [1, 2, 3, 4]
szum = 0
for i in range(len(lista)):
    szum += lista[i]
print("Összeg:", szum)

```

**Python példa (while ciklussal):**

```python
lista = [1, 2, 3, 4]
szum = 0
i = 0
while i < len(lista):
    szum += lista[i]
    i += 1
print("Összeg:", szum)

```

## **2. Megszámolás**

- **Lényege:** Megszámoljuk, hogy hány elem felel meg egy feltételnek.
- **Példa:** Hány szám nagyobb, mint 2?
    - Lista: `[1, 2, 3, 4]` → 2 szám (3, 4)

**Pszeudokóddal:**

```
Megszámolás:
    Változó
        i:Egész
        Db:=0
    Ciklus i=1-től N-ig
        Ha T(X[i]) akkor Db:=Db+1
    Ciklus vége
Eljárás vége.
```

**Python példa (for ciklussal):**

```python
lista = [1, 2, 3, 4]
db = 0
for i in range(len(lista)):
    if lista[i] > 2:
        db += 1
print("Megfelelő elemek száma:", db)

```

**Python példa (while ciklussal):**

```python
lista = [1, 2, 3, 4]
db = 0
i = 0
while i < len(lista):
    if lista[i] > 2:
        db += 1
    i += 1
print("Megfelelő elemek száma:", db)

```

## **3. Minimum/Maximum Kiválasztás**

- **Lényege:** Megkeressük a legnagyobb vagy legkisebb értéket a sorozatban.
- **Példa:** Legnagyobb érték a listában `[1, 7, 3, 4]` → 7

**Pszeudokóddal:**

```
Megszámolás:
    Változó
        i:Egész
        max_index:=1
    Ciklus i=2-től N-ig
        Ha X[i] > X[max_index]  akkor max_index:=i
    Ciklus vége
Eljárás vége.
```

**Python példa (for ciklussal, maximum):**

```python
lista = [1, 7, 3, 4]
max_index = 0
for i in range(1, len(lista)):
    if lista[i] > lista[max_index]:
        max_index = i
print("Maximum érték:", lista[max_index])

```

**Python példa (while ciklussal, maximum):**

```python
lista = [1, 7, 3, 4]
max_index = 0
i = 1
while i < len(lista):
    if lista[i] > lista[max_index]:
        max_index = i
    i += 1
print("Maximum érték:", lista[max_index])

```

## **4. Eldöntés (Van-e ilyen elem?)**

- **Lényege:** Eldöntjük, hogy van-e olyan elem, amely megfelel egy adott feltételnek.
- **Példa:** Van-e a listában `5`? `[1, 2, 3, 4]` → Nincs

**Pszeudokóddal:**

```
Eldöntés:
    Változó
        i:Egész
        i:=1
    Ciklus amíg i≤N és nem T(X[i])
        i:=i+1
    Ciklus vége
    Van:=i≤N
Eljárás vége.
```

**Python példa:**

```python
lista = [1, 2, 3, 4]
i = 0
while i < len(lista) and lista[i] != 5:
    i += 1
van = i < len(lista)
print("Van 5-ös a listában?", van)

```

## **5. Kiválasztás (Hol található a keresett elem?)**

- **Lényege:** Megkeressük, hogy hol (melyik indexen) van egy adott feltételnek megfelelő elem.
- **Példa:** Melyik indexen van `3`? `[1, 2, 3, 4]` → Index: 2

**Pszeudokóddal:**

```
Kiválasztás:
    i:=1
    Ciklus amíg nem T(X[i])
        i:=i+1
    Ciklus vége
    Ind:=i; Ért:=X[i]
Eljárás vége.
```

**Python példa:**

```python
lista = [1, 2, 3, 4]
i = 0
while lista[i] != 3:
    i += 1
print("3-as indexe:", i)

```

## 🔍 **6. Keresés (Van-e, és ha igen, hol?)**

- **Lényege:** Megkeressük, hogy van-e adott feltételnek megfelelő elem, és ha van, akkor hol található.
- **Példa:** Van-e a listában `4`? Hol található? `[1, 2, 3, 4]` → Igen, Index: 3

```
Keresés:
    Változó i:Egész
        i:=1
    Ciklus amíg i≤N és nem T(X[i])
        i:=i+1
    Ciklus vége
    Van:=i≤N
    Ha Van akkor Ind:=i; Ért:=X[i]
Eljárás vége.
```

**Python példa:**

```python
lista = [1, 2, 3, 4]
i = 0
while i < len(lista) and lista[i] != 4:
    i += 1
van = i < len(lista)
if van:
    print("Van 4-es, indexe:", i)

```
