### 1. Feladat: Átlag számítása

Készíts egy programot, amely:

1. Kéri a felhasználótól, hogy hány számot szeretne megadni.
2. Egyenként bekéri ezeket a számokat.
3. Kiszámítja és kiírja ezeknek a számoknak az átlagát.

### Példa kimenet:

```python
Hány számot szeretnél megadni? 3
Add meg az 1. számot: 4
Add meg a 2. számot: 6
Add meg a 3. számot: 8

Az átlag: 6.0

```

### Kód:

```python
# Átlag számítása

szamok_szama = int(input("Hány számot szeretnél megadni? ")) # bekérjük a felhasználótól, hogy meddig menjen el a lista

szamok = [] # lista létrehozása

for i in range(szamok_szama): # a for ciklus csak addig megy el, amennyit a felhasználó megadott
    szam = int(input(f"Add meg az {i+1}. számot: ")) # bekérjük a számokat
    szamok.append(szam) # belerakjuk a listába a számokat

atlag = sum(szamok) / len(szamok) # átlagszámítás

print(f"\\nAz átlag: {atlag}") # kiíratjuk az átlagot

```

### 2. Feladat: Páros számok kiszűrése

Készíts egy programot, amely:

4. Kéri a felhasználótól, hogy hány számot szeretne megadni.
5. Egyenként bekéri ezeket a számokat.
6. Kiírja a páros számokat a listából.

### Példa kimenet:

```
Hány számot szeretnél megadni? 5
Add meg az 1. számot: 4
Add meg a 2. számot: 7
Add meg a 3. számot: 2
Add meg a 4. számot: 9
Add meg az 5. számot: 6

A páros számok: [4, 2, 6]

```

### Kód:

```python
# Páros számok kiszűrése

szamok_szama = int(input("Hány számot szeretnél megadni? "))

szamok = []

for i in range(szamok_szama):
    szam = int(input(f"Add meg az {i+1}. számot: "))
    szamok.append(szam)

paros_szamok = [szam for szam in szamok if szam % 2 == 0]

print("\\nA páros számok:", paros_szamok)

### 4. Feladat: Legkisebb szám keresése

11. Egyenként bekéri ezeket a számokat.

```
Hány számot szeretnél megadni? 3

```

```
