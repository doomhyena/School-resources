### **2. Első program – "Hello, World!" (10 perc)**

### **Magyarázat**

"Minden programozási nyelvben az első lépés egy egyszerű 'Hello, World!' program írása. Ez segít abban, hogy lássuk, hogyan működik egy C# program."

### **Fontos elemek:**

- `using System;` – az alapvető függvénykönyvtárak használatához szükséges.
- `class Program {}` – a program fő osztálya.
- `static void Main()` – a program belépési pontja.
- `Console.WriteLine("Hello, World!");` – kiírja a szöveget a konzolra.

```csharp
using System;

class Program
{
    static void Main()
    {
        Console.WriteLine("Hello, World!");
    }
}

```

**Kérdések:**

- Mit gondoltok, miért fontos a `Main()` metódus?
- Mi lehet a `Console.WriteLine()` szerepe?

### **3. Alapvető szintaxis és adattípusok (30 perc)**

### **Magyarázat**

"A programokban adatokkal dolgozunk, ezért fontos megérteni a különböző adattípusokat."

### **Gyakran használt adattípusok:**

- `int` – Egész számok tárolására.
- `double` – Lebegőpontos számokhoz.
- `bool` – Igaz vagy hamis értékek tárolására.
- `string` – Szöveges adatokhoz.
- `char` – Egyetlen karakter tárolására.

**Példa**

```csharp
int age = 25;
double price = 19.99;
bool isRaining = false;
string name = "John";
char letter = 'A';

```

**Feladat:** "Hozzatok létre egy változót, amely tartalmazza a neveteket, és írassátok ki!"

### **4. Feltételek és ciklusok (30 perc)**

### **If-else szerkezet**

"A programok gyakran döntéseket hoznak a felhasználói input alapján. Erre szolgál az `if-else`."

### **If-else szerkezet működése:**

- A feltétel igaz (`true`) esetén végrehajt egy blokkot.
- Ha a feltétel hamis (`false`), akkor egy másik blokkot hajt végre.

**Példa:**

```csharp
int age = 18;
if (age >= 18)
{
    Console.WriteLine("Felnőtt vagy.");
}
else
{
    Console.WriteLine("Még kiskorú vagy.");
}

```

**Feladat:** "Kérjétek be a felhasználó életkorát, és mondjátok meg, hogy felnőtt-e!"

## **Második alkalom (1.5 óra) – Objektumorientált programozás és fájlkezelés**

### **1. Függvények és metódusok (20 perc)**

"A nagyobb programokat kisebb, logikus egységekre bontjuk – ezért használunk metódusokat."

### **Miért hasznosak a metódusok?**

- Kód újrafelhasználhatóság.
- Jobb olvashatóság.
- Kisebb hibalehetőség.

**Példa:**

```csharp
static int Osszeadas(int a, int b)
{
    return a + b;
}

```

**Feladat:** "Írjatok egy metódust, amely két számot összead, és visszaadja az eredményt!"

---

### **2. Osztályok és objektumok (40 perc)**

"A C# objektumorientált nyelv, ami azt jelenti, hogy osztályokban és objektumokban gondolkodunk."

### **OOP alapelemei:**

- **Osztályok** – Adatok és metódusok összekapcsolása.
- **Objektumok** – Az osztály példányai.

**Példa:**

```csharp
class Car
{
    public string Brand;
    public int Year;
}

```

**Feladat:** "Hozzatok létre egy `Car` osztályt, és példányosítsatok egy autót!"

### **3. Fájlkezelés (20 perc)**

"A programok gyakran dolgoznak fájlokkal, így most megnézzük, hogyan tudunk adatokat menteni és beolvasni."
