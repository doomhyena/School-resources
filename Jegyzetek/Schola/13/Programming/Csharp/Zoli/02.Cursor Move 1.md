### **Fő változók**

- **`x, y, xOld, yOld`**: A kurzor aktuális és előző pozíciójának koordinátái.
- **`step`**: Az a lépésszám, amit a kurzor az adott irányba mozog, mielőtt irányt változtatna.
- **`dir`**: A kurzor aktuális iránya (`Left`, `Right`, `Up`, `Down`).
- **`dirs`**: Az összes lehetséges irány tömbje.
- **`width`, `height`**: A keret szélessége és magassága.
- **`offsetX`, `offsetY`**: A konzol középpontjának eltolása, hogy a keret középen legyen.

---

### **1. Program inicializálása**

```cs
width = 50;
height = 25;
dir = Direction.Right;
x = xOld = width / 2;
y = yOld = height / 2;
step = rnd.Next(2, 9);
offsetX = Console.WindowWidth / 2 - x;
offsetY = Console.WindowHeight / 2 - y;
```

- A keret mérete 50x25.
- A kurzor a keret közepén indul.
- A `step` változó véletlenszerű értéket kap 2 és 8 között.
- Az eltolás biztosítja, hogy a keret a konzolablak közepére kerüljön.

---

### **2. Kurzor mozgatása**

A mozgás ciklusa a következő lépéseket tartalmazza:

1. A régi pozíció elmentése.
2. Az irány megváltoztatása, ha a `step` 0-ra csökken.
3. Az új pozíció kiszámítása.
4. A kurzor kirajzolása az új pozícióban zöld színnel.
5. A régi pozíció törlése (fekete színnel).
6. Rövid várakozás a következő lépés előtt.

---

### **3. Kurzor irányának megváltoztatása**

```cs
private static void ChangeDirection()
{
	if (step == 0)
	{
		dir = dirs[rnd.Next(dirs.Length)];
		step = rnd.Next(2, 9);
	}
	else {
		step--;
	}
}
```

- Ha a lépések elfogynak (`step == 0`), a program véletlenszerűen új irányt választ.
- Az új lépésszám ismét véletlenszerű értéket kap.

---

### **4. Kurzor új pozíciójának kiszámítása**

```cs
private static void PositionCalculation()
{
	switch (dir)
	{
		case Direction.Left:
			if (x > 1) x--;
			else dir = dirs[rnd.Next(dirs.Length)];
			break;
		case Direction.Right:
			if (x < width - 2) x++;
			else dir = dirs[rnd.Next(dirs.Length)];
			break;
		case Direction.Up:
			if (y > 1) y--;
			else dir = dirs[rnd.Next(dirs.Length)];
			break;
		case Direction.Down:
			if (y < height - 2) y++;
			else dir = dirs[rnd.Next(dirs.Length)];
			break;
	}
}
```

- A kurzor az aktuális irányba mozog.
- Ha a kurzor eléri a keret szélét, véletlenszerűen új irányt választ.

---

### **5. Keret kirajzolása**

```cs
private static void DrawRectangle(int width, int height)
{
	for (int y = 0; y < height; y++)
	{
		for (int x = 0; x < width; x++)
		{
			if (x == 0 || y == 0 || x == width - 1 || y == height - 1)
			{
				Draw(x, y, '█', ConsoleColor.Red);
			}
		}
	}
}
```

- A keret piros színnel van kirajzolva, és a határait `█` karakterekkel jelöli.

---

### **6. Rajzolás a megadott pozícióban**

```cs
private static void Draw(int x, int y, char c, ConsoleColor color)
{
	Console.SetCursorPosition(x + offsetX, y + offsetY);
	Console.ForegroundColor = color;
	Console.Write(c);
}
```

- A kurzort az eltolással korrigált pozícióba helyezi.
- A `c` karaktert az adott színnel rajzolja ki.

---

### **7. Képernyőfrissítés és kilépés**

A program addig fut, amíg a `exit` változó `false` marad. Ez biztosítja a végtelen mozgást, amíg a felhasználó kézzel le nem állítja a futtatást.

---

```cs
﻿using System;
using System.Collections.Generic;
using System.Linq;
using System.Text;
using System.Threading.Tasks;

namespace CursorMove
{
	internal class Program
	{
		static Random rnd = new Random();
		static int x, y, step, xOld, yOld;
		static Direction dir;
		static Direction[] dirs = new Direction[] { Direction.Up, Direction.Down, Direction.Left, Direction.Right };
		static int offsetX, offsetY;
		static int width, height;
		static bool exit = false;

		static void Main(string[] args)
		{
			Console.CursorVisible = false;

			//---------------------------------------
			width = 50;
			height = 25;
			dir = Direction.Right;
			x = xOld = width / 2;
			y = yOld = height / 2;
			step = rnd.Next(2, 9);
			offsetX = Console.WindowWidth / 2 - x;
			offsetY = Console.WindowHeight / 2 - y;

			Draw(x, y, '█', ConsoleColor.Green);
			DrawRectangle(width, height);
			do
			{
				// régi pozíció mentése
				xOld = x;
				yOld = y;
				// Irány változtatás
				ChangeDirection();
				//új pozíció számítás
				PositionCalculation();
				//várakoztatás
				System.Threading.Thread.Sleep(50);
				// kirajzolás
				Draw(x, y, '█', ConsoleColor.Green);
				// törlés
				Draw(xOld, yOld, ' ', ConsoleColor.Black);
				/*
				 * 1. kirajzolni a kurzort
				 * 2. letörölni a régi helyről
				 * 3. kitalálni az új koordinátát
				 * 4. várakoztatni
				 */
			} while (!exit);
		}

		private static void ChangeDirection()
		{
			if (step == 0)
			{
				dir = dirs[rnd.Next(dirs.Length)];
				step = rnd.Next(2, 9);
			}
			else {
				step--;
			}

		}
		private static void DrawRectangle(int width, int height)
		{
			for (int y = 0; y < height; y++)
			{
				for (int x = 0; x < width; x++)
				{
					if (x == 0 || y == 0 || x == width - 1 || y == height - 1)
					{
						Draw(x, y, '█', ConsoleColor.Red);
					}
				}
			}
		}
		private static void PositionCalculation()
		{
			switch (dir)
			{
				case Direction.Left:
					if (x > 1)
					{
						xOld = x;
						x--;
					}
					else
					{
						dir = dirs[rnd.Next(dirs.Length)]; // új irány 
					}
					break;
				case Direction.Right:
					if (x < width - 2)
					{
						xOld = x;
						x++;
					}
					else
					{
						dir = dirs[rnd.Next(dirs.Length)]; // új irány 
					}
					break;
				case Direction.Up:
					if (y > 1)
					{
						yOld = y;
						y--;
					}
					else
					{
						dir = dirs[rnd.Next(dirs.Length)]; // új irány 
					}
					break;
				case Direction.Down:
					if (y < height - 2)
					{
						yOld = y;
						y++;
					}
					else
					{
						dir = dirs[rnd.Next(dirs.Length)]; // új irány 
					}
					break;
				default:
					break;
			}
			//if (step == 0)

			//}

			//	step--;
			//}
		}
		private static void Draw(int x, int y, char c, ConsoleColor color)
		{
			Console.SetCursorPosition(x + offsetX, y + offsetY);
			Console.ForegroundColor = color;
			Console.Write(c);
		}
	}

	public enum Direction { Left, Right, Up, Down }
}
```
