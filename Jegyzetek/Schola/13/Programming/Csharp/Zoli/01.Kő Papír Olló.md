### **1. Lehetőségek és véletlenszerű választás**

```cs
string[] lehetosegek = new string[] { "kő", "papír", "olló" };
string gepValasztas = lehetosegek[rnd.Next(lehetosegek.Length)];
```

- A `lehetosegek` tömb tárolja a játék lehetséges opcióit: "kő", "papír", "olló".
- A `rnd.Next(lehetosegek.Length)` véletlenszerűen kiválaszt egy számot 0 és 2 között (a tömb indexei alapján), és ezt a számítógép választására használja.

---

### **2. Játékos választásának bekérése**

```cs
Console.Write("Választásod [kő/papír/olló]: ");
string jatekosValasztas = Console.ReadLine();
```

- A program bekéri a játékos választását a konzolon keresztül.

---

### **3. Eredmény kiértékelése**

A program három esetet kezel:

#### **a. Döntetlen**

```cs
if (gepValasztas == jatekosValasztas)
{
	Console.WriteLine("Döntetlen.");
	Console.WriteLine($"A gép választása: {gepValasztas}");
}
```

- Ha a játékos választása megegyezik a gép választásával, döntetlen az eredmény.

---

#### **b. Játékos győzelme**

```cs
else if (gepValasztas == "kő" && jatekosValasztas == "papír" ||
		 gepValasztas == "papír" && jatekosValasztas == "olló" ||
		 gepValasztas == "olló" && jatekosValasztas == "kő")
{
	Console.WriteLine("Nyertél");
	Console.WriteLine($"A gép választása: {gepValasztas}");
}
```

- A játékos akkor nyer, ha:
    - A gép "kő"-t választ, és a játékos "papírt".
    - A gép "papírt" választ, és a játékos "ollót".
    - A gép "ollót" választ, és a játékos "kő"-t.

---

#### **c. Gép győzelme**

```cs
else if (jatekosValasztas == "kő" && gepValasztas == "papír" ||
		 jatekosValasztas == "papír" && gepValasztas == "olló" ||
		 jatekosValasztas == "olló" && gepValasztas == "kő")
{
	Console.WriteLine("Vesztettél");
	Console.WriteLine($"A gép választása: {gepValasztas}");
}
```

- A játékos akkor veszít, ha:
    - A játékos "kő"-t választ, és a gép "papírt".
    - A játékos "papírt" választ, és a gép "ollót".
    - A játékos "ollót" választ, és a gép "kő"-t.

---

#### **d. Érvénytelen választás**

```cs
else
{
	Console.WriteLine("Értékelhetetlen választás.");
}
```

- Ha a játékos nem "kő", "papír" vagy "olló" értéket ad meg, a program hibás választást jelez.

---

### **4. Program vége**

```cs
Console.WriteLine("- VÉGE -");
Console.ReadKey();
```

- A program végeztével kiírja a "- VÉGE -" szöveget, majd vár a felhasználó billentyűleütésére, hogy bezáruljon.
---
```cs
﻿using System;
using System.Collections.Generic;
using System.Linq;
using System.Text;
using System.Threading.Tasks;

namespace ElsoApp
{
	internal class Program
	{
		static Random rnd = new Random();
		static void Main(string[] args)
		{
			string[] lehetosegek = new string[] { "kő", "papír", "olló" };

			string gepValasztas = lehetosegek[rnd.Next(lehetosegek.Length)];

			Console.Write("Választásod [kő/papír/olló]: ");
			string jatekosValasztas = Console.ReadLine();

			if (gepValasztas == jatekosValasztas)
			{
				Console.WriteLine("Döntetlen.");
				Console.WriteLine($"A gép választása: {gepValasztas}");
			}
			else if (gepValasztas == "kő" && jatekosValasztas == "papír" ||
						gepValasztas == "papír" && jatekosValasztas == "olló" ||
						gepValasztas == "olló" && jatekosValasztas == "kő")
			{
				Console.WriteLine("Nyertél");
				Console.WriteLine($"A gép választása: {gepValasztas}");
			}
			else if (jatekosValasztas == "kő" && gepValasztas == "papír" ||
						jatekosValasztas == "papír" && gepValasztas == "olló" ||
						jatekosValasztas == "olló" && gepValasztas == "kő"
				)
			{
				Console.WriteLine("Vesztettél");
				Console.WriteLine($"A gép választása: {gepValasztas}");
			}
			else
			{
				Console.WriteLine("Értékelhetetlen választás.");
			}
			//----------------------------------------
			Console.WriteLine("- VÉGE -");
			Console.ReadKey();
		}
	}
}
```
