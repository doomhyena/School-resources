Tábla létrehozása

```sql
CREATE TABLE Diak (
	id int PRIMARY KEY, 
    nev varchar(255),
    evfolyam int, 
    betu varchar(255),
    varos varchar(255)
);
```

1. Feladat

```sql
SELECT varos nev FROM Diak;
```

2. Feladat

```sql
SELECT * 
FROM Diak 
WHERE evfolyam = 10 AND betu = 'D';
UNION
SELECT * Diak 
WHERE evfolyam = 9 AND betu = 'A';
```

3. Feladat

```sql
SELECT * 
FROM Diak 
WHERE id BETWEEN 101 AND 110;
SELECT * FROM Diak WHERE id >= 101 AND id <= 110;
```
