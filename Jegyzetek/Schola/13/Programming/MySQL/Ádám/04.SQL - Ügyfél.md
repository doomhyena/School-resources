1. Listázz minden rekordot a '21' és '22' kódú, valamint a Londoni tartózkodási helyű ügyfeleket leszámítva.

```sql
SELECT * 
FROM ugyfelek 
WHERE kod NOT IN ('21', '22') AND tartozkodasi_hely != 'London';
```

2. Hány emberünk van az Egyesült államokban?

```sql
SELECT COUNT(*) AS ugyfelek_szama
FROM ugyfelek 
WHERE orszag = 'Egyesült Államok';
```

3. Összesen mennyi pénzt vettek ki a nem Kanadai ügyfeflek?

```sql
SELECT SUM(kivet) AS osszes_kivet
FROM ugyfelek 
WHERE orszag != 'Kanada';
```

4. Ki fizette be a legtöbb pénzt?

```sql
SELECT nev, MAX(befizet) AS legtobb_befizetett
FROM ugyfelek;
```

5. Ki vette ki a legtöbb pénzt a 10-nél nagyobb ID-jű ügyfelek közül?

```sql
SELECT nev, MAX(kivet) AS legtobb_kivet
FROM ugyfelek 
WHERE id > 10;
```

6. Hány olyan név van, amelyben 'o' a második betű?

```sql
SELECT COUNT(*) AS nevek_szama
FROM ugyfelek 
WHERE nev LIKE '_o%';
```

7. Listázd ki a 4 betűs nevű emberek országait.

```sql
SELECT orszag 
FROM ugyfelek 
WHERE CHAR_LENGTH(nev) = 4;
```

8. Kik azok az angolok, akik eddig több, mint 10 milliót raktak be?

```sql
SELECT nev 
FROM ugyfelek 
WHERE orszag = 'Anglia' AND befizet > 10000000;
```

9. Összesen hány ügyfelünk van itt nyilvántartva?

```sql
SELECT COUNT(*) AS ugyfelek_szama
FROM ugyfelek;
```

10. Törlődjenek azok az ügyfelek, akiket a 10-es menedzser kezel, vagy 10 milliónál többet vettek ki.

```sql
DELETE 
FROM ugyfelek 
WHERE kezelo_menedzser_id = 10 OR kivet > 10000000;
```
