```sql
CREATE TABLE Nobel (
    ev int,
    tema varchar(255),
    nyertes varchar(225)
```

Tábla megjelenítése:

```sql
SELECT * FROM Nobel; --Mindnent IS megjelenít
```

1.feladat: Csak az 1950-es rekordok kellenek:

```sql
SELECT ev, tema, nyertes 
FROM Nobel WHERE ev = 1950;
```

2.feladat: Ki nyert irodalmi nobel díjat 1962-bne?

```sql
SELECT nyertes 
FROM Nobel
WHERE ev = 1962 AND tema = 'Irodalom';
```

3.feladat: Albert Einstein mikor és miért kapott díjat?

```sql
SELECT ev, tema
FROM Nobel
WHERE nyertes = 'Albert Einstein';
```

4.feladat: 1978-től nézve kik nyertek béke nobeldíjat?

```sql
SELECT nyertes 
FROM Nobel 
WHERE ev >= 1970 AND tema = 'Béke';
```

5.feladat: Minden rekord kell az 1970 és 78 közti irodalmi díját kapottakról!

```sql
SELECT * 
FROM Nobel 
WHERE ev BETWEEN 1970 AND 1978 AND kategoria = 'Irodalom';
```

6.feladat: Mi lehet tudni az elnökökről? (Jimmy Carter és John Kennedy)

```sql
SELECT *
FROM Nobel 
WHERE nyertes IN ('Jimmy Carter','John Kennedy');
```

7.feladat: Listázzuk ki a ‘John’ nevű embereket

```sql
SELECT nyertes 
FROM Nobel 
WHERE nyertes LIKE 'John%';
```

8.feladat: Listázzuk az 1980-as Fizikai és az 1984-es Kémiai nobeldíjasok rekordjait.

```sql
SELECT *
FROM Nobel 
WHERE tema = 'Fizika' AND ev = 1980
UNION 
SELECT * 
FROM Nobel WHERE  tema = 'Kémia' AND ev = 1984;
```

9.feladat: Minden érdekel, ami nem kémia vagy fizika 1980 legyen

```sql
SELECT * 
FROM Nobel 
WHERE ev = 1980 
AND tema NOT IN ('Kémia', 'Fizika');
```

10.feladat: Jelenlenek meg a 1960 előtti nyertek Fizikából és akik 1990 után nyertek Irodalomból (Union lekérdezés)

```sql
SELECT * 
FROM Nobel 
WHERE ev < 1960 
AND tema = 'Fizika'
UNION
SELECT * 
FROM Nobel 
WHERE ev > 1990 
AND tema = 'Irodalom';
```

11.feladat: Mit lehet tudni Robert Solow-ról?

```sql
SELECT *
FROM Nobel
WHERE nyertes = 'Robert Solow';
```

12.feladat? Mit lehet tudni azokról akiknek ‘Sir’ van a nevében?

```sql
SELECT *
FROM Nobel
WHERE nyertes like 'Sir%'
ORDER BY ev DESC;
```
