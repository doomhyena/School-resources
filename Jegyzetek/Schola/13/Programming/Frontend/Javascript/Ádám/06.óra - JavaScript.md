1. feladat:
A függvény megnéz egy számlistát és kiszámolja a listaszámok reciprokának az összegét. A megoldásnak szintén vesszük a reciprokát és azt egy tizedesre kerekítve visszaadjuk.
Pl.: 
console.log(reciprok([6,3,6])) --> eredmény: 1.5
(mert: 1/6 + 1/3 + 1/6 = 2/3 --> 3/2 = 1.5)
Tippek: segédváltozó az eredménynek, for ciklus, +=, toFixed().
//--
function reciprok(számok){
  var összeg = 0;
  for (const elem of számok){//Alternatív for séma
    összeg += (1/elem); //1/elem, mert reciprok kell
  }
  var megoldás = 1/összeg; //eredménynek is a reciprokát vesszük
  megoldás = megoldás.toFixed(1); //kerekítés: számváltozó.toFixed(tizedes)
}
//---------------------------------
Bizonyos esetekben a függvényen belül meghívjuk önmagát (optimális esetben egy már meváltozott bemeneti paraméterre).
Pl.:
function függnénynév(adat){
  kód, amely megváltoztatja az "adat" tartalmát;
  függvénynév(adat);
}

2. feladat:
A függvény keresse meg a számlista legnagyobb értékét (de nem használható a max() függvény).
Pl.: maxkeres([0,12,87,4]) --> 87
//---
Logika:
  Ha csak egy elemű a lista, akkor értelemszerűen azt az 1 számot adjuk       vissza maximumként.
  Ha több szám van, akkor megvizsgáljuk a lista első 2 számát. A nagyobb       szám értéke bekerül a lisebb szám helyére is.
  Aztán töröljük a lista első számát.
  Végül meghívjuk a maxkeres() függvényt önmagán belül (és itt ugyebár egy     egyel rövisebb listára hívódik meg, mert előzőleg levágtunk egy számot).
//--- 
function maxkeres(számtömb){
  if (számtömb.length == 1) { //Ha 1 elemű a lista
      return számtömb[0];//akkor azt kiíratjuk
  //Ez az if csak az utolsó körben aktiválódik, akkor marad a lista 1 elemű.
  }
  if (számtömb[0] > számtömb[1]) { //Ha az 1. elem kisebb, mint második
      számtömb[1] = számtömb[0]; //akkor a 2. tartalma legyen átvéve az 1.-ből
  }
  //Levágjuk a lista első elemét: (tömbnév.splice(hely))
  számtömb = számtömb.splice(1);
  //Meghívjuk a függvényt a módosult számtömb-tartalomra:
  return maxkeres(számtömb)
}

console.log(maximumkeres([2,6,9,7]))
