# Csontos Kincső Java dolgozat
Java dolgozat Wuncs Dávid Asztali Alkalmazások II. Órájára
## Feladatleírás:
  1. Készítsen egy osztályt Sportoló néven az alábbi tulajdonságokkal:
  2. Készítse el a konstruktort úgy, hogy mindhárom tulajdonság szerepeljen mint paraméter!
  3. Készítse el az ovrride toString metódust is!
  4. Hozz létre egy globális listát 'sportolok' néven.
  5. Készítsen egy metódust 'feltolt' néven, amely feltölti a listát 100 darab sportolóval.
  6. Készítsen egy metódust 'ossz' néven mely a mintának megfelelően kiírja az outputra az összes sportoló nevét és, hogy a két hónapban összesen hány km-t tett neg.
  7. Készítsen egy metódust 'jan' néven mely a mintának megfelelően kiírja, hogy melyik sprotoló tete meg a legtöbb km-t januárban. (Azonosűg esetén elég egyet megjeleníteni)
