### **1. Lista inicializálása**

```java
ArrayList<Integer> szamok = new ArrayList();
szamok.add(17);
```

- Létrehoz egy `ArrayList` típusú listát, amelyhez az első elemként a `17` értéket adja hozzá.

---

### **2. Feladat (1. feladat)**

```java
for (int i = 0; i < 10; i++) {
    szamok.add(rnd.nextInt(100 - 1 + 1) + 1);  
}
System.out.println(szamok);
```

- Véletlenszerűen 10 darab 1 és 100 közötti számot generál, és hozzáadja a `szamok` listához.
- A lista tartalmát kiírja a konzolra.

---

### **3. Feladat (2. feladat)**

```java
if (szamok.get(2) % 2 == 0) {
    System.out.println("A harmadik elem páros");
} else {
    System.out.printf("A harmadik elem páratlan");
}
```

- Ellenőrzi a `szamok` lista harmadik elemét (indexelés 0-tól kezdődik), hogy páros-e, majd kiírja az eredményt.

---

### **4. Feladat (3. feladat)**

```java
for (int i = 1; i <= 20; i++) {
    System.out.println("7 × " + i + " = " + (7 * i));
}
```

- Kiírja a 7-es szorzótábla értékeit 1-től 20-ig.

---

### **5. Feladat (4. feladat)**

```java
for (int i = 1; i <= 200; i++) {
    System.out.println("7 × " + i + " = " + (7 * i));
}
```

- Hasonlóan az előző feladathoz, de most 1-től 200-ig írja ki a 7-es szorzótábla értékeit.

---

### **6. Feladat (5. feladat)**

```java
ArrayList<Integer> asd = new ArrayList();
int hossz = rnd.nextInt(5);
for (int i = 0; i < hossz; i++) {
    asd.add(rnd.nextInt(20, 100));
}
```

- Létrehoz egy új listát `asd` néven.
- Véletlenszerűen meghatározza a lista hosszát 0 és 4 közötti értékben (az `rnd.nextInt(5)` miatt).
- Hozzáad véletlenszerű számokat (20 és 100 közötti értékeket) a listához.

---

```java
import java.util.ArrayList;  
import java.util.Random;  
  
class Main {  
    public static void feladat(int n) {  
        System.out.println(n + ".feladat");  
    }  
    public static void main(String[] args) {  
        ArrayList<Integer> szamok = new ArrayList();  
        szamok.add(17);  
        Random rnd = new Random();  
  
        feladat(1);  
  
        for(int i = 0; i < 10; i++) {  
            szamok.add(rnd.nextInt(100-1+1)+1);  
        }  
        System.out.println(szamok);  
  
        feladat(2);  
        if(szamok.get(2) % 2 == 0) {  
            System.out.println("A harmadik elem páros");  
        } else {  
            System.out.printf("A harmadik elem páros");  
        }  
  
        feladat(3);  
        for (int i = 1; i <= 20; i++) {  
            System.out.println("7 × " + i + " = " + (7 * i));  
        }  
  
        feladat(4);  
  
        for (int i = 1; i <= 200; i++) {  
            System.out.println("7 × " + i + " = " + (7 * i));  
        }  
  
        feladat(5);  
  
        ArrayList<Integer> asd = new ArrayList();  
  
        int hossz = rnd.nextInt(5);  
        for(int i = 0; i < hossz; i++) {  
            asd.add(rnd.nextInt(20, 100));  
        }  
    }  
}
```
