### Első rész: Üres tömb feltöltése

```php
$lista = array();

array_push($lista, array());
array_push($lista, array());
array_push($lista, array());
array_push($lista, array());
array_push($lista, array());

print_r($lista);

```

**Magyarázat:**

- Itt létrehozol egy üres tömböt `$lista = array();`.
- Az `array_push()` függvény segítségével hozzáadsz új, üres tömböket a `$lista`hoz. Mivel itt 5-ször használod az `array_push()`t, a `$lista` végül egy 5 elemes tömb lesz, ahol minden elem egy üres tömb.
- A `print_r($lista);` függvény kiírja a tömb szerkezetét. Ez egy nagyon hasznos eszköz tömbök és más adatszerkezetek vizsgálatára.

Eredmény: Egy 5 elemes tömböt látsz, amely mindegyik eleme egy üres tömb lesz, így:

```php
Array
(
    [0] => Array
        (
        )
    [1] => Array
        (
        )
    [2] => Array
        (
        )
    [3] => Array
        (
        )
    [4] => Array
        (
        )
)

```

### Második rész: 2D tömb létrehozása

```php
$tomb = array();

for($i=0; $i<5; $i++){
    $uj_tomb = array();

    for($j=0; $j<5; $j++){
        array_push($uj_tomb, $j);
    }

    array_push($tomb, $uj_tomb);
}

print_r($tomb);
print_r($tomb[0]);
echo $tomb[1][3];

```

**Magyarázat:**

- Létrehozol egy új, üres tömböt `$tomb = array();`.
- Egy dupla `for` ciklus segítségével egy 5x5-ös mátrixot (2D tömböt) készítesz. Az első ciklus (az `$i`) 5-ször fut végig, míg a belső ciklus (az `$j`) is 5-ször ismétlődik.
- Minden belső ciklus végén hozzáadsz egy új elemet az aktuális sorhoz (azaz az `$uj_tomb`hoz), ami az éppen aktuális `$j` érték lesz (0-tól 4-ig).
- Minden külső ciklus végén hozzáadod az elkészült sort a `$tomb` tömbhöz.
- Végül a `print_r($tomb)` kiírja a teljes tömböt, a `print_r($tomb[0])` csak az első sort, az `echo $tomb[1][3]` pedig az első sor 4. elemét (mivel a PHP indexelése 0-tól indul).

Eredmény:

```php
Array
(
    [0] => Array
        (
            [0] => 0
            [1] => 1
            [2] => 2
            [3] => 3
            [4] => 4
        )
    [1] => Array
        (
            [0] => 0
            [1] => 1
            [2] => 2
            [3] => 3
            [4] => 4
        )
    ...
)

```

### Harmadik rész: Random számokkal feltöltött 10x10-es tömb

```php
$tomb = array();

for($i=0; $i<10; $i++){
    $uj_tomb = array();

    for($j=0; $j<10; $j++){
        $randomszam = rand(0, 10);
        array_push($uj_tomb, $randomszam);
    }

    array_push($tomb, $uj_tomb);
}

print_r($tomb);
```

**Magyarázat:**

- Itt is egy 2D tömböt készítesz, de most 10 sorból és 10 oszlopból áll.
- A belső ciklus minden futásakor generálsz egy random számot a `rand(0, 10)` függvénnyel, ami 0 és 10 közötti véletlenszámot ad vissza, és ezt hozzáadod az aktuális sorhoz.
- A végén az elkészült sorokat hozzáadod a `$tomb`hoz.
- A `print_r($tomb)` kiírja a teljes tömböt, amely egy 10x10-es mátrix, ahol minden elem egy random szám lesz 0 és 10 között.

### Negyedik rész: Névhez rendelt számok

```php
$nevek = array("András", "Balázs", "Cecília", "Dénes");

$tomb = array();

for($i=0; $i<count($nevek); $i++){
    $uj_tomb = array();
    array_push($uj_tomb, $nevek[$i]);

    $szamok = array();

    for($j=0; $j<5; $j++){
        $randomszam = rand(1, 5);
        array_push($szamok, $randomszam);
    }

    array_push($uj_tomb, $szamok);
    array_push($tomb, $uj_tomb);
}

print_r($tomb);
```

**Magyarázat:**

- Létrehozol egy `$nevek` tömböt, amely négy nevet tartalmaz.
- Egy ciklussal végigmész a neveken, és minden névhez létrehozol egy új tömböt, amely tartalmazza az adott nevet és egy másik tömböt, amely 5 darab random számot tartalmaz 1 és 5 között.
- Az `array_push()` segítségével először a nevet, majd a random számok tömbjét hozzáadod az új tömbhöz, és végül ezt a komplex tömböt hozzáadod a `$tomb`hoz.
- A `print_r($tomb)` segítségével az eredmény valami ilyesmi lesz:

```php

(

            [0] => András
```

### Utolsó rész: Asszociatív tömbök

```php
$nevek = array("nev" => "András", "szul_datum" => "2001.07.28");

print_r($nevek);
echo $nevek['szul_datum'];

$nevek = array();
$nevek['nev'] = "András";
print_r($nevek);

```

**Magyarázat:**

- Az asszociatív tömbök olyan tömbök, amelyekben az indexek nem számok, hanem kulcsok (pl. `"nev"`, `"szul_datum"`).
- Az első példában létrehozol egy asszociatív tömböt, ahol `"nev"` kulcshoz az "András" érték, míg a `"szul_datum"` kulcshoz a "2001.07.28" érték tartozik.
- A `print_r()` kiírja a teljes tömböt, míg az `echo $nevek['szul_datum'];` kinyomtatja az "András" születési dátumát.
- A második példában egy üres tömböt hozol létre, majd dinamikusan hozzáadsz egy elemet (kulcs-érték párral), ahol a `"nev"` kulcshoz az "András" értéket rendeled.
