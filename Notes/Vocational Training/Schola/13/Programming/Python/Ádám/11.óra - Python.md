## 1. feladat – Fájl beolvasása

```python
jelek = []  # ebbe olvassuk be a fájlban tárolt adatokat:
adat = open('C:\\Users\\kiss.adam\\downloads\\jel.txt')  # forrásfájl megnyitása
for sor in adat.readlines():  # menjen végig a forráson soronként
    ora, perc, mperc, x, y = sor.split()  # 1 sor 5 adatot tartalmaz
    jelek.append([int(ora), int(perc), int(mperc), int(x), int(y)])  # egyben hozzáfűzöm a jelek-hez
adat.close()  # a művelet végén mindig lezárjuk a fájlhozzáférést
```
### Lépések
- `jelek` – egy lista, amibe minden sor adatai bekerülnek.
- `open(...)` – megnyitja a `jel.txt` fájlt.
- `for sor in adat.readlines()` – soronként bejárja a fájlt.
- `sor.split()` – az aktuális sort szóközök mentén 5 részre bontja:
    - `ora`, `perc`, `mperc` – időadatok
    - `x`, `y` – koordináták
- `jelek.append([...])` – minden sort egy listaként tárol:
    - `[óra, perc, másodperc, x, y]` (mind `int`-é konvertálva)
- `adat.close()` – lezárja a fájlt.
A `jelek` lista tehát ilyen struktúrájú elemeket tartalmaz:
```python
[óra, perc, másodperc, x, y]
```

## 2. feladat – Egy jel koordinátáinak lekérdezése
```python
ssz = int(input('A jel sorszáma? '))
print(f'x={jelek[ssz-1][3]} y={jelek[ssz-1][4]}')
```
- `ssz` – a felhasználó által megadott jelsorszám (1-től indul).
- `jelek[ssz-1]` – mivel a lista indexelése 0-tól indul, ezért `sorszám - 1`.
- `[3]` – az x-koordináta indexe.
- `[4]` – az y-koordináta indexe.
Így a program kiírja a megadott sorszámú jel x és y értékeit.

## 3. feladat – Segédfüggvények

Itt jönnek a különböző számító függvények, amelyek az időt és a koordinátákat dolgozzák fel.

### 3.1. Két időpont között eltelt idő

```python
def eltelt(ido1,ido2):
    mp1 = ido1[0] * 3600 + ido1[1] * 60 + ido1[2]
    mp2 = ido2[0] * 3600 + ido2[1] * 60 + ido2[2]
    return abs(mp2 - mp1)
```

- `ido1` és `ido2` – formátuma: `[óra, perc, másodperc]`.
- Mindkét időpontot átszámolja összes másodpercre:
    - `óra * 3600 + perc * 60 + másodperc`
- `abs(mp2 - mp1)` – az eltérés abszolút értéke, hogy ne legyen negatív.
### 3.2. Másodpercből „óra:perc:másodperc” formátum

```python
def időbe(mperc):
    ora = mperc // 3600
    perc = (mperc % 3600) // 60
    mp = mperc % 60
    return str(ora) + ':' + str(perc) + ':' + str(mp)
```
- `mperc` – teljes idő másodpercben.
- `ora` – egész osztással jön ki: `mperc // 3600`.
- `perc` – maradékból számolva: `(mperc % 3600) // 60`.
- `mp` – a maradék másodperc: `mperc % 60`.
- Visszaad egy sztringet: `"óra:perc:mp"`.
### 3.3. Jeltávolság (Euklideszi távolság)
```python
def tavolsag(jel1,jel2):
    return ((jel2[3]-jel1[3])**2 + (jel2[4]-jel1[4])**2) ** 0.5
```
- `jel1` és `jel2` – mindkettő egy `[óra, perc, mp, x, y]` lista.
- `jel2[3] - jel1[3]` – x-koordináta különbsége.
- `jel2[4] - jel1[4]` – y-koordináta különbsége.
- A képlet: sqrt(Δx² + Δy²), itt hatványozással és 0.5-ös gyökvonással.
### 3.4. Időváltozás hibaszáma
```python
def idovaltozas_hibaszam(jel1,jel2):
    return (eltelt(jel2[:3], jel1[:3]) + 299) // 300 - 1
```

- `jel1[:3]` – az első 3 elem: `[óra, perc, mp]`.
- `eltelt(...)` – két jel időpontja közötti különbség másodpercben.
- `(érték + 299) // 300` – kvázi „felkerekítés” 300 mp-es blokkokra.
- Az egész képlet egy „hibaszámot” ad vissza, arra vonatkozóan, hány 300 mp-es egységnyi eltérés van az időben.
### 3.5. Koordinátaváltozás hibaszáma

```python
def koordinata_hibaszam(jel1,jel2):
    xh = (abs(jel2[3] - jel1[3])+9) // 10 - 1
    yh = (abs(jel2[4] - jel1[4])+9) // 10 - 1
    return max(xh,yh)
```
- `abs(jel2[3] - jel1[3])` – x koordináta különbségének abszolút értéke.
- `+9) // 10` – itt is „felkerekítős” osztás 10-es egységekre.
- Ugyanez y-ra is.
- `max(xh, yh)` – a két tengely hibaszáma közül a nagyobbat veszi, ez lesz a koordináta hibaszám.
### 3.6. Hiba vizsgálata két jel között
```python
def hiba(jel1,jel2):
    if idovaltozas_hibaszam(jel1,jel2) or koordinata_hibaszam(jel1,jel2):
        ido = str(jel2[0]) + ' ' + str(jel2[1]) + ' ' + str(jel2[2])
        if idovaltozas_hibaszam(jel1,jel2) > koordinata_hibaszam(jel1,jel2):
            return ido + ' ' + 'időeltérés' + str(idovaltozas_hibaszam(jel1,jel2))
        else:
            return ido + ' ' + 'koord. eltérés' + str(koordinata_hibaszam(jel1, jel2))
    else:
        return False
```
- Ha bármelyik hibaszám nem nulla (`or`):
    - összeállít egy szöveget, amiben:
        - benne van a második jel időpontja (`jel2[0..2]`),
        - és az, hogy időbeli vagy koordináta eltérés a nagyobb hiba.
- Ha nincs hiba (mindkét hibaszám 0), akkor `False`-t ad vissza.

## 4. feladat – Teljes időtartam kiszámítása
```python
print("Időtartam: ", időbe(eltelt(jelek[-1][:3], jelek[0][:3])))
```
- `jelek[0][:3]` – az első jel időpontja.
- `jelek[-1][:3]` – az utolsó jel időpontja (`-1` → utolsó elem).
- `eltelt(...)` – a kettő közti időkülönbség (másodpercben).
- `időbe(...)` – a kapott másodpercek konvertálása `óra:perc:másodperc` formára.

## 5. feladat – A koordináta rendszer téglalapja

```python
minx = min([jel for jel in jelek], key=lambda x: x[3])[3]
miny = min([jel for jel in jelek], key=lambda x: x[4])[4]
maxx = max([jel for jel in jelek], key=lambda x: x[3])[3]
maxy = max([jel for jel in jelek], key=lambda x: x[4])[4]
print(f"Bal alsó: {minx} {miny}, jobb felső: {maxx} {maxy}")
```
- `min(..., key=lambda x: x[3])` – azt a jelet keresi, ahol a 3. index (x koordináta) a legkisebb.
- Ugyanez y-ra (index 4) és maximumra is.
- Végül kiírja:
    - bal alsó sarok: `(minx, miny)`
    - jobb felső sarok: `(maxx, maxy)`
Ez adja meg azt a téglalapot, ami „körbefogja” az összes jelet.

## 6. feladat – Összes elmozdulás kiszámítása
```python
elmozdulas = 0
for i in range(len(jelek)-1):
    elmozdulas += tavolsag(jelek[i], jelek[i+1])
print(f"Elmozdulás: {elmozdulas:.3f}")
```
- Végigmegy az összes egymást követő jelpáron:
    - `(jelek[0], jelek[1])`
    - `(jelek[1], jelek[2])`
    - stb.
- Mindegyik pár között kiszámítja a távolságot a `tavolsag` függvénnyel.
- Ezeket összeadja – így megkapjuk a teljes elmozdulás hosszát.
- Kiírás: 3 tizedesjegyre formázva (`:.3f`).

## 7. feladat – Hibás jelek kiírása fájlba
```python
kimeneti = open('kimaradt.txt','w')
for i in range(len(jelek)-1):
    if hiba(jelek[i], jelek[i+1]):
        print(hiba(jelek[i], jelek[i+1]), file=kimeneti)
kimeneti.close()
```
- `open('kimaradt.txt','w')` – megnyit (vagy létrehoz) egy fájlt írásra.
- Végigmegy az összes egymást követő jelpáron.
- Ha a `hiba(jel1,jel2)` nem `False`, akkor:
    - a hiba leírását **a fájlba** írja (`file=kimeneti`).
- A végén lezárja a kimeneti fájlt.
A `kimaradt.txt` fájlban így soronként megjelennek a hibás jelek időpontjai és a hibatípus (időeltérés vagy koordináta-eltérés) a hozzá tartozó hibaszámmal.
