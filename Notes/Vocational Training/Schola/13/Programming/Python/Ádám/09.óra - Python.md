## 1. Zeneszerző osztály

```python
class Zeneszerzo:
    zszerzo = 0
    def __init__(self,nev,orszag,mufaj):
        self.nev = nev
        self.orszag = '{}i régió'.format(orszag)
        self.mufaj = mufaj
        Zeneszerzo.zszerzo += 1
```
## Cél
Zeneszerzők adatainak tárolása és a példányok számolása.
## Attribútumok

- **osztályattribútum:**  
    `zszerzo` – megszámolja, hány Zeneszerzo objektum jött létre.
- **példányattribútumok:**
    - `self.nev` – zeneszerző neve
    - `self.orszag` – „ország + i régió” formában tárolt adat
    - `self.mufaj` – műfaj (pl. rock, pop, jazz)

## Működés

Minden új példány létrehozásakor a konstruktor:
1. feltölti az adatokat,
2. módosítja az ország paraméter megjelenését,
3. növeli az osztály szintű számlálót.
## Példa
```python
zsz1 = Zeneszerzo('XY','Magyarország','Rock')
print(zsz1.orszag)
```

# 2. Számoló osztály

```python
class Szamolo:
    def __init__(self, szam):
        self.egyesek = szam
        self.harmasok = szam//3
        self.kilencesek = szam//9
    def kiirat(self):
        return f"Egyesek:{self.egyesek}, hármasok: {self.harmasok}, kilencesek: {self.kilencesek}."
```

## Cél

Megállapítani, hogy egy adott szám hányszor tartalmazza az:
- 1-et,
- 3-at,
- 9-et.

## Attribútumok
- `self.egyesek` – maga az eredeti szám
- `self.harmasok` – hány egész 3-as fér bele
- `self.kilencesek` – hány egész 9-es fér bele
A `//` egész osztást jelent.
## Metódus
- **`kiirat()`** – formázott szövegként visszaadja a kiszámolt értékeket.
## Példa
```python
ertek1 = Szamolo(12)
print(ertek1.kiirat())
```

# 3. KedvencÉtel osztály

```python
class KedvencEtel:
    def __init__(self,nev,szeret=[],nemszeret=[]):
        self.nev = nev
        self.szeret = szeret
        self.nemszeret = nemszeret

    def izlés(self,kaja):
        szoveg = '{} {}-t eszik és '.format(self.nev,kaja)
        if kaja in self.szeret:
            return szoveg + 'szereti azt.'
        elif kaja in self.nemszeret:
            return szoveg + 'és ki nem állhatja.'
        else:
            return szoveg + '!'
```

## Cél

Nyilvántartani egy személy:
- kedvenc ételeit,
- nem kedvelt ételeit,
majd eldönteni, hogy egy megadott fogást szeret-e.
## Attribútumok
- `self.nev` – a személy neve
- `self.szeret` – lista a kedvelt fogásokról
- `self.nemszeret` – lista a kerülendő fogásokról
## Metódus

### `izlés(kaja)`
Sablonszöveg alapján eldönti:
- ha az étel a szeretett listában van → „szereti azt”
- ha a nem szeretett listában van → „ki nem állhatja”
- különben → „!”
## Példa
```python
evo1 = KedvencEtel("Béla",["rizs","alma"],["brokkoli","répa"])
print(evo1.izlés("alma"))
```
