### **1. Metódus (függvény) felépítése**

A metódusok vagy függvények olyan kódrészletek, amelyek egy adott feladatot hajtanak végre, és újra felhasználhatók a program különböző részein.

#### **Általános felépítés:**

```python
def metódusnév(paraméterek):
    # Érdemi műveletek
    return eredmény

print(metódusnév(konkrét_értékek))
```

🔹 `def` – A metódus kezdetét jelzi.  
🔹 `metódusnév` – Érdemes beszédes nevet adni neki.  
🔹 `paraméterek` – A metódus bemeneti adatai.  
🔹 `return` – Visszatérési érték, amelyet a metódus eredményként ad vissza.  
🔹 `print(...)` – A metódust meghívjuk, és kiírjuk az eredményét.

### **2. Ismétlődő elemek keresése egy listában**

```python
def ismétlődők(lista):
    egyediek = []  # Egyedi számok listája
    ismétlődők = []  # Ismétlődő számok listája
    for i in lista:
        if i not in egyediek:
            egyediek.append(i)  # Ha még nem láttuk ezt a számot, elmentjük
        else:
            ismétlődők.append(i)  # Ha már láttuk, akkor ismétlődőként tároljuk
    if len(ismétlődők) > 0:
        return sorted(ismétlődők)  # A lista tartalmát növekvő sorrendbe tesszük és visszaadjuk
    else:
        return None  # Ha nincs ismétlődés, visszatérünk None értékkel
```

📌 **Mit csinál a metódus?**  
Megvizsgálja a bemeneti listát, és kigyűjti azokat a számokat, amelyek többször is előfordulnak.

### **3. Minden X-edik elem összeadása**

```python
def xösszeg(lista, x):
    segéd = 1  # Számláló változó
    összeg = 0  # Összegző változó
    for i in lista:
        if segéd == x:  # Ha az aktuális elem X-edik elem
            összeg += i  # Hozzáadjuk az összeghez
            segéd = 0  # Visszaállítjuk a számlálót
        segéd += 1
    return összeg
```

📌 **Mit csinál a metódus?**  
Megadott lépésenként összeadja a lista adott indexű elemeit.

🔹 **Példa:**  
`print(xösszeg([1,2,3,4,5,6,7],2))`  
**Eredmény:** `12` (mert az 2., 4., és 6. elem 2+4+6=12).

### **4. Egy szám felírható-e két egymást követő egész szám összegeként?**

`print(felírható(9)) # True`

### **6. Lista összege, ha nincs benne 5-ös**

```python
def számoló(lista):
    összeg = 0
    for i in lista:
        if i == 5:
            return 0  # Ha van 5-ös, azonnal visszatérünk 0-val
```

### **7. Medián kiszámítása**

```python
def medián(lista):
    hossz = len(lista)
    rendezett = sorted(lista)  # Sorba rendezzük a listát
    if hossz % 2 == 1:  # Ha páratlan elemszámú
        return rendezett[hossz // 2]  # A középső elem
    else:
        x1 = rendezett[hossz // 2 - 1]
        x2 = rendezett[hossz // 2]
        return round((x1 + x2) / 2, 1)  # Két középső elem átlaga
```

📌 **Mit csinál a metódus?**  
Kiszámítja egy számlista mediánját (középső érték).

### **8. Két szám számjegyeinek különbsége**

```python
def jegykül(szám1, szám2):
    összeg = 0
    alt1 = str(szám1)
    alt2 = str(szám2)
    for i in range(len(alt1)):  # Végigmegyünk a számjegyeken
        összeg += abs(int(alt1[i]) - int(alt2[i]))
    return összeg
```

📌 **Mit csinál a metódus?**  
Összegzi a két szám megfelelő helyi értékein lévő számjegyek különbségét.

🔹 **Példa:**  
`jegykül(234, 489) → 2+5+5 = 12`

### **9. Akkumulatív összegzés**

```python
def akkumulativ(lista):
    if len(lista) == 0:
        return []
    megoldás = []
    összeg = 0
    for i in lista:
        összeg += i
        megoldás.append(összeg)
    return megoldás
```

📌 **Mit csinál a metódus?**  
Létrehoz egy új listát, amelyben minden elem az addig összeadott elemek összegét tartalmazza.

🔹 **Példa:**  
`akkumulativ([1,2,3,4]) → [1,3,6,10]`
