## 1. feladat – Fájl beolvasása

```python
be = open('C:\\Users\\kiss.adam\\downloads\\melyseg.txt')
sor = int(be.readline())
oszlop = int(be.readline())
adatok = []  # ebben lesz a fájl teljes tartalma
for sor in be.readlines():  # végigmegyünk a fájl sorain
    egysor = []  # soronként akarom felépíteni az adatok tömb leendő tartalmát
    x = sor.split()
    for i in x:
        egysor.append(int(i))  # felépítem az 1 darab sort
    adatok.append(egysor)
be.close()
```
### Mit csinál ez a rész?
- Megnyitja a `melyseg.txt` fájlt.
- Az első sor: `sor` → a rács sorainak száma.
- A második sor: `oszlop` → a rács oszlopainak száma.
- A többi sor: maga a mélységi mátrix.

Minden további sor:
- `sor.split()` → a sort szétdarabolja szóközök mentén.
- Minden elemet `int`-té alakít és beletesz egy `egysor` listába.
- Az `egysor` listát hozzáadja az `adatok` listához.
A végén:
- `adatok` egy kétdimenziós lista (mátrix), szerkezete:  
    `adatok[sor_index][oszlop_index]` → az adott pont mélysége deciméterben.

## 2. feladat – Egy adott pont mélységének lekérdezése
```python
sorid = int(input("Sor azonosítója?"))
oszlopid = int(input("Oszlop azonosítója?"))
print("Az ott mért mélység ", adatok[sorid-1][oszlopid-1], 'dm volt.')
```

- A felhasználó megad:
    - egy sorszámot,
    - egy oszlopszámot.
- A program a `adatok[sorid-1][oszlopid-1]` elemből olvassa ki a mélységet.
    - `-1` azért kell, mert a listák indexelése 0-tól indul, a bemenet meg 1-től.

## 3. feladat – A tó felszíne és átlagos mélysége
```python
terulet, melyseg = 0, 0
for sor in adatok:
    for i in sor:
        if i:  # "Ha az érték létezik" → ha nem 0
            terulet += 1
            melyseg += i
print("A tó felszíne: {} m2, átlag mélysége pedig {:.2f} m.".
      format(terulet, melyseg/terulet/10))
```

### Logika

- Végigmegy az összes cellán.
- Ha az érték nem 0 (tehát van víz, nem szárazföld):
    - `terulet += 1` → számolja, hány ilyen cella van.
    - `melyseg += i` → összeadja a mélységeket.
- A tó felszíne:
    - egyszerűen a nem 0 értékű cellák száma (mivel minden cella 1 m²-nek számít).
- Átlagos mélység:
    - `melyseg / terulet` → átlag deciméterben,
    - `/10` → átváltás méterbe,
    - `"{:.2f}"` → két tizedesre formázva.

## 4. feladat – A tó legmélyebb pontja
```python
legmelyebb = 0
for sor in adatok:
    for i in sor:
        if i > legmelyebb:
            legmelyebb += i
print("Legmélyebb rész: ", legmelyebb, " dm.")
print("Ezen legmélyebb pont koordinátái: ")
for i, sor in enumerate(adatok):
    for j, hely in enumerate(sor):
        if adatok[i][j] == legmelyebb:
            print("({}, {})".format(i+1,j+1), end=" ")
```

### Mit kéne csinálnia?

Cél: Megtalálni a legnagyobb mélységértéket, és kiírni:
- mekkora ez a mélység,
- melyik koordinátákon fordul elő (sor, oszlop, 1-től sorszámozva).
### Fontos megjegyzés
A következő rész:

```python
if i > legmelyebb:
    legmelyebb += i
```
Logikailag inkább így lenne helyes:
```python
if i > legmelyebb:
    legmelyebb = i
```
Mert a legmélyebb értéket nem összeadni kell, hanem lecserélni a nagyobbra.  
Ha a programodban így van, akkor több lépcsőben fogja növelni, és hibás lesz a legmélyebb érték.

Az alsó rész:
```python
for i, sor in enumerate(adatok):
    for j, hely in enumerate(sor):
        if adatok[i][j] == legmelyebb:
            print("({}, {})".format(i+1,j+1), end=" ")
```

- `enumerate(adatok)` → sorok mellé ad sorindexet (`i`).
- `enumerate(sor)` → soron belüli index (`j`).
- Ha az adott pozíción a mélység megegyezik a `legmelyebb` értékkel:
    - kiírja a koordinátákat (1-től sorszámozva).

## 5. feladat – A tó partvonalának hossza

```python
hossz = 0
for i, sor in enumerate(adatok):
    for j, hely in enumerate(sor):
        if 0 < i < len(adatok) and 0 < j < len(sor) and adatok[i][j] > 0:
            if adatok[i-1][j] == 0:
                hossz += 1
            if adatok[i+1][j] == 0:
                hossz += 1
            if adatok[i][j-1] == 0:
                hossz += 1
            if adatok[i][j+1] == 0:
                hossz += 1
print("A tó partvonalának hossza: ", hossz, " m.")
```

### Mit tekint partvonalnak?
- Partvonal ott van, ahol egy vízben lévő cella (`adatok[i][j] > 0`) mellett:
    - valamelyik szomszédos cella (fel, le, bal, jobb irányban) 0 értékű, azaz szárazföld.
- Minden ilyen „él” 1 méter hosszú a feladat feltétele szerint.

### Logika
- `enumerate(adatok)` → sorindex (`i`) és sor tartalma (`sor`).
- `enumerate(sor)` → oszlopindex (`j`) és az ottani érték (`hely`).

A vizsgálat:
```python
if 0 < i < len(adatok) and 0 < j < len(sor) and adatok[i][j] > 0:
```

- Csak olyan cellákat vizsgál, amelyek:
    - nem az első sorban vannak (i > 0),
    - nem az első oszlopban vannak (j > 0),
    - és víz van rajtuk (`> 0`).

Ezután megnézi a 4 szomszédos cellát.  
Ha valamelyik 0, akkor az adott irányban nő a partvonal hossza 1-el.
Megjegyzés: a szélső sorok/oszlopok vizsgálatánál vigyázni kell az indexeléssel, hogy ne lépjen ki a tömbből. A feltétel itt ezt próbálja védeni, de csak részben (alsó/felső határokra külön oda kell figyelni).

## 6. feladat – Diagram generálása fájlba
```python
sorid = int(input("Keresett azonosító: "))
i = 1
kimenet = open("diagram.txt", "w")  # w = write = írásra nyitjuk meg
for egysor in adatok:
    print("{0:02d}".format(i)+round((egysor[sorid]/10))*'*', file=kimenet)
    i += 1
kimenet.close()
```
Cél:  
Egy új fájl (`diagram.txt`) létrehozása, ahol soronként csillagok jelölik a mélységet.
### Mit csinál?
- Bekér egy azonosítót: `sorid`.
- Megnyitja a `diagram.txt` fájlt írásra.
- Végigmegy az `adatok` sorain (`for egysor in adatok`).
- Minden sorra kiír:
    - sorszámot két számjeggyel (pl. 01, 02, 03…) → `"{0:02d}".format(i)`
    - utána csillagokat: a `egysor[sorid]` szerinti mélységet méterbe konvertálja (osztás 10-zel) és annyi `*`-ot ír.

### Fontos megjegyzés
Az indexelés itt kicsit zavaros:
```python
egysor[sorid]
```
- Ha `sorid` egy oszlopazonosító lenne, akkor:
    - valószínűleg `sorid-1`-et kéne használni (mivel a bemenet 1-től indul, a lista 0-tól).
- Jelen formában könnyen indexelési hibához vezethet, ha `sorid` túl nagy.

A csillagszám:
```python
round((egysor[sorid]/10))*'*'
```
- mélységet dm-ből m-be váltja, kerekít,
- majd annyi `*` jelet ír ki.
