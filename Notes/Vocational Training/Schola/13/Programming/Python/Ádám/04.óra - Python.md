### 1. `listátkészít(szám)`

Ez a függvény egy `szám × szám` méretű négyzetes listát hoz létre, amely minden helyen a `szám` értéket tartalmazza.

```python
def listátkészít(szám):
    return [[szám] * szám] * szám
```

### 2. `vizsgál(szöveg)`

Ez a függvény eldönti, hogy egy adott matematikai egyenlőtlenség (pl. `"3<7<11"`) igaz-e.

```python
def vizsgál(szöveg):
    lista = [int(i) if i.isdigit() else i for i in szöveg.split()]
    for i in range(0, len(lista) - 2, 2):
        if lista[i + 1] == ">" and lista[i] <= lista[i + 2]:
            return False
        elif lista[i + 1] == "<" and lista[i] >= lista[i + 2]:
            return False
    return True
```

**Lépések:**

1. A stringet listává alakítja (számok és műveleti jelek).
2. Végigmegy a listán, ellenőrzi, hogy az egyenlőtlenség igaz-e.

Alternatív, rövidebb megoldás:

```python
def vizsgál2(szöveg):
    return eval(szöveg)
```

### 3. `hányszor(start, vég, jegy)`

Megszámolja, hogy egy számjegy hányszor fordul elő egy adott intervallumban.

```python
def hányszor(start, vég, jegy):
    intervallum = range(start, vég + 1)
    számol = 0
    for szám in intervallum:
        stringSzám = str(abs(szám))  # Negatív számokat is kezel
        for karakter in stringSzám:
            if int(karakter) == jegy:
                számol += 1
    return számol
```

**Példa:** `hányszor(51, 55, 5) → 6`  
(Számok: 51, 52, 53, 54, 55 → a "5" hatszor szerepel.)

### 4. `ellentétek(lista1, lista2)`

Megnézi, hogy a két lista minden elemében egymás ellentétei-e.

```python
def ellentétek(lista1, lista2):
    egyformák = set(lista1) == set(lista2)
    if egyformák:
        for i in range(len(lista1)):
            if lista1[i] == lista2[i]:
                return False
        return True
    return False
```

**Példa:**

```python
ellentétek(["1", "0", "1"], ["0", "1", "0"])  # True
ellentétek(["1", "0", "1"], ["1", "0", "1"])  # False
```

**Megjegyzés:** A `set(lista1) == set(lista2)` ellenőrzi, hogy ugyanazok az elemek vannak-e a két listában.
