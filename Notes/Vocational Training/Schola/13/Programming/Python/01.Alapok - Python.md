### 1. **Változók és Alapvető Műveletek**

A programozásban változókat használunk adatok tárolására, és ezekkel különböző műveleteket hajthatunk végre.

```python
i = 5
j = "alma"

```

- Itt két változót hoztunk létre: az `i` egy egész számot (integer) tárol, míg a `j` egy szöveget (string). Pythonban nem kell megadni a változók típusát, mert a nyelv magától felismeri.

### Műveletek:

```python
k = 7
l = i + k
print(l)

```

- Itt hozzáadjuk az `i` értékét a `k`hoz, és az eredményt tároljuk az `l` változóban, majd kiírjuk a képernyőre. A `print()` függvény a változó vagy eredmény megjelenítésére szolgál.

### Összetettebb műveletek:

```python
k = k + 1
k += 1

```

- Mindkét sor azt jelenti, hogy növeljük `k` értékét 1-gyel. Az első sorban hosszabban, a második sorban rövidebben van leírva ugyanaz.

### 2. **Függvények (metódusok)**

A függvények segítségével ismétlődő kódot egyszerűsíthetünk. Egy függvény blokkon belül végrehajt egy adott műveletet, és az eredményt visszaadja.

### Faktoriális számítása:

```python
def faktorialis(szam):
    megoldas = 1
    for i in range(szam):
        megoldas *= i + 1
    return megoldas

```

- A `faktorialis()` függvény kiszámítja egy szám faktoriálisát. A faktoriális egy szám összes pozitív egész számának szorzata 1-től kezdve (pl. 4! = 4×3×2×1 = 24).
- A `for i in range(szam):` egy ciklus, ami 0-tól indul, és egészen `szam`ig fut, és minden egyes ciklusban megszorozza a `megoldas` nevű változót az `i+1` értékkel.

### 3. **Feltételes műveletek (if-else)**

A feltételek segítségével különböző műveleteket végezhetünk el attól függően, hogy egy logikai feltétel teljesül-e vagy sem.

```python
def faktorialis2(szam):
    megoldas = 1
    if szam == 0:
        return megoldas
    else:
        for i in range(szam):
            megoldas *= i + 1
    return megoldas

```

- Ha a bemeneti érték 0, akkor visszatérünk 1-gyel (mivel 0! = 1). Minden más esetben a megszokott faktoriális számítást végezzük.

### 4. **Listák (tömbök)**

A listák olyan adatszerkezetek, amelyek több elemet tartalmazhatnak. Az elemekhez indexek alapján férünk hozzá, ahol az indexelés 0-val kezdődik.

```python
lista1 = [4, 5, 1, 3]
elem1 = lista1[0]

```

- A `lista1` egy lista, amely négy elemet tartalmaz: [4, 5, 1, 3]. A `lista1[0]` az első elem (4) értékét adja vissza.

### Legnagyobb szám megtalálása a listában:

```python
def legnagyobb(lista):
    maximum = lista[0]
    for i in lista:
        if i > maximum:
            maximum = i
    return maximum

```

- Ez a függvény végigmegy a lista elemein, és minden egyes elem esetében megnézi, hogy nagyobb-e, mint az aktuális maximum. Ha igen, akkor az adott elem lesz az új maximum. A végén visszatérünk a legnagyobb értékkel.

### Minimum és maximum beépített függvényekkel:

```python
def minmax(lista):
    lk = min(lista)
    ln = max(lista)
    return lk, ln

```

- A `min()` és `max()` függvények automatikusan visszaadják a legkisebb és legnagyobb értéket egy listából.

### 5. **Karakterek keresése szövegben**

Különböző szövegfeldolgozó műveleteket hajthatunk végre a listákkal és stringekkel.

### Magánhangzók számlálása:

```python
def mhszámol(szo):
    maganhangzok = ['a', 'e', 'i', 'o', 'u']
    osszesen = 0
    for i in szo:
        if i in maganhangzok:
            osszesen += 1
    return osszesen

```

- Itt végigmegyünk egy szó karakterein, és megszámoljuk, hányszor szerepel magánhangzó a listában.

### 6. **Ciklusok**

A ciklusok segítségével ismétlődő műveleteket hajtunk végre. A két fő ciklus típus: `for` és `while`.

### While ciklus:

```python
def felez(szam1, szam2):
    szamol = 0
    while szam1 > szam2:
        szam1 = szam1 / 2
        szamol += 1
    return szamol - 1

```

- A `while` ciklus addig fut, amíg a `szam1` nagyobb, mint `szam2`. Minden körben elfelezzük `szam1` értékét, és jegyezzük, hogy hányszor történt felezés. Amint `szam1` kisebb lesz, a ciklus leáll.

### 7. **Random számok és játékok**

Az alábbi példában egy játékot hoztunk létre, ahol a felhasználónak 1-10 közötti véletlenszerű számot kell kitalálnia.

```python
from random import randint

véletenszám = randint(1, 10)
próbákSzáma = 3
while próbákSzáma > 0:
    tipp = int(input("Tippelj:"))
    if tipp == véletenszám:
        print("Nyertél!")
        break
    else:
        próbákSzáma -= 1
        print("Hibás tipp!")
else:
    print("Vesztettél!")

```

- A `randint(1, 10)` egy véletlenszerű számot generál 1 és 10 között. A felhasználónak 3 próbálkozása van a szám kitalálására. Ha helyes tippet ad, nyer, különben veszít.

### 8. **Rekurzió**

Rekurzió akkor történik, amikor egy függvény meghívja önmagát. Erre jó példa a következő kódrészlet, amely egy lista összes számát összeadja, még akkor is, ha vannak benne al-listák.

```python
def listaösszeg(lista):
    megoldas = 0
    for i in lista:
        if type(i) == int:
            megoldas += i
        else:
            megoldas += listaösszeg(i)
    return megoldas

```

- Ez a függvény végigmegy a listán. Ha egy elem szám (int), hozzáadja az `összeg`hez. Ha az elem egy másik lista, akkor újra meghívja magát, hogy azt is összeadja.

### 9. **Összetettebb függvények**

Az alábbi kódban néhány matematikai és logikai műveletet láthatunk, amelyek különböző feladatokat oldanak meg.

### Szorzás hatványozással:

```python
def szoroz(x, y):
    return x * (2**y)

```

- Itt a `x` számot megszorozzuk a 2 hatványával (`2^y`). Például, ha `y = 3`, akkor a végeredmény `x * 8` lesz (mivel 2^3 = 8).

### Szorzás két szám között, hatványozva:

```python
def hatvanyozas(x, y):
    eredmeny = x ** y
    return eredmeny

```
