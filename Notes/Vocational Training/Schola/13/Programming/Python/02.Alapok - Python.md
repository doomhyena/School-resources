### 1. **Változók és Alapvető műveletek**

Például itt kezdődik a kódod:

```python
i = 5  #i nevű változó, mögöttes érték: 5
j = "alma"  #szöveges változó

```

- **Változók**: Ezek olyan "tárolók", amelyek értékeket tartalmaznak. Nem kell megadnod a típusukat, mert a Python automatikusan kitalálja a típust, például az `i` egy egész szám (integer), a `j` pedig szöveg (string).
- **Műveletek**: Később például `l = i + k` számolást is végzünk, ahol két számot adunk össze és tároljuk az eredményt egy új változóban.

### 2. **Függvények**

A következő lépés a **függvények**, amelyeket a `def` kulcsszóval definiálsz. Ezek egy adott feladat végrehajtására szolgálnak, és később többször is meghívhatók. Például:

```python
def faktorialis(szam):
    megoldas = 1
    for i in range(szam):
        megoldas *= i + 1
    return megoldas

```

Ez a függvény **faktoriális számítást** végez: `n! = n * (n-1) * ... * 1`. Például, ha `n=4`, akkor az eredmény `4*3*2*1=24`.

### 3. **Feltételes műveletek**

Az if-else feltételek segítségével ellenőrizheted, hogy egy adott feltétel igaz-e vagy sem:

```python
if szam == 0:
    return megoldas
else:
    for i in range(szam):
        megoldas *= i + 1
    return megoldas

```

Ez a kód például ellenőrzi, hogy a szám 0-e. Ha igen, külön kezeli ezt az esetet (0! = 1), egyébként a szokásos faktoriális műveletet végzi el.

### 4. **Listák és műveletek listákkal**

A **listák** összetett objektumok, amelyek több elemet tartalmaznak, például számokat vagy szövegeket. Például:

```python
lista1 = [4,5,1,3]

```

- A listákban tárolt elemeket az **indexük** alapján érheted el. Például: `lista1[0]` az első elem (ami 4).
- A listákkal kapcsolatban hasznos funkciókat is készíthetsz, mint például a legnagyobb vagy legkisebb elem megtalálása egy listában:

```python
def legnagyobb(lista):
    maximum = lista[0]
    for i in lista:
        if i > maximum:
            maximum = i
    return maximum

```

### 5. **Szövegfeldolgozás**

A jegyzetedben van példa olyan függvényekre is, amelyek szövegekkel dolgoznak, például **magánhangzók számolása**:

```python
def mhszámol(szo):
    maganhangzok = ['a','e','i','o','u']
    osszesen = 0
    for i in szo:
        if i in maganhangzok:
            osszesen += 1
    return osszesen

```

Ez a kód egy adott szóból megszámolja a magánhangzókat.

### 6. **While ciklus**

A **while ciklus** addig fut, amíg egy adott feltétel igaz marad. Például:

```python
def felez(szam1, szam2):
    szamol = 0
    while szam1 > szam2:
        szam1 = szam1 / 2
        szamol += 1
    return szamol

```

Itt addig felezi az első számot, amíg az kisebb nem lesz a másodiknál, és közben számolja, hányszor történt felezés.

### 7. **Véletlenszám-generálás**

Ez a rész a véletlenszám generálásról szól, ami például játékoknál hasznos. A `randint` függvény egy adott tartományon belüli véletlenszerű számot generál:

```python
from random import randint
véletenszám = randint(1,10)

```

### 8. **Összetett tömbök (Nested Lists)**

Az összetett tömbök olyan listák, amelyek más listákat tartalmaznak. Itt is fontos megérteni a rekurzió működését, például a lista összegzésére:

```python
def listaösszeg(lista):
    megoldas = 0
    for i in lista:
        if type(i) == int:
            megoldas += i
        else:
            megoldas += listaösszeg(i)
    return megoldas

```
