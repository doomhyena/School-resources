### Magyarázat:

1. **Változók deklarálása:**
    
    - `int szam1;` és `int szam2;`: Két egész szám típusú változó, amibe a felhasználó által megadott számokat fogjuk tárolni.
    - `int sum = 0;`: Ez a változó tárolja a két szám összegét, alapértelmezett értéke 0.
2. **Felhasználói bemenet:**
    
    - A program először kéri a felhasználótól az első számot: `Console.Write("Kerem az elso szamot: ");`.
    - A `Console.ReadLine()` segítségével beolvassa a felhasználó által megadott értéket, majd a `Convert.ToInt32()` konvertálja azt egész számra (`int`).
    - Ugyanezt teszi a második szám esetében is.
3. **Számítás:**
    
    - A két szám összege `sum = szam1 + szam2;` kifejezéssel kerül kiszámításra.
4. **Eredmény kiíratása:**
    
    - A program kiírja az első szám értékét, a második szám értékét és azok összegét: `Console.WriteLine($"A ket szam osszege: {sum}");`.
5. **Program befejezése:**
    
    - A `Console.ReadKey();` megállítja a konzol ablakot, amíg a felhasználó nem nyom meg egy billentyűt, hogy bezárja.

### Példa futás:

```
Kerem az elso szamot: 5
Az elso szam erteke: 5
Kerem az masodik szamot: 8
A masodik szam erteke: 8
A ket szam osszege: 13
```
### Program:

```cs
using System;

namespace fristproject
{
    class Program
    {
        static void Main(string[] args)
        {

            int szam1;
            int szam2;
            int sum = 0;
            
            Console.Write("Kerem az elso szamot: ");
            szam1 = Convert.ToInt32(Console.ReadLine());
            Console.WriteLine($"Az elso szam erteke: {szam1}");
            Console.Write("Kerem az masodik szamot: ");
            szam2 = Convert.ToInt32(Console.ReadLine());
            Console.WriteLine($"A masodik szam erteke: {szam2}");

            sum = szam1 + szam2;
            Console.WriteLine($"A ket szam osszege: {sum}");
            
            Console.ReadKey();
        }
    }
}
```
