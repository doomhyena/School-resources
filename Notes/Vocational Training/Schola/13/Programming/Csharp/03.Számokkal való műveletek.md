### Magyarázat:

1. **Változók deklarálása:**
    
    - `double sum = 0;`: A változó tárolja az 1-től 10-ig terjedő számok összegét.
    - `double atlag = 0.00;`: Ez a változó tárolja az 1-től 10-ig terjedő számok átlagát.
    - `double szorzata = 1;`: Ez a változó tárolja az 1-től 10-ig terjedő számok szorzatát.
2. **Összeg számítása (for ciklussal):**
    
    - A `for` ciklus az 1-től 10-ig (beleértve) lépkedve összeadja az összes számot, és tárolja a `sum` változóban.
3. **Átlag számítása:**
    
    - Az átlagot úgy számoljuk ki, hogy az összeg (`sum`) értékét elosztjuk 10-tel: `atlag = sum / 10;`.
4. **Szorzat számítása (while ciklussal):**
    
    - A `while` ciklus az 1-től 10-ig terjedő számokat szorozza össze, és tárolja a `szorzata` változóban.
5. **Eredmények kiírása:**
    
    - A program kiírja az összeget, az átlagot és a szorzatot a konzolra.
6. **Program befejezése:**
    
    - A `Console.ReadKey();` megállítja a konzol ablakot, amíg a felhasználó nem nyom meg egy billentyűt, hogy bezárja.

### Példa futás:

```
Számok összege:
55
Számok átlaga:
5.5
Számok szorzata:
3628800
```
### Program:

```cs
using System;

namespace secondproject
{
    class Program
    {
        static void Main(string[] args)
        {
            double sum = 0;
            double atlag = 0.00;
            double szorzata = 1;
            for (int i = 0; i <= 10; i++)
            {
                sum += i;
            }
            atlag = sum / 10;
            int j = 1;
            while (j <= 10)
            {
                szorzata = szorzata * j;
                j++;
            }

            Console.WriteLine("Számok összege:");
            Console.WriteLine(sum);
            Console.WriteLine("Számok átlaga:");
            Console.WriteLine(atlag);
            Console.WriteLine("Számok szorzata:");
            Console.WriteLine(szorzata);
   
            Console.ReadKey();
        }
    }
}
```
