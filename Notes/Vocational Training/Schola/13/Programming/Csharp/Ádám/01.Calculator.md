### **1. Form1.Designer.cs**

Ez a fájl a Windows Forms alkalmazásban a felület megjelenését és a vezérlők (pl. gombok, szövegdobozok) elhelyezkedését tartalmazza. A **`InitializeComponent`** metódus felelős a form felépítéséért, például a gombok, címkék és a szövegdobozok létrehozásáért és elhelyezéséért a formon. A vezérlők közül minden gomb egy-egy eseményt generál, mint például a számok gombjainak kattintása, műveleti gombok, törlés vagy az egyenlőség gomb.

- **label1**: A műveletet és az aktuális számot jeleníti meg (pl. "5 +").
- **tbx_Display**: Az eredmény és a felhasználó által beírt számok itt jelennek meg.
- A többi **Button** elem mindegyike egy-egy számot vagy műveleti jelet képvisel. Az eseménykezelők például a **Number_Click** vagy **Operation_Click** metódusokat hívják.

### **2. Form1.cs**

Ez a fájl tartalmazza a program logikai részét, az események kezelését és a számításokat. A `Form1` osztály a `Form` osztályt örökli, és a különböző gombok eseményeit itt kezeljük. Az egyes metódusok magyarázata:

- **`Number_Click`**: A számok gombjainak kattintásakor hívódik. Ha az előző gomb műveleti gomb volt, akkor a kijelzőt felülírja a szám gombbal, különben hozzáfűzi a számot.
- **`Operation_Click`**: A műveleti gombok (pl. +, -, *, /) kattintásakor hívódik. Az aktuális számot (a kijelző szövegét) átalakítja double típusúvá, és elmenti az `a` változóba. A műveleti jelet is elmenti, és az előző műveletet (ha van) mutatja a címkén.
- **`btn_Del_Click`**: A törlés gombot kezeli. Minden változót és kijelzőt töröl.
- **`btn_Eq_Click`**: Az egyenlőség gombot kezeli. A műveletet végrehajtja az előzőleg megadott számokon és műveleti jeleken, majd az eredményt kiírja a kijelzőre.
- **`Last` enum**: Segít az alkalmazásnak nyomon követni, hogy az utolsó bejegyzett gomb szám vagy művelet volt-e, hogy helyesen kezdhessük el a következő lépést.

### **3. Program.cs**

Ez a fájl a program belépési pontját tartalmazza, vagyis az alkalmazás elindítását. A **`Main`** metódusban inicializálja az alkalmazást, engedélyezi a vizuális stílust és elindítja a fő formot (`Form1`), ami tartalmazza a felhasználói felületet.

### Összefoglaló

- A program egy egyszerű számológép, amely az operációs gombok (`+`, `-`, `*`, `/`) és a számgombok segítségével lehetőséget biztosít számítások végrehajtására.
- A felhasználó interakciója eseményeket vált ki, amelyek meghatározzák a számítások logikáját, és az eredményt megjelenítik a képernyőn.
- A **`Form1`** osztály tartalmazza a számológép logikai működését, és kezeli az eseményeket, míg a **`Program.cs`** fájl indítja el az alkalmazást.

﻿using System;
using System.Collections.Generic;
using System.ComponentModel;
using System.Data;
using System.Drawing;
using System.Linq;
using System.Text;
using System.Threading.Tasks;
using System.Windows.Forms;

namespace Calculator
{
	public partial class Form1 : Form
	{
		double a, b, c;
		string operation;
		Last LastClick;

		public Form1()
		{
			InitializeComponent();
			operation = string.Empty;
			LastClick = Last.Not;
		}

		private void Number_Click(object sender, EventArgs e)
		{
			string buttonText = (sender as Button).Text;
			// az előző gombnyomás műveleti vagy szám gomb volt e
			if (LastClick != Last.Number)
			{
				tbx_Display.Text = buttonText;
			}
			else
			{
				tbx_Display.Text += buttonText;
			}
			LastClick = Last.Number;
		}

		private void Operation_Click(object sender, EventArgs e)
		{
			operation = (sender as Button).Text;
			double.TryParse(tbx_Display.Text, out a);
			label1.Text = string.Format($"{a} {operation}");
			LastClick = Last.Operation;
		}

		private void btn_Pre_Click(object sender, EventArgs e)
		{

		}

		private void btn_DP_Click(object sender, EventArgs e)
		{

		}

		private void btn_Del_Click(object sender, EventArgs e)
		{
			operation = tbx_Display.Text = label1.Text = string.Empty;
			a = b = c = 0;
			LastClick = Last.Not;
		}

		private void btn_Eq_Click(object sender, EventArgs e)
		{
			if (operation != string.Empty)
			{
				if (LastClick == Last.Operation)
				{
					a = c;
				}
				else
				{
					double.TryParse(tbx_Display.Text, out b);
				}

				switch (operation)
				{
					case "+":
						c = a + b;
						break;
					case "-":
						c = a - b;
						break;
					case "/":
						c = a / b;
						break;
					case "*":
						c = a * b;
						break;
					default:
						break;
				}
				tbx_Display.Text = c.ToString();
				label1.Text = string.Format($"{a} {operation} {b}");
				LastClick = Last.Operation;
			}
		}
	}

	public enum Last { Operation, Number, Not }
}
