## 1. PaintForm.Designer.cs

Ez a fájl felelős a felhasználói felület (UI) vizuális elemeinek elrendezéséért és létrehozásáért. Néhány fontos részlete:

- **Menüsor (menuStrip1):**
    
    - A "File" menüpont alatt találhatóak az olyan almenük, mint az **Open**, **Save**, **Print** és **Exit**. Ezekhez eseménykezelők vannak hozzárendelve (pl. `openToolStripMenuItem_Click`, `saveToolStripMenuItem_Click`, `exitToolStripMenuItem_Click`).
- **Oldalpanelek:**
    
    - **panel1 (bal oldali panel):** Ebben találhatóak a beállító elemek, például:
        
        - **Pen méret:** Egy `NumericUpDown` vezérlő (`num_PenSize`), amellyel a toll vastagságát lehet megadni.
        - **Pen szín:** Egy gomb (`btn_Color`), amely a jelenlegi toll színét mutatja, és a `ColorDialog` segítségével lehet új színt választani.
        - **Rajzolási módok:** Egy `GroupBox` tartalmazza a különböző rajzolási módokat kiválasztó radio gombokat, mint például:
            - **Free hand (szabadkézi rajz)**
            - **Line (vonal)**
            - **Triangle (háromszög)**
            - **Rectangle (téglalap)**
            - **Circle (kör)**
        - **Clear gomb:** Egy gomb (`btn_Clear`), amellyel törölhető a rajzfelület.
    - **panel2 (fő rajzfelület):**
        
        - Ebben két `PictureBox` található:
            - **pbx_Canvas:** A végleges rajzfelület, amelyre a rajzolt alakzatok véglegesen kerülnek.
            - **pbx_Transparent:** Egy átlátszó réteg, amely a felhasználói interakció (például vonal húzás közben) előnézetét szolgálja. Ez a vezérlő a `pbx_Canvas`-hoz van hozzárendelve, így a végleges kép mögött vagy fölött jelenhet meg.
- **Egyéb vezérlők:**
    
    - **ColorDialog, OpenFileDialog, SaveFileDialog:** Ezek a beépített párbeszédablakok a szín kiválasztásához, fájlok megnyitásához illetve mentéséhez használatosak.

Az összes UI elemhez a Designer fájlban automatikusan generált kód felelős, és az eseményekhez kapcsolt metódusokat a kódtámogatásban (PaintForm.cs) találjuk.

## 2. PaintForm.cs

Ez a fájl tartalmazza a program logikáját, azaz hogyan működik a rajzolás, milyen események váltják ki a rajzolási műveleteket, és hogyan kezeljük a felhasználói interakciókat.

- **Változók és objektumok:**
    
    - `isMouseDown`: Logikai változó, ami azt jelzi, hogy az egér lenyomva van-e.
    - `points`: Egy `List<Point>`, amely a szabadkézi rajzolás során gyűjti a pontokat.
    - `pen`: A rajzoláshoz használt toll (szín és vastagság).
    - `a, b, c, d, nullPoint`: `Point` típusú változók, amelyek segítségével a rajzolási módokhoz (pl. vonal, háromszög) szükséges pontokat tároljuk.
    - `mode`: Egy `DrawMode` típusú változó, amely a jelenleg kiválasztott rajzolási módot (Freehand, Line, Triangle, Rectangle, Circle) jelzi.
- **Form betöltése (PaintForm_Load):**
    
    - Inicializálja a változókat, beállítja az alapértelmezett rajzolási módot (Freehand), létrehozza a tollat a kiválasztott színnel és mérettel, illetve létrehozza a `pbx_Canvas` és `pbx_Transparent` képterületeit (mint új `Bitmap`-eket, amelyek mérete a PictureBox-ok méretével egyezik meg).
- **Egér események a rajzfelületen (pbx_Canvas és pbx_Transparent):**
    
    - **MouseDown:**
        - A rajzolási mód alapján különböző műveletek indulnak el:
            - _Freehand:_ Az egér lenyomásakor rögzítjük az induló pontot, és elkezdjük gyűjteni a pontokat a szabadkézi vonalhoz.
            - _Line, Triangle:_ Az első kattintáskor az `a` pontot, majd további kattintásokkal a `b` illetve `c` pontokat rögzítjük.
            - Téglalap és kör esetén (jelen kódban) a megfelelő műveletek nincsenek megvalósítva.
    - **MouseMove:**
        - _Freehand:_ Amíg lenyomva tartjuk az egeret, minden mozgást rögzítünk és vonallal összekötjük az előző és az aktuális pontot. Így folytonos vonal keletkezik.
        - _Line:_ Ha már van rögzített kezdőpont (`a`), akkor a `pbx_Transparent` rétegen ideiglenesen kirajzolunk egy vonalat a kezdőponttól az aktuális egérpozícióig. Ez vizuális visszajelzést ad a vonal elhelyezkedéséről, mielőtt véglegesítenénk.
        - _Triangle:_ Az első két pont rögzítése után a harmadik pont mozgatásával ideiglenesen kirajzoljuk a háromszöget a transparent rétegen, így a felhasználó látja az alakzatot, mielőtt befejezné a rajzolást.
    - **MouseUp:**
        - Az egér felengedésekor, a rajzolási módnak megfelelően véglegesítjük a rajzot:
            - _Freehand:_ Ha több pontot gyűjtöttünk, akkor a `Graphics.DrawCurve` metódussal egy simított görbét rajzolunk a `pbx_Canvas` képére.
            - _Line:_ Ha a két pont (`a` és `b`) rendelkezésre áll, akkor végleges vonalat rajzolunk a végleges képterületre, majd töröljük az ideiglenes pontokat.
            - _Triangle:_ Ha már három pontunk van, akkor a háromszög oldalait rajzoljuk meg, majd reseteljük a pontokat.
- **További eseménykezelők:**
    
    - **num_PenSize_ValueChanged:** Amikor a toll mérete változik, új tollat hozunk létre az új mérettel.
    - **btn_Color_Click:** A toll színét változtatja, egy `ColorDialog` segítségével, és frissíti a toll beállítását.
    - **Rb_Mode_CheckedChanged:** A radio gombok eseménykezelője, amely a kattintott gomb neve alapján beállítja a rajzolási módot (`DrawMode`), pl. Freehand, Line, Triangle, Rectangle vagy Circle.
    - **exitToolStripMenuItem_Click:** Kilép a programból.
    - **saveToolStripMenuItem_Click:** Elindítja a fájl mentés párbeszédablakot, és megpróbálja elmenteni a rajzot egy kép formátumban (Jpeg).
    - **openToolStripMenuItem_Click:** Megnyit egy fájlt, és beállítja a megnyitott képet a `pbx_Canvas` háttérképének.
    - **PaintForm_Resize:** Az ablak átméretezésekor újrarajzolja a transparent réteget, illetve megpróbálja megtartani a rajz tartalmát az új méretekhez igazítva.
    - **btn_Clear_Click:** Törli a rajzfelületet, új üres `Bitmap`-eket rendelve a PictureBox-okhoz.

A különböző rajzolási módok (szabadkézi, vonal, háromszög) pontgyűjtéssel és a megfelelő rajzolási műveletek végrehajtásával működnek. A téglalap és kör rajzolás esetén a kód csak vázlatosan van jelen (nincs megvalósítva), de az elv hasonló lenne a többi módhoz.

## 3. Program.cs

Ez a fájl tartalmazza az alkalmazás belépési pontját:

- **Main metódus:**
    - Inicializálja a Windows Forms környezetet (`Application.EnableVisualStyles()`, `SetCompatibleTextRenderingDefault`).
    - Elindítja a `PaintForm` példányát, ami a rajzoló ablakot jelenti.

## Összefoglalva

- **Felhasználói felület:** A Designer fájl definiálja a menüt, az oldalsó panelt beállítási elemekkel (pen méret, szín, rajzolási módok) és a rajzfelületet két réteggel (végleges rajz és átlátszó előnézet).
- **Rajzolási logika:** A PaintForm.cs-ben kerül megvalósításra, ahol az egér események (MouseDown, MouseMove, MouseUp) alapján különböző rajzolási módokat (szabadkézi, vonal, háromszög stb.) támogat a program.
- **Fájlműveletek:** A menü segítségével a felhasználó képet nyithat meg, elmentheti a rajzot, illetve kiléphet az alkalmazásból.
- **Dinamikus beállítások:** A toll színét és méretét a felhasználó módosíthatja, és a rajzolási módot radio gombokkal választhatja ki.

Ez a program tehát egy egyszerű, de sokoldalú rajzolóalkalmazás, amely lehetőséget ad az alapvető rajzolási műveletek gyakorlására Windows Forms környezetben.

﻿using System;
using System.Collections.Generic;
using System.ComponentModel;
using System.Data;
using System.Drawing;
using System.Drawing.Drawing2D;
using System.Drawing.Imaging;
using System.Linq;
using System.Text;
using System.Threading.Tasks;
using System.Windows.Forms;

namespace MyPaint
{
	public partial class PaintForm : Form
	{
		bool isMouseDown;
		List<Point> points;
		Pen pen;
		Point a, b, c, d, nullPoint;

		public PaintForm()
		{
			InitializeComponent();
		}

		private void PaintForm_Load(object sender, EventArgs e)
		{
			isMouseDown = false;
			mode = DrawMode.Freehand;
			points = new List<Point>();
			pen = new Pen(btn_Color.BackColor, (float)num_PenSize.Value);
			pbx_Canvas.Image = new Bitmap(pbx_Canvas.ClientSize.Width, pbx_Canvas.ClientSize.Height);
			pbx_Transparent.Parent = pbx_Canvas;
			pbx_Transparent.Image = new Bitmap(pbx_Canvas.ClientSize.Width, pbx_Canvas.ClientSize.Height);
		}

		private void pbx_Canvas_MouseDown(object sender, MouseEventArgs e)
		{
			switch (mode)
			{
				case DrawMode.Freehand:
					#region Freehand
					isMouseDown = true;
					a = e.Location;
					points.Clear();
					points.Add(a);
					#endregion
					break;
				case DrawMode.Line:
					#region Line
					if (a == nullPoint)
					{
						a = e.Location;
					}
					else if (b == nullPoint)
					{
						b = e.Location;
					}
					#endregion
					break;
				case DrawMode.Triangle:
					#region Triangle
					if (a == nullPoint)
					{
						a = e.Location;
					}
					else if (b == nullPoint)
					{
						b = e.Location;
					}
					else if (c == nullPoint)
					{
						c = e.Location;
					}
					#endregion
					break;
				case DrawMode.Rectangle:
					break;
				case DrawMode.Circle:
					break;
				default:
					break;
			}
		}

		private void pbx_Canvas_MouseUp(object sender, MouseEventArgs e)
		{
			switch (mode)
			{
				case DrawMode.Freehand:
					#region Freehand
					isMouseDown = false;
					if (points.Count > 1)
					{
						using (Graphics g = Graphics.FromImage(pbx_Canvas.Image))
						{
							g.DrawCurve(pen, points.ToArray());
							g.SmoothingMode = SmoothingMode.AntiAlias;
						}
						pbx_Canvas.Invalidate();
					}
					#endregion
					break;
				case DrawMode.Line:
					#region Line
					if (a != nullPoint && b != nullPoint)
					{
						using (Graphics g = Graphics.FromImage(pbx_Canvas.Image))
						{
							g.SmoothingMode = SmoothingMode.AntiAlias;
							g.DrawLine(pen, a, b);
						}
						pbx_Canvas.Invalidate();
						ClearPoints();
					}
					#endregion
					break;
				case DrawMode.Triangle:
					#region Triangle
					if (a != nullPoint && b != nullPoint && c != nullPoint)
					{
						using (Graphics g = Graphics.FromImage(pbx_Canvas.Image))
						{
							g.SmoothingMode = SmoothingMode.AntiAlias;
							g.DrawLine(pen, a, b);
							g.DrawLine(pen, a, c);
							g.DrawLine(pen, c, b);
						}
						pbx_Canvas.Invalidate();
						ClearPoints();
					}
					#endregion
					break;
				case DrawMode.Rectangle:
					break;
				case DrawMode.Circle:
					break;
				default:
					break;
			}
		}

		private void ClearPoints()
		{
			a = b = c = d = nullPoint;
		}

		private void pbx_Canvas_MouseMove(object sender, MouseEventArgs e)
		{
			switch (mode)
			{
				case DrawMode.Freehand:
					#region Freehand
					points.Add(e.Location);
					if (isMouseDown && a != nullPoint)
					{
						using (Graphics g = Graphics.FromImage(pbx_Canvas.Image))
						{
							g.SmoothingMode = SmoothingMode.AntiAlias;
							g.DrawLine(pen, a, e.Location);
						}
						pbx_Canvas.Invalidate();
						a = e.Location;
					}
					#endregion
					break;
				case DrawMode.Line:
					#region Line
					if (a != nullPoint)
					{
						using (Graphics g = Graphics.FromImage(pbx_Transparent.Image))
						{
							g.Clear(Color.Transparent);
							g.DrawLine(pen, a, e.Location);
						}
						pbx_Transparent.Invalidate();
					}
					#endregion
					break;
				case DrawMode.Triangle:
					#region Triangle
					if (a != nullPoint && b == nullPoint && c == nullPoint)
					{
						using (Graphics g = Graphics.FromImage(pbx_Transparent.Image))
						{
							g.Clear(Color.Transparent);
							g.DrawLine(pen, a, e.Location);
						}
						pbx_Transparent.Invalidate();
					}
					else if (a != nullPoint && b != nullPoint && c == nullPoint)
					{
						using (Graphics g = Graphics.FromImage(pbx_Transparent.Image))
						{
							g.Clear(Color.Transparent);
							g.DrawLine(pen, a, b);
							g.DrawLine(pen, b, e.Location);
							g.DrawLine(pen, a, e.Location);
						}
						pbx_Transparent.Invalidate();
					}
					#endregion
					break;
				case DrawMode.Rectangle:
					break;
				case DrawMode.Circle:
					break;
				default:
					break;
			}
		}

		private void num_PenSize_ValueChanged(object sender, EventArgs e)
		{
			pen = new Pen(btn_Color.BackColor, (float)num_PenSize.Value);
		}

		private void btn_Color_Click(object sender, EventArgs e)
		{
			if (colorDialog1.ShowDialog() == DialogResult.OK)
			{
				btn_Color.BackColor = colorDialog1.Color;
				pen = new Pen(btn_Color.BackColor, (float)num_PenSize.Value);
			}
		}

		private void Rb_Mode_CheckedChanged(object sender, EventArgs e)
		{
			switch ((sender as RadioButton).Name.Split('_')[2])
			{
				case "FH":
					mode = DrawMode.Freehand;
					break;
				case "Line":
					mode = DrawMode.Line;
					break;
				case "Tri":
					mode = DrawMode.Triangle;
					break;
				case "Rec":
					mode = DrawMode.Rectangle;
					break;
				case "Circle":
					mode = DrawMode.Circle;
					break;
				default:
					break;
			}
		}

				}
				catch (Exception)

					throw;
				}
			}
		}

		private void PaintForm_Resize(object sender, EventArgs e)
		{
			pbx_Transparent.Image = new Bitmap(pbx_Canvas.ClientSize.Width, pbx_Canvas.ClientSize.Height);

			Bitmap temp = new Bitmap(pbx_Canvas.Width, pbx_Canvas.Height);

			using (Graphics g = Graphics.FromImage(temp))
			{
				g.DrawImage(
					pbx_Canvas.Image,
					new Rectangle(0, 0, pbx_Canvas.Image.Width, pbx_Canvas.Image.Height),
					new Rectangle(0, 0, pbx_Canvas.Image.Width, pbx_Canvas.Image.Height),
					GraphicsUnit.Pixel);
			}
			pbx_Canvas.Image = temp;


			//pbx_Canvas.Image = new Bitmap(pbx_Canvas.ClientSize.Width, pbx_Canvas.ClientSize.Height);
		}
