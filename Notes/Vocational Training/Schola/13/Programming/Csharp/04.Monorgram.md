### Magyarázat:

1. **Változók deklarálása:**
    
    - `string keresztnev = "";`: A keresztnevet tárolja.
    - `string vezeteknev = "";`: A vezetéknevet tárolja.
    - `string monogram = "";`: A monogramot tárolja, amit a program később kiszámít.
2. **Felhasználói input:**
    
    - A `Console.ReadLine();` segítségével bekéri a felhasználótól a vezetéknevet és keresztnevet.
3. **Monogram kiszámítása:**
    
    - A `vezeteknev[0]` az első karaktert adja vissza a vezetéknevből.
    - A `keresztnev[0]` az első karaktert adja vissza a keresztnevből.
    - Ezeket a karaktereket összefűzi, így létrehozva a monogramot.
4. **Monogram kiírása:**
    
    - A program kiírja a kiszámított monogramot.
5. **Program befejezése:**
    
    - A `Console.ReadKey();` megállítja a konzol ablakot, amíg a felhasználó nem nyom meg egy billentyűt, hogy bezárja.

### Példa futás:

```
Kérem a Vezetéknevét: Kovács
Kérem a Keresztnevét: Péter
A monogram: KP
```
### Program:

```cs
using System;

namespace thirdproject
{
    class Program
    {
        static void Main(string[] args)
        {
            string keresztnev = "";
            string vezeteknev = "";
            string monogram = "";

            Console.Write("Kérem a Vezetéknevét: ");
            vezeteknev = Console.ReadLine();
            Console.Write("Kérem a Keresztnevét: ");
            keresztnev = Console.ReadLine();

            monogram = vezeteknev[0].ToString();
            monogram += keresztnev[0].ToString();

            Console.Write("A monogram: ");
            Console.WriteLine(monogram);
            Console.ReadKey();

        }
    }
}
```
