## **Kommentelési lehetőség**

### Adatbázis módosítása

1. **Hozz létre egy új táblát** az adatbázisban `comments` néven:
    
    ```sql
    CREATE TABLE comments (
        id INT PRIMARY KEY AUTO_INCREMENT,
        userid INT(255),
        postid INT(255),
        content VARCHAR(255)
    );
    
    ```
    

---

### PHP kód módosítása

2. **Kommentek hozzáadása**:
    
    - Minden poszt alatt legyen egy `form`, ahol a felhasználók hozzászólhatnak.
    
    ```php
    	if (isset($_POST['comment-btn'])) {
    		$userid = $_SESSION['id'];
    ```

3. **Kommentek megjelenítése**:
    
    - A posztok alatt kérdezd le és jelenítsd meg a hozzászólásokat.
    
    ```php

            $talalt_kommentek = $conn->query($lekerdezes);
    
            if ($talalt_kommentek && $talalt_kommentek->num_rows > 0) {
                echo "<div style='max-width: 200px; margin: 10px auto' method='post'>";

                }

            }
    
            echo "<form style='max-width: 200px; margin: 10px auto' method='post' action='index.php?posztid={$poszt['id']}'>";
            echo "<textarea name='comment-content' placeholder='Írd meg a véleményed...'></textarea><br>";
            echo "<input type='submit' name='comment-btn' value='Hozzászólás'>";
            echo "</form>";
        }
    ```

## **Felhasználói aktivitás (statisztika)**

### Adatbázis lekérdezések

5. **Posztok száma felhasználónként**:
    
    ```php
    $userid = $_GET['userid'];
    $lekerdezes = "SELECT COUNT(*) AS posztok_szama FROM posztok WHERE userid=$userid";
    $talalat = $conn->query($lekerdezes)->fetch_assoc();
    echo "Posztok száma: " . $talalat['posztok_szama'];
    
    ```
    
6. **Kedvelések száma, amit a posztjai kaptak**:
    
    ```php
    $lekerdezes = "SELECT COUNT(*) AS kedvelesek FROM likes
                   WHERE postid IN (SELECT id FROM posztok WHERE userid=$userid)";
    $talalat = $conn->query($lekerdezes)->fetch_assoc();
    echo "Kedvelések száma: " . $talalat['kedvelesek'];
    
    ```
    
7. **Kommentek száma a posztjain**:
    
    ```php
    $lekerdezes = "SELECT COUNT(*) AS kommentek FROM comments
                   WHERE postid IN (SELECT id FROM posztok WHERE userid=$userid)";
    $talalat = $conn->query($lekerdezes)->fetch_assoc();
    echo "Kommentek száma: " . $talalat['kommentek'];
    
    ```
