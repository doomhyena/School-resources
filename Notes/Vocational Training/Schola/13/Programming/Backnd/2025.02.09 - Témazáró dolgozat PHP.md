## Fájl feltöltése
### 1. Konfiguráció betöltése és munkamenet indítása
```php
require "config.php";
session_start();
```
- **`require "config.php";`**: Betölti a `config.php` fájlt, amely  tartalmazza az adatbázis kapcsolati adatokat vagy más konfigurációs beállításokat.
- **`session_start();`**: Elindítja a munkamenetet, ami lehetővé teszi, hogy a felhasználói adatokat (pl. bejelentkezési állapot) tároljuk a session változókban.

### 2. Bejelentkezés ellenőrzése
```php
if(!isset($_SESSION['id'])){
    header("Location: reg.php");
} else {
```
- **`if(!isset($_SESSION['id']))`**: Ellenőrzi, hogy a felhasználó be van-e jelentkezve. Ha nincs bejelentkezve (azaz a `$_SESSION['id']` nincs beállítva), akkor átirányítja a felhasználót a `reg.php` oldalra.
- **`header("Location: reg.php");`**: Átirányítja a felhasználót a `reg.php` oldalra, ahol valószínűleg bejelentkezhet vagy regisztrálhat.

### 3. Felhasználó fájljainak kezelése
```php
$mappa = getcwd();
$eleresi_ut = $mappa."\\users\\".$_SESSION['username'];
$felhasznalo_fileok = scandir($eleresi_ut);
$files = array_splice($felhasznalo_fileok, 2);
$size = 0;
$eleresi_ut .= "\\";
```
- **`$mappa = getcwd();`**: Lekéri az aktuális munkakönyvtár elérési útját.
- **`$eleresi_ut = $mappa."\\users\\".$_SESSION['username'];`**: Az elérési út a `users` mappán belül a felhasználó nevével ellátott almappára mutat.
- **`$felhasznalo_fileok = scandir($eleresi_ut);`**: Beolvassa a felhasználó mappájában található összes fájlt és mappát.
- **`$files = array_splice($felhasznalo_fileok, 2);`**: Eltávolítja az első két elemet (`.` és `..`), amelyek a mappa aktuális és szülő mappájára mutatnak.
- **`$size = 0;`**: Inicializálja a változót, amely a fájlok összméretét fogja tárolni.
- **`$eleresi_ut .= "\\";`**: Hozzáad egy perjelet az elérési úthoz, hogy a fájlnevekhez hozzá lehessen fűzni.

### 4. Fájlok méretének kiszámítása
```php
foreach($files as $file){
    $size += filesize($eleresi_ut.$file);
}
$gbsize = $size/1000000000;
$gbsize = round($gbsize, 3);
```
- **`foreach($files as $file)`**: Végigmegy az összes fájlon a felhasználó mappájában.
- **`$size += filesize($eleresi_ut.$file);`**: Hozzáadja az aktuális fájl méretét az összmérethez.
- **`$gbsize = $size/1000000000;`**: Átszámítja a méretet gigabájtokra.
- **`$gbsize = round($gbsize, 3);`**: Kerekíti a méretet 3 tizedesjegyre.

### 5. HTML oldal generálása
```php
<!DOCTYPE html>
<html>
   <head>
       <title>Főoldal</title>
       <meta charset='UTF-8'>
       <meta name='description' content='Rövid leírás az oldal tartalmáról'>
       <meta name='keywords' content='Keresést, Segítő, Szavak, Vesszővel, Elválasztva'>
       <meta name='author' content='Neved'>
       <meta name='viewport' content='width=device-width, initial-scale=1.0'>
       <link rel='stylesheet' href='css/style.css'>
   </head>
   <body>
   <div class="header">
        <p>
            <?= $_SESSION['name']; ?>
            <span><a href="upload.php"><button>Feltöltés</button></a></span>
            <span><?= $gbsize; ?> / 10 GB</span>
        </p>
    </div>
    <div class="files">
        <ul>
            <?php 
                $eleresi_ut = "users/".$_SESSION['username']."/";
                foreach($files as $file){
                    echo "<li><a href='".$eleresi_ut.$file."' download>$file</a></li>";
                }
            ?>
        </ul>
    </div>
   </body>
</html>
```
- **`<title>Főoldal</title>`**: Az oldal címe.
- **`<meta>`**: Metaadatok, amelyek segítik a keresőmotorok optimalizálását és az oldal megjelenítését.
- **`<link rel='stylesheet' href='css/style.css'>`**: CSS stíluslap betöltése.
- **`<div class="header">`**: A fejléc rész, ahol a felhasználó neve, egy feltöltés gomb és a felhasznált tárhely mérete látható.
- **`<div class="files">`**: A fájlok listáját tartalmazó rész, ahol minden fájlra egy letöltési link jelenik meg.
## Mappa létrehozás
### 1. Konfiguráció betöltése
```php
require "config.php";
```
- **`require "config.php";`**: Betölti a `config.php` fájlt, amely valószínűleg tartalmazza az adatbázis kapcsolati adatokat (pl. `$conn` változó, amely az adatbázishoz való csatlakozást tárolja).

---

### 2. Regisztrációs űrlap feldolgozása
```php
if(isset($_POST['reg-btn'])){
```
- **`if(isset($_POST['reg-btn']))`**: Ellenőrzi, hogy a felhasználó elküldte-e a regisztrációs űrlapot. A `reg-btn` a regisztrációs gomb neve.

---

### 3. Űrlap adatainak lekérése
```php
$username = $_POST['username'];
$password = $_POST['password'];
$name = $_POST['name'];
```
- **`$_POST['username']`**, **`$_POST['password']`**, **`$_POST['name']`**: Az űrlapról érkező adatokat tárolja változókban:
  - `$username`: A felhasználó által megadott felhasználónév.
  - `$password`: A felhasználó által megadott jelszó.
  - `$name`: A felhasználó teljes neve.

---

### 4. Felhasználónév ellenőrzése
```php
$lekerdezes = "SELECT * FROM users WHERE username='$username'";
$talalt_felhasznalo = $conn->query($lekerdezes);

if(mysqli_num_rows($talalt_felhasznalo) == 0){
```
- **`$lekerdezes`**: SQL lekérdezés, amely ellenőrzi, hogy a megadott felhasználónév (`$username`) már létezik-e az adatbázisban.
- **`$conn->query($lekerdezes)`**: Végrehajtja az SQL lekérdezést az adatbázison.
- **`mysqli_num_rows($talalt_felhasznalo) == 0`**: Ellenőrzi, hogy a lekérdezés eredménye üres-e (azaz nincs ilyen felhasználónév).

---

### 5. Új felhasználó regisztrálása
```php
$conn->query("INSERT INTO users VALUES(id, '$username', '$password', '$name')");
```
- **`INSERT INTO users VALUES(id, '$username', '$password', '$name')`**: Beszúr egy új rekordot az `users` táblába a megadott felhasználónévvel, jelszóval és névvel. Az `id` mező valószínűleg automatikusan növekvő (auto-increment) az adatbázisban.

---

### 6. Felhasználó mappájának létrehozása
```php
$mappa = getcwd();
$eleresi_ut = $mappa."\\users\\".$username;

if(mkdir($eleresi_ut, 0777)){
    echo "Tárhely sikeresen létrehozva!";
} else {
    echo "Nem sikerült létrehozni a tárhelyet!";
}
```
- **`$mappa = getcwd();`**: Lekéri az aktuális munkakönyvtár elérési útját.
- **`$eleresi_ut = $mappa."\\users\\".$username;`**: Az elérési út a `users` mappán belül a felhasználó nevével ellátott almappára mutat.
- **`mkdir($eleresi_ut, 0777)`**: Létrehozza a felhasználó mappáját a megadott elérési úton. A `0777` engedélyeket állít be (olvasás, írás, végrehajtás mindenkinek).
- **`echo "Tárhely sikeresen létrehozva!";`**: Sikeres mappalétrehozás esetén üzenetet jelenít meg.
- **`echo "Nem sikerült létrehozni a tárhelyet!";`**: Hiba esetén hibaüzenetet jelenít meg.

---

### 7. Hibakezelés, ha a felhasználónév már foglalt
```php
} else {
    echo "Már van ilyen nevű felhasználó!";
}
```
- Ha a felhasználónév már létezik az adatbázisban, akkor a program kiírja, hogy "Már van ilyen nevű felhasználó!".

---

### 8. HTML űrlap
```html
<!DOCTYPE html>
<html>
   <head>
       <title>Regisztráció</title>
       <meta charset='UTF-8'>
       <meta name='description' content='Rövid leírás az oldal tartalmáról'>
       <meta name='keywords' content='Keresést, Segítő, Szavak, Vesszővel, Elválasztva'>
       <meta name='author' content='Neved'>
       <meta name='viewport' content='width=device-width, initial-scale=1.0'>
   </head>
   <body>
        <h1>Regisztráció</h1>
        <form method="post">
            <input type="text" name="username" placeholder="Felhasználónév">
            <br><br>
            <input type="password" name="password" placeholder="Jelszó">
            <br><br>
            <input type="text" name="name" placeholder="Teljes neved">
            <br><br>
            <input type="submit" name="reg-btn" value="Regisztrálok!">
        </form>
        <a href="login.php">Bejelentkezés</a>
   </body>
</html>
```
- **`<form method="post">`**: A regisztrációs űrlap, amely POST metódussal küldi el az adatokat.
- **`<input type="text" name="username" placeholder="Felhasználónév">`**: Felhasználónév mező.
- **`<input type="password" name="password" placeholder="Jelszó">`**: Jelszó mező.
- **`<input type="text" name="name" placeholder="Teljes neved">`**: Teljes név mező.
- **`<input type="submit" name="reg-btn" value="Regisztrálok!">`**: Regisztrációs gomb.
- **`<a href="login.php">Bejelentkezés</a>`**: Egy link a bejelentkezési oldalra.
