## Config.php

### **1. Adatbázis-kapcsolat létrehozása**

```php
$conn = new mysqli("localhost", "root", "", "013_csontoskincso_2");

```

- **`new mysqli`**: Ez egy PHP osztály, amelyet MySQL adatbázisokkal való kapcsolódáshoz használunk.
- **Paraméterek**:
    - `"localhost"`: Az adatbázis-szerver címe. Ebben az esetben a szerver lokálisan fut, tehát a gépen belül érhető el.
    - `"root"`: Az adatbázis felhasználóneve. A `root` a MySQL alapértelmezett rendszergazdai felhasználója.
    - `""`: A jelszó. Üres, ami azt jelzi, hogy nincs beállítva jelszó a `root` felhasználóhoz.
    - `"013_csontoskincso_2"`: Az adatbázis neve, amelyhez csatlakozni szeretne.

---

### **2. Kapcsolódási hiba kezelése**

```php
if($conn->connect_error){
    die("Sikertelen kapcsolódás! ".$conn->connect_error);
}

```

- Ellenőrzi, hogy a kapcsolat sikeres volt-e:
    - **`$conn->connect_error`**: Ez tartalmazza a hibaüzenetet, ha a kapcsolat sikertelen.
- Ha hiba történt:
    - A **`die`** függvénnyel a program leáll, és kiírja a hibaüzenetet (`"Sikertelen kapcsolódás!"` és az `connect_error` tartalma).

--- 

```php
<?php 

	$conn = new mysqli("localhost", "root", "", "a13_youtube");
	
	if($conn->connect_error){
		die("Sikertelen kapcsolódás! ".$conn->connect_error);
	}

?>
```

## Index.php:
### **PHP működése:**

1. **Adatbázis-kapcsolat és munkamenet indítása:**
    
    - A `require "config.php";` betölti a konfigurációt, ami valószínűleg adatbázis-kapcsolatot és egyéb beállításokat tartalmaz.
    - `session_start();` indítja a munkamenetet, hogy kezelhesse a bejelentkezett felhasználókat.
2. **Felhasználói adatlekérdezés:**
    
    - Ha van `id` a munkamenetben (`if(isset($_SESSION['id']))`), akkor lekérdezi az adott felhasználót az adatbázisból, hogy személyre szabott tartalmat jeleníthessen meg.
        
        ```php
        $lekerdezes = "SELECT * FROM users WHERE id=$userid";
        $talalt_felhasznalo = $conn->query($lekerdezes);
        $felhasznalo = $talalt_felhasznalo->fetch_assoc();
        ```
        
    - Ha nincs bejelentkezett felhasználó, akkor `$userid = 0`, így az oldal nem jelenít meg felhasználóra vonatkozó adatokat.
3. **Navigációs menü:**
    
    - Ha a felhasználó nincs bejelentkezve (`$userid == 0`), akkor csak a **bejelentkezés** lehetőség jelenik meg.
    - Ha a felhasználó be van jelentkezve, akkor a navigációs menü az alábbi lehetőségeket kínálja:
        - A felhasználó neve (link a saját csatornájára)
        - Feltöltés
        - Videók listája
4. **Videók lekérdezése:**
    
    - Az oldalon megjelenő videókat a következő SQL-lekérdezéssel gyűjti össze:
        
        ```php
        $lekerdezes = "SELECT * FROM videos ORDER BY id DESC";
        ```


    ```php
    if($felhasznalo['password'] == $password) {
    ```
    
    Javasolt javítás:
    
    ```php
    if (password_verify($password, $felhasznalo['password'])) {
    ```

## Video.php:
### **PHP működése:**

1. **Adatbázis-kapcsolat és session:**
    
    - A `require "config.php";` betölti a konfigurációs fájlt, amely tartalmazza az adatbázis-kapcsolatot.
    - A `session_start();` lehetővé teszi, hogy a felhasználói session adatokat, például a bejelentkezett felhasználó ID-ját, nyomon követhessük.
2. **Videó adatainak lekérése:**
    
    - A `videoid` paraméter az URL-ből származik, és a kiválasztott videó ID-ját tartalmazza.
    - A `UPDATE videos SET viewCount=viewCount+1 WHERE id=$videoid;` sor növeli a videó megtekintéseinek számát egy egységgel.
    - Az adatbázis-ból lekérjük a videó adatait (például videó neve, fájl neve, feltöltő ID-ja).
3. **Feliratkozás/Leiratkozás funkció:**
    
    - Ha a felhasználó be van jelentkezve (nem vendég), akkor feliratkozhat egy csatornára (a videót feltöltő felhasználóra).
    - A feliratkozás és leiratkozás egy-egy SQL lekérdezés formájában történik, és ezek a műveletek a `feliratkozasok` táblába kerülnek mentésre.
    - Ha a felhasználó nincs bejelentkezve, akkor átirányítjuk a `login.php` oldalra.
4. **Kommentelés funkció:**
    
    - A bejegyzett kommentek az adatbázisba kerülnek, és a felhasználó bejegyzését a videó alatt megjelenítjük.
    - A kommentek a `comments` táblába kerülnek, és a videó alatti kommentek megjelenítéséhez az SQL lekérdezés sorrendje `ORDER BY id DESC`-re van állítva, hogy az új kommentek felül jelenjenek meg.
5. **További videók ajánlása:**
    
    - Az oldal vége felé az adatbázisból lekérdezésre kerülnek más videók, amelyek nem az aktuális videó. Azokat véletlenszerű sorrendben jeleníti meg a rendszer.

---

### **HTML és megjelenés:**

6. **Videó lejátszó:**
    
    - Az oldal tetején egy `<video>` elem található, amely az aktuális videót lejátsza. A videó fájl elérési útja az adatbázisból kerül lekérésre.
7. **Videó neve és feltöltője:**
    
    - A videó neve és a feltöltő felhasználó neve (linkkel) jelenik meg.
    - A feltöltő nevére kattintva a felhasználó elérheti annak csatornáját (`channel.php?channelid=`).
8. **Feliratkozás/Leiratkozás gombok:**
    
    - A felhasználó feliratkozhat vagy leiratkozhat a csatornáról, ha nem ő a feltöltő.
    - A megfelelő gombot (Feliratkozás vagy Leiratkozás) dinamikusan jelenítjük meg a `feliratkozasok` tábla lekérdezésének alapján.
9. **Feliratkozók és megtekintések:**
    
    - Az oldal megjeleníti a videó feliratkozóinak számát és megtekintéseinek számát.
10. **Kommentelés:**
    
    - A videó alatt található egy kommentmező, ahol a felhasználók megoszthatják véleményüket a videóról.
    - Az összes komment a felhasználó nevekkel és azok bejegyzéseivel együtt megjelenik.
11. **További videók listázása:**
    
    - Az oldal alján egy lista található, amely az összes egyéb videót véletlenszerű sorrendben jeleníti meg. A felhasználók könnyen navigálhatnak a többi videóra.

## CSS/Styles.css:
### **Alapértelmezett stílusok:**

```css
* {
	margin: 0;
	padding: 0;
	font-family: sans-serif;
}
```

- Az összes elemre vonatkozik (`*`), hogy a margókat és paddingot nullázzuk, és minden szöveget a sans-serif betűtípussal jelenítünk meg.

---

### **Navigáció:**

```css
nav {
	overflow: hidden;
	box-shadow: 0px 0px 10px #dedede;
}
```

- A navigációs menü (`nav`) doboza egy árnyékot kap (`box-shadow`), hogy vizuálisan kiemelkedjen. Az `overflow: hidden` biztosítja, hogy a menü nem folyik ki a dobozból, ha a tartalom nagyobb lenne.

```css
nav ul {
	list-style-type: none;
}
```

- A navigációs lista elemei (`ul`) nem tartalmaznak felsorolásjeleket (eltávolítja a `list-style-type`).

```css
nav ul li a {
	float: left;
	padding: 10px;
	color: #555;
	font-weight: bold;
	text-decoration: none;
}
```

- Az egyes navigációs linkek (`a`) balra úsznak (`float: left`), 10px-es belső margót kapnak, szürke színűek lesznek, félkövérek, és az alapértelmezett aláhúzás eltávolításra kerül.

```css
nav ul li a.right-menu {
	float: right;
}
```

- A `right-menu` osztályú linkek jobbra úsznak a navigációs menüben.

---

### **Formázás (Űrlapok):**

```css
form {
	max-width: 300px;
	margin: 50px auto;
	padding: 10px;
	box-shadow: 0px 0px 10px #dedede;
	border-radius: 10px;
}
```

- Az űrlap maximális szélessége 300px, középre van igazítva (`margin: 50px auto`), és árnyékot kap, hogy kiemelkedjen. A sarkok lekerekítettek.

```css
label {
	display: block;
	text-align: center;
	font-weight: bold;
	font-size: 20px;
	margin-bottom: 10px;
}
```

- A címkék (`label`) középre vannak igazítva, félkövérek és 20px-es betűmérettel.

```css
input {
	display: block;
	margin-bottom: 10px;
	width: 100%;
}
```

- Az űrlapmezők (`input`) blokkszintűek, 10px-es alsó margóval, és a teljes elérhető szélességet kitöltik.

```css
input[type=text], input[type=password] {
	padding: 5px;
	font-size: 15px;
	border: 2px solid red;
	border-radius: 5px;
	box-sizing: border-box;
}
```

- A szöveges és jelszó beviteli mezők (`input[type=text], input[type=password]`) 5px-es belső margót és 15px-es betűméretet kapnak. A bordereket piros színűre állítja, és lekerekíti őket.

```css
input[type=submit] {
	background: #222;
	border: none;
	padding: 7px;
	border-radius: 5px;
	color: white;
	font-weight: bold;
}
```

- A beküldő gomb (`input[type=submit]`) sötét háttérrel, fehér szöveggel és félkövér betűkkel van formázva.

---

### **Kártya és videó elemek:**

```css
.card-box {
	overflow: hidden;
}
```

- A kártyákat tartalmazó doboz (`card-box`) elrejti a kilógó tartalmat.

```css
.card {
	float: left;
	width: calc(20% - 20px);
	margin: 10px;
	box-shadow: 0px 0px 10px #dedede;
	box-sizing: border-box;
	overflow: hidden;
}
```

- A kártyák (`card`) balra úsznak, és 20%-os szélességgel jelennek meg, míg a margin és árnyék hozzájárul a vizuális elválasztáshoz.

```css
.card img {
	width: 100%;
}
```

- A kártyákban lévő képek kitöltik a kártya teljes szélességét.

---

### **Videó és tartalom:**

```css
.about-video{
	padding: 10px;
}
```

- A videók leírása (`about-video`) 10px-es belső margót kap.

```css
.about-video a.video-name {
	display: block;
	font-weight: bold;
	font-size: 20px;
	text-decoration: none;
	color: black;
	margin-bottom: 10px;
}
```

- A videó neve (`video-name`) kiemelt, félkövér, 20px-es betűméretű, fekete színű és aláhúzás nélkül.

```css
.about-video a.uploader-name {
	display: block;
	font-size: 16px;
	color: #555;
	text-decoration: none;
}
```

- A feltöltő neve (`uploader-name`) kisebb, szürke színű és aláhúzás nélkül jelenik meg.

```css
.video {
	margin: 10px;
	padding: 10px;
	border: 1px solid #dedede;
	box-shadow: 0px 0px 10px #dedede;
	border-radius: 10px;
}
```

- A videók (esetleg hivatkozásokat tartalmazó blokk) 10px-es margót, árnyékot, 1px-es határvonalat és lekerekített sarkokat kapnak.

```css
.video a {
	text-decoration: none;
	color: black;
}
```

- A videó linkjei (a címek például) aláhúzás nélkül és fekete színnel jelennek meg.

```css
.video .edit {
	float: right;
}
```

- A videó szerkesztésére vonatkozó gomb vagy link a jobb oldalra úszik.

---

### **Javaslatok és megjegyzések:**

1. **Responsive dizájn:**
    
    - A kód nem tartalmaz médiakérdéseket (media queries), így nem biztos, hogy mobilbarát lesz. Ha szeretnéd, hogy az oldal jobban alkalmazkodjon különböző képernyőméretekhez, akkor ajánlott médiakérdéseket alkalmazni.
