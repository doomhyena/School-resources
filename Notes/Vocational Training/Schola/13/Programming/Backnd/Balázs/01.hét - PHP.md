### **Kommentek**

A kód különböző típusú kommenteket tartalmaz:

- **Egy soros kommentek** (`//`)
- **Többsoros kommentek** (`/* */`)

A kommentelés megkönnyíti a kód értelmezését és a különböző részek ideiglenes letiltását.

---

### **Változók definiálása és műveletek**

A `$a`, `$b`, `$c`, `$d` változók több művelethez vannak használva.

**Alapműveletek:**

- Összeadás (`$a + $b`)
- Kivonás (`$a - $b`)
- Szorzás (`$a * $b`)
- Osztás (`$a / $b`)
- Maradékos osztás (`$a % $b`)

Ezeket különböző feladatokban használja a kód.

---

### **1. Feladatsor** _(kommentelve van)_

- Különböző egyszerű matematikai műveleteket hajt végre `$a`, `$b`, és `$c` változókkal.
- Ezek a műveletek tartalmaznak zárójeles kifejezéseket is:
    
    ```php
    echo ($a + $c) / $b;
    ```
    

---

### **2. Feladatsor**

Ezek a feladatok aktívak a kódban, és az alábbi műveleteket végzik el:

1. **Szorzás:**
    
    ```php
    echo $a * $b;  // 10 * 5 = 50
    ```
    
2. **Maradékos osztás:**
    
    ```php
    echo $b % $c;  // 5 % 2 = 1
    ```
    
3. **Szorzás és osztás kombinációja:**
    
    ```php
    echo ($a * $c) / $d;  // (10 * 2) / 3 ≈ 6.6667
    ```
    
4. **Két külön szorzás eredményeinek összeadása:**
    
    ```php
    echo ($a * $c) + ($b * $d);  // (10 * 2) + (5 * 3) = 20 + 15 = 35
    ```
