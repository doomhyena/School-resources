## Java alapok

### 1. Kiíratás a konzolra

```java
System.out.println("Helló Világ!");
System.out.println("Legyen szép napod!");
System.out.println("Java-t tanulni jó!");
```

- `System.out.println()` → kiír a konzolra, és új sort kezd.
- Szöveget idézőjelek között kell írni.
- Számokat és műveleteket is ki lehet íratni:

```java
System.out.println(3);        // szám kiíratás
System.out.println(3 + 3);    // művelet
System.out.println(2 * 5);    // szorzás
```
### 2. Változók

#### Példa:

```java
String nev = "Csontos Kincső";
int eletkor = 20;
int szam = 15;
```

- **Deklarálás**: megmondjuk a típusát (`int`, `String`, stb.) és adunk neki nevet.
- **Értékadás**: `=` operátorral történik.
- Kiíratás:

```java
System.out.println(nev);
System.out.println(eletkor);
System.out.println(szam);
```

### 3. Alapvető adattípusok (primitívek)

- `int` → egész szám (pl. `5`)
- `float` → lebegőpontos szám (pl. `5.99f`)
- `double` → nagyobb pontosságú lebegőpontos (pl. `19.99d`)
- `char` → karakter (pl. `'D'`)
- `boolean` → logikai érték (`true` / `false`)

```java
int myNum = 5;
float myFloatNum = 5.99f;
char myLetter = 'D';
boolean myBool = true;
```

### 4. String és összefűzés

```java
String firstName = "Kincső ";
String lastName = "Csontos";
String fullName = firstName + lastName;
System.out.println(fullName);  // Kincső Csontos
```

- `+` operátorral össze lehet fűzni szövegeket.
### 5. Számok összeadása és szöveg + szám különbség

```java
int x = 5;
int y = 6;
System.out.println(x + y); // 11
System.out.println("The sum is " + x + y);   // "The sum is 56"
System.out.println("The sum is " + (x + y)); // "The sum is 11"
```

**Fontos:** ha szöveggel fűzöd, akkor a szám is szövegként csatolódik. Zárójelben számoltatni kell, ha tényleges összeget akarsz.
### 6. További példák számokra

```java
int z = 50;
System.out.println(x + y + z); // 61

double asd = 19.99d;
float f1 = 35e3f;   // 35 * 10^3 = 35000.0
double d1 = 12E4d;  // 12 * 10^4 = 120000.0
```

### 7. Boolean értékek

```java
boolean isJavaFun = true;
boolean isPHPFun = false;
System.out.println(isJavaFun); // true
System.out.println(isPHPFun);  // false
```

### 8. Karakterek (ASCII kód alapján is lehet)

```java
char myGrade = 'A';
char myVar1 = 65, myVar2 = 66, myVar3 = 67;
System.out.println(myGrade); // A
System.out.println(myVar1);  // A (65 ASCII)
System.out.println(myVar2);  // B (66 ASCII)
System.out.println(myVar3);  // C (67 ASCII)
```

### 9. Egyszerű számolás több változóval

```java
int items = 50;
float costPerItem = 9.99f;
float totalCost = items * costPerItem;
char currency = '$';

System.out.println("Tételek száma: " + items);
System.out.println("Tételek ára: " + costPerItem + currency);
System.out.println("Teljes költség = " + totalCost + currency);
```

### 10. Primitív és nem primitív típusok

- **Primitív típusok**: `int`, `float`, `double`, `char`, `boolean`
    
    - kisbetűvel kezdődnek, mindig értéket tartalmaznak
- **Nem primitív típusok**: `String`, tömbök, osztályok
    - nagybetűvel kezdődnek, metódusaik is vannak
    - lehetnek `null` értékűek is

### 11. `var` kulcsszó

A `var` automatikusan felismeri a típust:

```java
var myNumber = 5;       // int
var myDouble = 9.98;    // double
var myChar = 'D';       // char
var myBoolean = true;   // boolean
var myString = "Hello"; // String
```

### 12. Típuskonverzió

Automatikus (kisebb → nagyobb):

```java
int myInt = 9;
double myDoub = myInt;  // automatikus konverzió int → double
System.out.println(myInt);  // 9
System.out.println(myDoub); // 9.0
```

##  Műveletek, logika és Stringek

### 1. Százalék számítása

```java
int maxScore = 500;
int userScore = 423;
double percentage = (double) userScore / maxScore * 100.0d;
System.out.println("A felhasználó százalékos aránya: " + percentage);
```

- `(double) userScore` → **típuskonverzió** (casting), hogy az osztás ne egész számot adjon.
- Eredmény: `84.6...`
### 2. Összeadás példák

```java
int sum1 = 100 + 50;    // 150
int sum2 = sum1 + 250;  // 400
int sum3 = sum2 + sum2; // 800
```

### 3. Aritmetikai műveletek

```java
int a = 10, b = 3;
System.out.println(a + b); // 13
System.out.println(a - b); // 7
System.out.println(a * b); // 30
System.out.println(a / b); // 3 (egész osztás!)
System.out.println(a % b); // 1 (maradék)
```

Ha lebegőpontos számokkal osztunk:

```java
double d = 10.0d, e = 3.0d;
System.out.println(d / e); // 3.333...
```

### 4. Inkrementálás és dekrementálás

```java
int c = 5;
++c;  // növel 1-gyel
System.out.println(c); // 6
--c;  // csökkent 1-gyel
System.out.println(c); // 5
```

Példa „valós szituációra”:

```java
int peopleInRoom = 0;
peopleInRoom++; // belépett valaki
peopleInRoom++; // újabb ember
peopleInRoom++; // újabb ember
System.out.println(peopleInRoom); // 3
peopleInRoom--; // egy ember kimegy
System.out.println(peopleInRoom); // 2
```

### 5. Rövidített értékadás

```java
int savings = 100;
savings += 50; // ugyanaz, mint savings = savings + 50;
System.out.println("Teljes megtakarítás: " + savings); // 150
```

### 6. Logikai összehasonlítás

```java
int age = 18;
System.out.println(age >= 18); // true
System.out.println(age < 18);  // false
```

Jelszó hossz példa:

```java
int passwordLength = 5;
System.out.println(passwordLength >= 8); // false
System.out.println(passwordLength < 8);  // true
```

### 7. Logikai műveletek (AND, OR, NOT)

```java
boolean isLoggedIn = true;
boolean isAdmin = false;

System.out.println(isLoggedIn && !isAdmin); // true && true → true
System.out.println(isLoggedIn || isAdmin);  // true || false → true
System.out.println(!isLoggedIn);            // false
```

- `&&` → ÉS
- `||` → VAGY
- `!` → NEM
### 8. String műveletek

```java
String txt = "ABCDEFGHIJKLMNOPQRSTUVWXYZ";
System.out.println(txt.length()); // hossz = 26

String helloworld = "Hello World";
System.out.println(helloworld.toUpperCase()); // "HELLO WORLD"
System.out.println(helloworld.toLowerCase()); // "hello world"
```

### 9. Speciális karakterek Stringben

- `\n` → új sor
- `\t` → tabulátor
- `\b` → visszalépés (backspace)
- `\r` → sor elejére ugrás (carriage return)
- `\f` → form feed (ritka, régi nyomtatós dolog)
### 10. Math osztály

```java
System.out.println(Math.max(5, 10)); // 10
System.out.println(Math.min(5, 10)); // 5
```

A `Math` osztályban rengeteg hasznos függvény van (pl. `sqrt()`, `abs()`, `pow()`, stb.).
### 11. Logikai példa

```java
int g = 10, h = 9;
System.out.println(g > h); // true
```

Szavazási példa:

```java
int myAge = 20;
int votingAge = 18;
System.out.println(myAge >= votingAge); // true
```

### Összegzés
- Hogyan lehet **százalékot számolni** és típusokat konvertálni
- Az **aritmetikai műveleteket** (`+`, `-`, `*`, `/`, `%`)
- Az **inkrementálást/dekrementálást** (`++`, `--`)
- Az **összehasonlító és logikai operátorokat** (`>=`, `<`, `&&`, `||`, `!`)
- Alapvető **String műveleteket** (hossz, kis/nagybetű)
- Speciális karaktereket `\n`, `\t`, stb.
- A **Math osztály** használatát (`max()`, `min()`).

## Feltételes szerkezetek (`if`, `else`, `else if`, `?:`)

### 1. Az `if` szerkezet alapja

```java
if (feltétel) {
    // ha a feltétel igaz, ez a kód fut le
}
```

👉 Példa:

```java
boolean isRaining = true;

if (isRaining) {
    System.out.println("Vigyél esernyőt");
}
```

---

### 2. `if – else` szerkezet

```java
if (feltétel) {
    // ha igaz
} else {
    // ha hamis
}
```

Példa:

```java
boolean isRainingOrNot = false;

if (isRainingOrNot) {
    System.out.println("Vigyél esernyőt!");
} else {
    System.out.println("Ma nem fog esni az eső, nem kell esernyő!");
}
```
### 3. `if – else if – else` láncolat

Ha több lehetőséget akarunk ellenőrizni:

```java
int weather = 2; // 1 = esős, 2 = napos, 3 = felhős

if (weather == 1) {
    System.out.println("Vigyél esernyőt.");
} else if (weather == 2) {
    System.out.println("Vegyél fel napszemüveget.");
} else {
    System.out.println("Nem kell se esernyő, se napszemüveg.");
}
```

Példa napszakra:

```java
int time = 14;
if (time < 10) {
    System.out.println("Jó reggelt.");
} else if (time < 18) {
    System.out.println("Jó napot.");
} else {
    System.out.println("Jó estét.");
}
```
### 4. Ternary operator (`? :`)

Ez egy rövidített `if – else`, egy sorban.

```java
String result = (feltétel) ? "ha igaz" : "ha hamis";
```

Példa:

```java
int timeie = 20;
String result = (timeie < 18) ? "Jó napot." : "Jó estét.";
System.out.println(result);
```
### 5. Logikai operátorok `if`-ben

Az `if`-feltételben használhatunk logikai műveleteket:
- `&&` (ÉS)
- `||` (VAGY)
- `!` (NEM)

Példa:

```java
if (a > b || a > c) {
    System.out.println("Legalább egy feltétel teljesül.");
} else {
    System.out.println("Vagy mégsem.");
}
```

### 6. Komplex feltételek

```java
boolean isLoggedInOrNot = true;
boolean isAdminOrNot = false;
int securityLevel = 3;

if (isLoggedInOrNot && (isAdminOrNot || securityLevel <= 2)) {
    System.out.println("Engedély megadva.");
} else {
    System.out.println("Engedély megtagadva.");
}
```

Példa magyarázat:
- Beléphet, ha be van jelentkezve **ÉS** (admin **VAGY** alacsony a securityLevel).

### 7. Példa konkrét érték ellenőrzésére

```java
int doorCode = 1337;

if (doorCode == 1337) {
    System.out.println("Helyes kód. Az ajtó nyitva van.");
} else {
    System.out.println("Rossz kód. Az ajtó zárva marad.");
}
```
### Összegzés

A feltételes szerkezetekkel:
- tudsz döntést hozni a programodban
- kezelheted a **különböző eseteket** (`if`, `else if`, `else`)
- gyorsíthatsz `ternary operator`-ral
- kombinálhatsz feltételeket logikai operátorokkal

## `switch` és `while` ciklusok

### 1. `switch` szerkezet

A `switch` akkor hasznos, ha sok lehetséges értéket kell ellenőrizni **ugyanarra a változóra**.  
Szintaxisa:

```java
switch (kifejezés) {
    case érték1:
        // kódblokk
        break;
    case érték2:
        // kódblokk
        break;
    ...
    default:
        // ha egyik sem illik
}
```

Példa: hét napjai

```java
int day = 4;
switch (day) {
    case 1:
        System.out.println("Hétfő");
        break;
    case 2:
        System.out.println("Kedd");
        break;
    case 3:
        System.out.println("Szerda");
        break;
    case 4:
        System.out.println("Csütörtök");
        break;
    case 5:
        System.out.println("Péntek");
        break;
    case 6:
        System.out.println("Szombat");
        break;
    case 7:
        System.out.println("Vasárnap");
        break;
    default:
        System.out.println("Várjuk a hétvégét");
}
```

Magyarázat:
- A `case` ág fut le, amelyik illeszkedik a `day` értékére.
- A `break` megakadályozza, hogy a program tovább fusson a következő ágra is.
- A `default` olyan, mint az `else`: akkor fut, ha egyik `case` sem illik.
###  2. `while` ciklus

A `while` addig ismétli a kódot, amíg a feltétel **igaz**.

### Alap szintaxis:

```java
while (feltétel) {
    // ismételt kód
}
```
### Példa 1: számlálás

```java
int i = 0;

while (i < 5) {
    System.out.println(i);
    i++;
}
```

👉 Eredmény: `0 1 2 3 4`

---

### Példa 2: sosem fut le

```java
while (i < 5) {
    System.out.println("Ez soha nem lesz kiírva.");
}
```

👉 Itt `i` már **nem kisebb 5-nél**, ezért a ciklus **egyszer sem fut le**.
### Példa 3: visszaszámlálás

```java
int countdown = 3;

while (countdown > 0) {
    System.out.println(countdown);
    countdown--;
}

System.out.println("Boldog új évet!");
```

 Eredmény:

```
3
2
1
Boldog új évet!
```
###  Összegzés

- A **`switch`**: több ág közötti választásra jó, ha egy változót több értékre ellenőrzünk.
- A **`while`**: addig fut, amíg a feltétel igaz → jó ciklikus ismétlésekhez.
- Vigyázni kell, hogy a `while`-ból **ki tudjon lépni** (különben végtelen ciklus lesz).

##  `do-while`, `for`, `for-each`, tömbök, `break` és `continue`

### 1. `do-while` ciklus

A `do-while` **mindig lefut egyszer**, mert előbb hajtja végre a kódot, és csak utána ellenőrzi a feltételt.

```java
do {
    // kódblokk
} while (feltétel);
```

👉 Példa:

```java
int i = 0;
do {
    System.out.println(i);
    i++;
} while (i < 5);
```

- Eredmény: `0 1 2 3 4`
- Akkor is lefut legalább egyszer, ha a feltétel már az elején hamis.
### 2. `for` ciklus

A `for` ideális, ha **pontosan tudjuk, hányszor kell futnia**.

Szintaxis:

```java
for (kezdőérték; feltétel; lépés) {
    // kódblokk
}
```

Példa:

```java
for (int j = 0; j < 5; j++) {
    System.out.println(j);
}
```

Eredmény: `0 1 2 3 4`

### Különleges esetek

- Összegzés:

```java
int sum = 0;
for (int sz = 1; sz <= 5; sz++) {
    sum = sum + sz;
}
System.out.println("Az összeg: " + sum); // 15
```

- Visszaszámlálás:

```java
for (int v = 5; v > 0; v--) {
    System.out.println(v);
}
```

- Hamis feltétellel → egyszer sem fut:

```java
for (int zs = 10; zs < 5; zs++) {
    System.out.println("Ez soha nem íródik ki");
}
```

---

### 3. `for-each` ciklus

Tömb vagy lista **minden elemén végigmegy** – egyszerűbb, mint sima `for`.

Példa:

```java
String[] carsi = {"Volvo", "BMW", "Ford", "Mazda"};

for (String car : carsi) {
    System.out.println(car);
}
```

Számokra:

```java
int[] numbers = {10, 20, 30, 40};

for (int num : numbers) {
    System.out.println(num);
}
```

---

### 4. `break` és `continue`

- **`break`**: megszakítja a ciklust.
- **`continue`**: átugorja az adott iterációt, és megy tovább.

Példák:

```java
for (int as = 0; as < 10; as++) {
    if (as == 4) {
        break; // kilép a ciklusból
    }
    System.out.println(as);
}
```

```java
int u = 0;
while (u < 10) {
    if (u == 4) {
        u++;
        continue; // átugorja a 4-et
    }
    System.out.println(u);
    u++;
}
```

Real-life példa: negatív számokat átugrunk, nullánál leállunk:

```java
int[] num = {3, -1, 7, 0, 9};

for (int n : num) {
    if (n < 0) {
        continue; // negatív számok kihagyása
    }
    if (n == 0) {
        break; // megáll a ciklus
    }
    System.out.println(n);
}
```

---

### 5. Tömbök (`arrays`)

Tömb: több érték tárolása **azonos típusban**.

Példa inicializálással:

```java
String[] cars = {"Volvo", "BMW", "Ford", "Mazda"};

System.out.println(cars[0]); // Volvo
cars[0] = "Opel";            // első elem átírása
System.out.println(cars[0]); // Opel
System.out.println(cars.length); // elemszám
```

Másik forma:

```java
String[] cars = new String[4];
cars[0] = "Volvo";
cars[1] = "BMW";
cars[2] = "Ford";
cars[3] = "Mazda";
```

Rövidített forma:

```java
String[] cars = new String[] {"Volvo", "BMW", "Ford", "Mazda"};
```

### Összegzés

- **`do-while`**: legalább egyszer fut.
- **`for`**: fix ismétlésszámhoz jó.
- **`for-each`**: tömb/lista elemein végigmenni egyszerűen.
- **`break`/`continue`**: ciklusvezérlés.
- **Tömbök**: több azonos típusú adatot tartalmaznak.

## Metodúsok
### 1. Mi az a metódus?

- Egy **kódblokk**, amit **csak akkor futtat a program, ha meghívod**.
    
- **Paramétereket** (adatokat) is átadhatsz neki.
    
- Célja: **újrafelhasználhatóság** → egyszer írod meg, többször használod.
    

Szintaxis:

```java
visszatérésiTípus metódusNév(paraméterek) {
    // végrehajtandó kód
    return érték; // ha van visszatérési érték
}
```

---

### 2. Void metódusok (nem adnak vissza értéket)

Példa:

```java
static void myMethod() {
    System.out.println("Lefutottam%");
}
```

Meghívás:

```java
myMethod(); // kiírja: Lefutottam%
```

Paraméterrel:

```java
static void Names(String name) {
    System.out.println(name + " a neve.");
}
```

Meghívás:

```java
Names("Kincső");
```

---

### 3. Paraméter több típussal

Több paraméter is lehet:

```java
static void Person(String fname, int age) {
    System.out.println(fname + " is " + age);
}
```

Meghívás:

```java
Person("Anja", 31);
```

---

### 4. Feltételes logika metódusban

Példa – életkor ellenőrzés:

```java
static void checkAge(int age) {
    if (age < 18) {
        System.out.println("Hozzáférés megtagadva - Nem vagy elég idős!");
    } else {
        System.out.println("Engedély megadva - Elég idős vagy!");
    }
}
```

Meghívás:

```java
checkAge(20);
```

---

### 5. Visszatérési érték (`return`)

Metódus, ami számot ad vissza:

```java
static int Matek(int x) {
    return 5 + x;
}
```

Meghívás:

```java
System.out.println(Matek(3)); // 8
```

---

### 6. Metódusok túlterhelése (overloading)

Ha **ugyanaz a metódusnév**, de más paraméterekkel → a Java tudja, melyiket kell futtatni.

Példák:

```java
static int plusMethod(int x, int y) {
    return x + y;
}

static double plusMethod(double x, double y) {
    return x + y;
}
```

Meghívás:

```java
int myNum1 = plusMethod(8, 5);       // int változat
double myNum2 = plusMethod(4.3, 6.26); // double változat
```

Eredmény:

```
int: 13
double: 10.56
```

---

### 7. Rekurzió (metódus önmagát hívja meg)

Egy metódus **saját magát hívja** → amíg a feltétel igaz.

Egyszerű összegzés:

```java
public static int sum(int k) {
    if (k > 0) {
        return k + sum(k - 1); // önmagát hívja
    } else {
        return 0;
    }
}
```

`sum(10)` → `55`

Két paraméterrel:

```java
public static int sum(int start, int end) {
    if (end > start) {
        return end + sum(start, end - 1);
    } else {
        return end;
    }
}
```

`sum(5, 10)` → `45`

---

# Rövid összefoglaló

**Metódus** = kódblokk, amit hívni kell.  
**Void** → nem ad vissza értéket.  
**Paraméteres metódus** → adatot vihetsz be.  
**`return` metódus** → értéket ad vissza.  
**Overloading** → ugyanaz a név, más paraméterekkel.  
**Rekurzió** → metódus önmagát hívja.
