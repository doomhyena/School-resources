1. véletlenszám-generálás + formázott kiírás,
2. konstans értékek felhasználása,
3. fájlba írás.

---

### **1. `main` metódus**

```java
public static void main(String[] args) {
    double a = Math.E;
    int b = new Random().nextInt(0, 1000);
    String sz = "Lajos";
```

- `Math.E` → a természetes logaritmus alapja (≈ 2.71828).
- `new Random().nextInt(0, 1000)` → generál egy véletlen **egész számot 0 és 999 között**.
- `sz = "Lajos"` → egy egyszerű szöveg/string változó.

---

### **2. Kiírás formázással**

```java
System.out.printf(
    "Valós szám: %f \nEgész szám: %d \nSzöveg: %s \n2 tizedesre kerekített valós szám: %.2f\n",
    a, b, sz, a
);
```

Itt a `printf`-et használod (formázott kiírás).

- `%f` → valós szám lebegőpontos formátumban (pl. `2.718282`)
- `%d` → egész szám (int)
- `%s` → szöveg (string)
- `%.2f` → lebegőpontos szám **2 tizedesre kerekítve** (pl. `2.72`).

Tehát a konzolon kb. így fog kinézni:

```
Valós szám: 2.718282 
Egész szám: 537 
Szöveg: Lajos 
2 tizedesre kerekített valós szám: 2.72
```

### **3. Fájlba írás (`kiir` metódus)**

```java
private static void kiir() {
    try {
        FileWriter fw = new FileWriter("proba.txt");
        fw.write("Csontos Kincső");
        fw.close();
    } catch (IOException e) {
        throw new RuntimeException(e);
    }
}
```

- `FileWriter fw = new FileWriter("proba.txt");` → létrehoz egy fájlt a projekt mappában **proba.txt** néven.
- `fw.write("Csontos Kincső");` → beleírja a szöveget.
- `fw.close();` → lezárja a fájlt (nagyon fontos, mert különben nem biztos, hogy minden adat belekerül).
- `catch (IOException e)` → ha valami gubanc van (pl. nem tud írni a fájlba), kivételt dob.    

Ez a rész gondoskodik arról, hogy **konzolra írás mellett fájlba is mentse az adatokat**.
