**import requests**  
- Betölti a _requests_ könyvtárat, amivel HTTP kéréseket tudsz küldeni.

---

## **GET - adat lekérése**

**response = requests.get("[https://jsonplaceholder.typicode.com/todos/1](https://jsonplaceholder.typicode.com/todos/1)")**  
- GET kérés küldése a megadott URL-re.  
- Ez lekéri az 1-es todo elemet.

**print(response.json())**  
- A válasz JSON formátumú tartalmát írja ki.  
- Python automatikusan dictté alakítja.

**print(response.status_code)**  
- Kiírja a válasz státuszkódját (pl. 200).

---
## **POST – új adat létrehozása**

**data = { ... }**  
- Egy új todo elem adatai, amit elküldesz a szervernek.

**response = requests.post("[https://jsonplaceholder.typicode.com/todos](https://jsonplaceholder.typicode.com/todos)", json=data)**  
- POST kérés: új adatot hoz létre.  
- A `json=data` rész automatikusan JSON-né alakítja a Python dictet.

**print(response.json())**  
- A szerver által visszaküldött új objektumot jeleníti meg.

---
## **PUT - adat teljes frissítése**

**data = { ... }**  
- Az új, teljes objektum, ami leváltja a régit.

**response = requests.put("[https://jsonplaceholder.typicode.com/todos/100](https://jsonplaceholder.typicode.com/todos/100)", json=data)**  
- PUT kérés: a `/100` erőforrást teljes egészében felülírja.

**print(response.json())**  
- Kiírja a módosított objektumot.

---
## **PATCH - adat részleges frissítése**

**data = { ... }**  
- Csak azokat a mezőket tartalmazza, amiket módosítani akarsz.

**response = requests.patch("[https://jsonplaceholder.typicode.com/todos/100](https://jsonplaceholder.typicode.com/todos/100)", json=data)**  
- PATCH kérés: csak a megadott mezők változnak.

**print(response.json())**  
- Kiírja a részben módosított objektumot.

---
## **DELETE - adat törlése**

**response = requests.delete("[https://jsonplaceholder.typicode.com/todos/100](https://jsonplaceholder.typicode.com/todos/100)", json=data)**  
- DELETE kérés: törli a megadott azonosítójú erőforrást.  
– A JSON rész igazából itt nem szükséges.

**print(response.json())**  
- A válasz JSON tartalmát írja ki.  
- A szolgáltató gyakran üres objektummal válaszol.
## Teljes kód:
```python
import requests  
  
# GET - adat lekérése  
  
response = requests.get("https://jsonplaceholder.typicode.com/todos/1")  
  
print(response.json())  
print(response.status_code)  
  
# POST - új adat létrehozása  
  
data = {  
    "title": "Valami új dolog",  
    "completed": False  
}  
  
response = requests.post(  
    "https://jsonplaceholder.typicode.com/todos",  
    json=data  
)  
  
print(response.json())  
  
# PUT - adat teljes frissítése  
  
data = {  
    "title": "Valami új dolog",  
    "completed": True  
}  
  
response = requests.put(  
    "https://jsonplaceholder.typicode.com/todos/100",  
    json=data  
)  
  
print(response.json())  
  
# PATCH - adat részleges frissítése  
  
data = {  
    "title": "Valami új dolog",  
    "completed": True  
}  
  
response = requests.patch(  
    "https://jsonplaceholder.typicode.com/todos/100",  
    json=data  
)  
print(response.json())  
  
# DELETE - adat törlése  
  
response = requests.delete(  
    "https://jsonplaceholder.typicode.com/todos/100",  
    json=data  
)  
  
print(response.json())
```
