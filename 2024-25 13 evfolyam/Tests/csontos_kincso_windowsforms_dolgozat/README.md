# Csontos Kincső Windows Forms Dolgozat
## Leírás
Kiss Ádám óráján készült C# Windows Forms Dolgozat leírása és megoldása
## Feladat
Dolgozat-nyilvántartó rendszer:
  - A felhasználó megadhatja egy adott dolgozat tantárgyát, témakörét, max pontszámát és az osztály által elért átlagot. Egyik se maradhasson üres a felvitel előtt.
  - Egy nagy szövegmező megjelníti a program futása során felvitt sorokat (a tetején van fejléc a fent említett címekkel).
  - Egy törlés gomb kiüríti az eddig megadott tartalmat a dobozból (de a fejléc ott marad).
  - Bezárás gomb
  - Egy cimke jelezze ki az aktuális dátumot valahol az ablakban.
  - Egy cimke mutassa az eddig felvitt sorok számát (ürítéskor ez is nullázódjon).
  - Súgó gomb, ami nyit egy új ablakot, amelyben magyarázatként fel lesz sorolva a felvihető értékek 
    listája (tárgy, téma, max pontszám,elért átlag) és egy 'Vissza a főoldalra' gomb.
